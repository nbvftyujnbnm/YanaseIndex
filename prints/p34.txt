(2) 中央アジアとアフリカ
①中央アジア イラン同様に、アレクサンドロス大王の支配 ⇒ セレウコス朝シリアの支配
○[1 バクトリア](前225-前139) … アム川上流で[2 ギリシア人]がセレウコス朝から独立
→ インダス川上流域まで勢力を拡大。滅亡後、この地域には大月氏が侵入。後１世紀まで支配
月氏(民族系統不明の遊牧民)が匈奴の冒頓単于に敗れ、タリム盆地東部からイリ地方に逃れたもの
○[3 クシャーナ朝](１世紀～３世紀) … イラン系遊牧民。大月氏の支配を脱する ササン朝
→ 西北インドに侵入して王朝を開き、次第にインド化。シャープール１世に侵略され衰退
※[4 ガンダーラ地方] … ギリシア風の仏教美術が栄えた。[1 バクトリア]の文化が影響
○[5 エフタル] … ４世紀後半、アム川・シル川流域で活動したイラン系遊牧民
→ ５世紀にはササン朝を圧迫、６世紀にはインドのグプタ朝に侵入し衰退させる
○トルコ化とイスラーム化 … ９世紀ごろには、中央アジアは「トルキスタン」と称されるように
・トルコ化 … 東から。６世紀に突厥が[5 エフタル]を滅ぼす。９世紀にウイグルが西走
・イスラーム化 … 西から。７世紀にウマイヤ朝、８世紀にアッバース朝の支配 キルギスの圧迫
※[6 ソグド人]の衰退 … 前６世紀(アケメネス朝侵入)以前から、中央アジアで東西交易に従事
→ イラン系、拠点サマルカンド。ソグド語/文字を使用(アラム文字の影響)。「ソグディアナ」
②アフリカ
○[7 クシュ王国](前10 世紀～後４世紀) … ナイル川上流に成立。最古の黒人王朝
・エジプト進出 … 前８世紀に末期王朝を支配。前７世紀に[8 アッシリア]の侵入をうけ後退
・[9 メロエ]遷都(前６世紀) … さらに南方へ後退。遷都後は とも呼称
・文化 … 高度な鉄器文明。製鉄で栄える。独自の象形文字(メロエ文字)も存在、未解読
○[10 アクスム王国](紀元前後～９世紀) … エチオピア北部。４世紀にメロエ王国を滅ぼす
・紅海～インド洋交易で繁栄(３世紀) … ローマ帝国衰退に際して。ササン朝の商人と抗争
・キリスト教国家 … ４世紀にコプト教会(キリスト単性説を唱えるエジプトの教会)を受容
・衰退 … ７世紀にイスラーム勢力が勃興、交易路を断たれる カルケドン公会議(451)後も存続
３．地中海世界の形成とオリエントとの融合
(1) エーゲ文明とポリスの形成 地中海世界。オリエント世界の影響を受けつつ独自の文明が発展
①[11 エーゲ文明] … ギリシア文明に先立つ、エーゲ海に成立した[12 青銅器]文明の総称
○[13 クレタ文明](ミノス文明、前20 世紀～前15 世紀ごろ)
・宮殿の建設 … クレタ島の[14 クノッソス]などに、複雑な構造の宮殿がつくられる
・平和的な海洋文明 … 宮殿には城壁がなく、海洋生物を描く鮮やかな壁画や壺などが残される
・未解読文字 … クレタ絵文字、 1900 年、[15 エヴァンズ](英)が発掘
・衰退 … [16 アカイア人]の攻撃を受ける(古代ギリシア人の第１次南下)
