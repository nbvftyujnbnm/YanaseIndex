③大戦中の情勢
〇秘密外交の展開 … 戦後の敵領土・植民地の分配を秘密条約で決定。戦後、紛争の原因となる
〇新兵器の導入 … [1 戦車・ 毒ガス・ 飛行機・ 潜水艦]など。南北戦争以来の機関銃も使用
→ 毒ガスはハーグ議定書(1907)やジュネーヴ議定書(1925)で禁止されたが使用され続けた
〇[2 総力戦体制] … 一般市民をも巻き込み広範な被害をもたらす、今日の戦争の姿となる
・国内の工業生産力 … 新兵器の開発・大量生産の上で重要となった
・軍需生産の優先 … 生活必需品の減産、配給制の開始。青少年や女性の労働増加
→ 英領[3 インド]からは約150 万人が前線に動員されるなど、植民地人も大量に動員
・挙国一致内閣の成立 … 労働組合・社会主義政党も含め、国内対立を解消し戦争に注力
→ 例えば英のロイド＝ジョージ内閣(1916)、仏のクレマンソー内閣(1917)
※[4 第２インターナショナル]の瓦解 … 各国の社会主義者も自国の戦争を支持していった
・ナショナリズムの高揚 … 当初各国民は熱狂的に戦争を支持、短期間で決着すると楽観視
→ ソンムの戦いなどで膠着状態が明らかとなり、民衆は生活への不満を表明し始める
⇒ 女性の社会進出が進んだため、女性参政権獲得をめざす動きもみられはじめた
④戦局の変化と終戦 [11 ポーランド]、[12 バルト三国]
〇[5 ロシア]の離脱 [13 フィンランド]、ウクライナ独立
・[6 三月革命]( 1917.3) … 皇帝ニコライ２世が退位、[7 ロマノフ朝]倒壊。帝政終焉
→ 臨時政府(戦争継続を主張)と、ソヴィエト勢力との二重権力状態となる
・[8 十一月革命](1917.11) … 社会主義政権成立、「9 平和に関する布告」を発表
・[10 ブレスト＝リトフスク条約](1918.3) … ソヴィエト政府が独露で単独講和
→ ロシアは自領を大幅に縮小、代償として独墺土ブルガリアとの全戦闘を停止
〇 [14 アメリカ]の参戦( 1917.4) … 翌年から本格的に戦闘参加、ドイツ降伏を決定づける
・ドイツの[15 無制限潜水艦作戦] … イギリス海軍の海上封鎖に対抗して開始(1915.2)
→ Ｕボートとよばれる潜水艦で、警告なく無差別攻撃。商船を撃沈し、英を食糧難に陥れる
・[16 ルシタニア号]撃沈事件(1915.5) … イギリス豪華客船を独潜水艦が撃沈
→ 乗客約1200 人が死亡、うちアメリカ人約130 人。世論の批判が高まり独は作戦を中断
⇒ 独は戦局打開のため1917 年２月に作戦再開。アメリカ参戦の一因となる
・参戦の動機 … アメリカ国内における対ドイツ世論の沸騰。日本の権益拡大への警戒も
→ 英仏に対する多額の資金援助を、連合国側を勝利させることで回収する狙いもあった
〇ドイツの降伏( 1918) … 1917 年から国内では軍部が独裁。社会主義政党・労働者の反発
・同盟国側の降伏 … 1918 年秋に、オーストリア・トルコ・ブルガリアが相次いで降伏
・ドイツ革命 … 1918 年11 月の[17 キール軍港]における水兵反乱が全土に波及
→ ヴィルヘルム２世退位、帝政崩壊。臨時政府(社会民主党中心)が休戦条約に調印
⇒ 第一次世界大戦の終結(1918)。パリ講和会議(1919)で講和の条件が話し合われた
