②オランダの繫栄と海外進出
○政治体制 … ７州による[1 連邦制]。連邦議会が存在し、共和政がとられた
・連邦議会 … ホラント州(アムステルダムが存在し、経済力が高い)が軍事・行政の中心
→ 議会の強制力は弱く、各州の自治権が強かった
・オランダ総督 … 形式上の君主。オラニエ家の世襲が法制化され、事実上のオランダ王家
○中心都市の推移 … オランダでは干拓による近郊農業や、漁業も発展
・アントウェルペン … 16 世紀の商業革命をうけて発展。ネーデルラント南部に存在
→ 独立戦争で破壊されたほか、新教徒の商人が北部に逃れたため衰退
・[2 アムステルダム] … 17 世紀前半から商業の中心地として繫栄
→ 宗教的な自由さから、ユダヤ教徒・ユグノー・ピューリタンが移住したことも一員
・交易内容 … 高い造船技術がいかされた。バルト海での中継貿易、大西洋での奴隷貿易など
○オランダ[3 東インド会社](1602-1799) … 世界初の株式会社。略称はＶ・Ｏ・Ｃ
・設立の経緯 … 独立戦争でスペインとの通商が断絶。自力でのアジア交易が求められる
→ 複数の貿易会社が設立・競合していたが、英東インド会社の設立(1600)をうけて一元化
・権限 … インド洋から太平洋にかけての貿易権と支配権(総督の任命)、敵対勢力との交戦権
→ 敵国であるスペイン・ポルトガル同君連合への攻撃・海賊行為も重要業務のひとつ
※[4 グロティウス] …『5 海洋自由論』を著し、東インド会社を擁護。のちパリに亡命
○オランダ (1621-1674) … 西インド諸島を含むアメリカ全体を管轄
・業務内容 … 東インド会社と同じく、独立戦争の一環としてスペイン船へ攻撃
・ニューネーデルラント … 北米に建設した植民地(1624)。中心地[6 ニューアムステルダム]
○オランダ東インド会社の勢力拡大
・ジャワ島進出 … 島西部のバンテン王国のジャカルタを占領(1619)。[7 バタヴィア]に改称
→ アジア交易の拠点とし、ここから「オランダ領東インド」を形成していく
⇒ 18 世紀にはジャワ中部～東部のマタラム王国を屈服させ、ジャワ全島を支配
・[8 モルッカ諸島]の支配 … 当初はイギリスとの共同管理が試みられた
→ [9 アンボイナ事件](1623)で[10 イギリス]排除。このあと英はインド支配に注力
・アジアでの中継貿易 … 中国の絹・生糸と日本の銀を交換。[11 日本]の鎖国後も交易継続
・[12 台湾]占領(1624) … 明の支配が及んでいなかった。ゼーランディア城を建設
→ 明の滅亡後、遺臣の[13 鄭成功]が奪還(1661)。康熙帝のとき清の支配下となる
・ポルトガルからの拠点奪取 … [14 マラッカ](1641)・[15 セイロン](1658)など
・[16 ケープ植民地] … アフリカ南端。15 世紀末にポルトガルが入植するも維持できず放棄
→ オランダが1652 年にケープタウン建設。このときの入植者が[17 ブール(ボーア)人]
・[18 タスマン] … 東インド会社の探検家。17 世紀半ばにオーストラリアとその周辺を探索
→ 大陸の東方のニュージーランド、南方のタスマニア島を発見
