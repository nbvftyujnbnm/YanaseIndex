(3) ギリシアの独立運動
①独立運動のおこり 古代ギリシア文化を、ヨーロッパの文化の源流として尊重・理想化するもの
〇[1 ギリシア独立戦争](1821-29)の勃発 … ギリシア人青年が主導。仏革命の影響も
〇ギリシア愛護主義(フィルヘレニズム) … 19 世紀前半、ロマン主義全盛のヨーロッパでおこる
・[2 バイロン] … イギリス詩人。義勇兵としてギリシア独立を支援(1824、現地で病死)
・[3 ドラクロワ] … 独立戦争を題材に『キオス島の虐殺』(1824)。ロマン主義の代表的絵画
②戦争の経過 1805 年、オスマン帝国から任命された このときデカブリストの乱
〇オスマン帝国による抑圧 … 支配下のエジプト総督[4 ムハンマド＝アリー]の協力を得る
→ コンスタンティノープル総主教を殺害。メッテルニヒや神聖同盟も鎮圧側に回る
〇[5 ロシア]の動き … アレクサンドル１世が没すると、ニコライ１世が即位(1825)
→ 南下政策に基づき先帝の方針を転換、ギリシア独立の支援に回る。神聖同盟の機能停止
⇒ これをうけて[6 イギリス]・[7 フランス]もギリシア支援に転換
○[8 ナヴァリノの海戦](1827) … 英仏露連合艦隊が、エジプト・トルコ連合艦隊を破る
〇ギリシアの独立 … ウィーン体制成立後のヨーロッパで初の国境線変更。体制が大きく動揺
・アドリアノープル条約(1829) … 土が黒海北岸を露に割譲。また土がギリシアの独立も承認
・[9 ロンドン会議](1827-31) … 英仏露などが、断続的に諸問題について話し合っていた
→ 1830 年、ギリシア独立を国際的に承認。翌年、ギリシアに王政を強要
２．イギリスの覇権と自由主義
(1) パックス＝ブリタニカ 仏革命の影響で独立の機運が高まったが、トーリ党ピット内閣が抑圧
①大ブリテン＝アイルランド連合王国の成立 … 英の[10 アイルランド併合](1801)による
②イギリスの覇権 … ナポレオンの敗退とともに確立。自由主義に支えられ、他国を圧倒
〇[11 ヴィクトリア女王](位1837-1901) … 最盛期の女王。 「ヴィクトリア朝」「第二帝国」
〇[12 ロンドン万国博覧会](1851) … 世界初の万博。イギリスの繁栄を象徴した
(2) 自由主義的諸改革 トマス＝クックが団体旅行を企画 全面ガラスの水晶宮を建設
①宗教的差別の撤廃 … トーリ党ウェリントン内閣。カトリック・ピューリタン・国教徒の平等
〇[13 審査法の廃止](1828) … 制定は1673 年。非国教徒でも公職につけるようにした
〇[14 カトリック教徒解放法](1829) … 審査法廃止後もカトリック差別は残っていた
→ アイルランド人[15 オコンネル]が尽力、カトリックも公職への道が開かれた
②労働運動の解放と選挙権の拡大 ※団結禁止法(1799-1824)廃止 … 労働組合の設立が可能に
〇[16 第１回選挙法改正](1832) … 1830 年から1834 年はホイッグ党グレー内閣
・[17 腐敗選挙区]の廃止 … 過疎化により不正行為が常態化していた選挙区を廃止
・選挙権の拡大 … 商工業者など中間層にも選挙権を与えた。ブルジョワジーの政治参加が実現
〇一般工場法(1833) … 12 時間労働・９歳未満の労働禁止など。初の徹底した工場法
〇[18 チャーティスト運動] … [19 人民憲章](1838)を掲げる、労働者の政治運動
