④社会主義思想の出現
〇背景 … 資本主義の確立。工場制機械工業の成立
・[1 資本家(ブルジョワジー)]の台頭 … 工場経営者は国家経済を左右する力をもった
・[2 労働者(プロレタリアート)]階級の形成 … 厳しい生活環境におかれた
→ [3 労働組合]を形成、1830 年代以降、ヨーロッパ諸国で組織増加。労働運動を展開
〇19 世紀イギリスにおける動き
・ラッダイト運動 … 産業革命で職を失った手工業者による機械打ちこわし運動。1810 年代に最盛
・チャーティスト運動 … 人民憲章(1838)を掲げる、労働者の政治運動。1848 年を境に衰退
・[4 工場法] … 労働者保護のための法律。1802 年が最初、1833 年が本格的。ほか複数
〇[5 空想的社会主義] … 資本主義の弊害をうけ、労働者のための理想社会をめざす
・[6 ロバート＝オーウェン](英) … 資本家の努力による穏健な変革をはかる
→ アメリカで土地を購入、共産社会建設を実験するも失敗(1825)
・[7 サン＝シモン](仏) … 労使の対立を重視せず、双方の利益と福祉をめざす
→ アメリカ独立戦争(1775-83)に義勇兵として参加
・[8 フーリエ](仏) … 産業的協同組合的な理想社会(ファランジュ)実現をめざすが失敗
・[9 ルイ＝ブラン](仏) … 二月革命(1848)を主導、第二共和政における急進共和派
→ 国立作業場を設置(1848)。生産の国家統制をはかるが、穏健共和派に敗北し亡命
〇[10 無政府主義(アナーキズム)] … 国家を否定。労働組合を通じ労働者の政治参画をめざす
・[11 プルードン](仏)… 私有財産を否定、「所有とは窃盗である」。無政府主義の創始者
→ マルクスの論敵。第二帝政で危険思想として有罪判決、1858 年に亡命
・バクーニン(露)… ベルリン三月革命(1848)に参加したのち、ロシアにおいてシベリア流刑
→ 第１インターナショナルの権威主義的運営に反発、労働者個人の直接行動を重視
〇[12 科学的社会主義] … [13 マルクス]・[14 エンゲルス]によって体系化
・概要 … マルクスは従来の社会主義を「空想的」と批判
→ 資本主義社会を分析し否定、革命による社会主義社会の実現を説く
・ (独) … ヘーゲル左派の唯物論、古典派経済学の労働価値説などを吸収
→ 独自の史的唯物論をうちたて、社会主義革命を理論づけた。1883 年没
アダム＝スミスやリカードらの議論。人間の労働が価値を生み、労働の量によって商品の価値が決まる
・エンゲルス(独) … ヘーゲル左派の影響をうける。マルクスを援助、その死後には著作を出版
・[15 唯物史観(史的唯物論)] … 生産力と生産関係の矛盾からおこる階級闘争により社会が発展
→ 古代奴隷制・中世農奴制・近代資本主義を経て社会主義・共産主義に至るとした
・『16 共産党宣言』(1848) … マルクス・エンゲルスの共著。「万国の労働者よ、団結せよ」
→ 全世界の労働者の団結による政権獲得(社会主義革命)を主張
・『17 資本論』(1867) … マルクス著。資本主義を徹底的に批判、マルクス経済学の基礎
