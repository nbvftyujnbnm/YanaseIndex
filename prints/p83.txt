②[1 アッバース朝]( 750-1258) … 都[2 バグダード]
○王朝の特徴 … このころシャリーアの体系化がすすむ
・アッバース家 … ハーシム家の一族。ムハンマドの叔父アル＝アッバースの子孫たち
・中央集権の強化 … カリフは「神の使徒の代理」でなく「神の代理」とみなされるようになる
・官僚制の整備 … アラブ人・非アラブ人の平等化をはかり、[3 イラン人]官僚を多く登用
・「4 イスラーム帝国」…「アラブ帝国」たるウマイヤ朝の税制を改革
→ マワーリーのジズヤを免除。ハラージュはすべての土地所有者に課された
⇒ アラブ人と非アラブ人の区別なく、ムスリムの平等が達成された
○アブー＝アルアッバース(位750-754) … アッバース朝初代カリフ
・王朝の成立 … カリフを名乗ってウマイヤ朝を破り、「サッファーフ(殺戮者)」とよばれた
・[5 タラス河畔の戦い](751) … [6 唐]軍を破り、その捕虜から[7 製紙法が伝来]
○マンスール(位754-775) … アッバース朝２代目カリフ
・[8 バグダード]の造営 … 円形の城壁をもつ新都を建設。現イラクの首都
・後ウマイヤ朝の成立(756) … ウマイヤ朝の残存勢力がイベリア半島で独立
○[9 ハールーン＝アッラシード](位786-809) … アッバース朝５代目カリフ、全盛期
・フランク王国との関係 … 両国ともにビザンツ帝国・後ウマイヤ朝と対抗していた
→ カール大帝と使節を交換するなど、友好関係を保ったとされる
・『千夜一夜物語(アラビアン＝ナイト)』… インド・アラビアなどの説話の集大成
→ 多くはハールーン＝アッラシードのころのアッバース朝が舞台
・ギリシア語文献のアラビア語訳 … ムセイオンの文献をバグダードに移動
→ ハールーン＝アッラシードの子マームーンにより知恵の館(バイト＝アルヒクマ)が設立
○衰退 … ９世紀以降、各地のアミールが自立
・トルコ系奴隷の採用 … [10 マムルーク]とよばれ、軍人として用いられた
・ザンジュの乱(868-884) … イラク南部で、黒人奴隷を中心とした反乱がおこる
(3) イスラーム文化の特色
①特徴 … アラビア語が共通語。クルアーン(コーラン)を記す言葉として絶対的権威をもった
○多様性 … イスラーム教を中心として、多くの先行文明と融合しながら形成された
○都市性 … バグダードやカイロ(ファーティマ朝の都)などを中心として発展
○普遍性 … 民族をこえて広がり、各地で地域性がみられるようになった
→ イベリア半島ではギリシア・ローマの古代文明の色を残し、ルネサンスの原動力となった
②イスラーム固有の学問と美術 アフリカ・中国・インド・東南アジアに進出
○神学 … [11 ガザーリー](1058-1111、イラン系)がセルジューク朝で活動
→ イスラーム神秘主義(12 スーフィズム)を理論化。形式的な信仰を批判、「神との合一」
⇒ 独特な修行を行い、その実践者をスーフィーという。12 世紀には教団も形成
