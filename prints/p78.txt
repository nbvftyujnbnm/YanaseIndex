２．近代以前の海上交易
(1) 海洋社会と海の道の成立 … 陸路は紀元前から草原の道・オアシスの道が存在
①[1 季節風貿易] … 紀元前後に帆船航海が一般化。「海の道」が開かれる
○中心的商人 … 紀元前から港市を建設した[2 ギリシア人]・フェニキア人が引き続き活躍
※『エリュトゥラー海案内記』… １世紀、ローマ帝国支配下のエジプトで成立
→ インドの東西両岸や、アフリカの東岸について詳細な記述がみられる
○[3 サータヴァーハナ(アーンドラ)朝] … 紀元前後、インドのデカン高原で発展
→ ローマ帝国と季節風貿易を行い繫栄。インド洋沿岸で港市が発展
○東南アジアの港市国家 … 南シナ海・マラッカ海峡など。いずれも季節風貿易で繁栄
・扶南 … １世紀、メコン下流。港市オケオではローマ金貨などが出土
・チャンパー … ２世紀、ベトナム中部。チャム人がインド文化の影響をうけたもの
・シュリーヴィジャヤ王国 … ７世紀、スマトラ。唐の義浄がここで『南海寄帰内法伝』を著す
・シャイレーンドラ朝 … ８世紀ジャワ ※５世紀のジャワ島には東晋の法顕が立ち寄った
・マジャパヒト王国 … 13 世紀、ジャワ最後のヒンドゥー王国。元の遠征軍を利用して建国
⇒ 東南アジアには、４世紀ごろからヒンドゥー教・大乗仏教が伝播
②ムスリム商人の活躍 … ８世紀ごろから、アラブ人・ペルシア人が進出。「海の道」の発展
○ペルシア湾ルート … 初期に栄える。アッバース朝の都バグダードにつながる航路
※[4 ダウ船] … 大きな三角形の帆が特徴的な大型木造帆船。インド洋交易などで多用された
○紅海ルート … 10 世紀ごろ、ファーティマ朝の都カイロに交易の中心がうつる
○中国商人との交易 … 中国の陶磁器(宋の5 青磁・白磁など)や、東南アジアの香辛料など
→ 宋代の三大発明(6 火薬・7 羅針盤・印刷術)もイスラーム経由でヨーロッパに伝播
※「陶磁の道」… 唐～清代のすぐれた陶磁器が、海の道をわたってイスラーム圏に到達
○宗教の伝播 … イスラーム教が各地に広まる
→ ミャンマー(8 パガン朝)やスリランカには上座部仏教が伝わる
(2) ヨーロッパ商人の参入と交易の活性化
①地中海交易圏 … イタリア商人が活躍。ヨーロッパ世界では11 世紀ごろから地域間交易が活性化
○[9 東方(レヴァント)貿易] … イタリア海港都市が、ビザンツ帝国やイスラームと交易
→ とくにヴェネツィアが十字軍の東方遠征を支援。第４回十字軍で交易路を確保
○航海技術の発展 … イタリア半島は伝統的に航海技術にすぐれていた
→ 中国から伝わった羅針盤が13 世紀に実用化。大型船の建造技術も確立された
○東地中海諸都市の国際化 … キリスト教商人とムスリム商人が共住した
※プレスター＝ジョン伝説 … 12 世紀には、遥か東方にキリスト教徒がいると信じられた
→ その指導者プレスター＝ジョンが、イェルサレム奪回のためやってくるとされた
⇒ 元皇帝への使者派遣や、大航海時代のポルトガルによるアフリカ探索の背景の一つ
