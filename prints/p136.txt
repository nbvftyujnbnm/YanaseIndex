〇[1 タタール](韃靼) … 高原東部にて、北元滅亡後も王家を戴いていたモンゴル諸部族
・ダヤン＝ハン … 1487 年にハンとなり、エセン死後に再び分裂していたモンゴルを再建
・[2 アルタン＝ハン] … ダヤンの孫。朝貢貿易の再開を求め毎年のように明に侵攻・略奪
→ 1550 年には北京を包囲。1571 年に明の張居正の政策で和議、朝貢貿易を再開
※「3 北虜南倭」… 16 世紀の明を苦しめた、アルタン＝ハンの略奪と後期倭寇の被害
→ 明は、南倭については1567 年の海禁緩和で対応している。これをうけて貿易が活性化
〇チベット … モンゴル高原との位置関係は、タリム盆地をはさんで南西方面
・[4 ツォンカパ] … 戒律を重んじチベット仏教を改革、15 世紀初めに[5 黄帽派]を創始
・ソナム＝ギャムツォ … 黄帽派の僧。[2 アルタン＝ハン]の帰依をうける
→ アルタンから尊号[6 ダライ＝ラマ]を献じられ、ダライ＝ラマ３世となる
⑤滅亡 1616 年、ヌルハチが後金を建国 17 世紀からラサのポタラ宮殿に居住
〇[7 万暦帝(神宗)](14 代、位1572-1620) 軍事費の抑制
・[8 張居正] … 1567 年から内閣大学士を務め、1571 年にはアルタン＝ハンと和議
→ [9 一条鞭法]の全国化など改革をすすめ財政再建。死(1582)後は宮廷の浪費が復活
[10 墨銀・ 日本銀]の流入を背景として、16 世紀半ばから江南でみられた税制。
[11 地税(土地税)と丁税(人頭税)を銀でそれぞれ一括納入]させた。[12 徭役]が消滅
・[13 豊臣秀吉の朝鮮侵略] … 1590 年代に２回。明は朝鮮に援軍を送り、軍事費が増大
・[14 東林党] … 宦官台頭に反対する官僚勢力。1604 年に顧憲成が再興した東林書院が拠点
→ 宦官勢力は非東林派を形成。政治的混乱が深まる 宦官の魏忠賢によって1625 年に閉鎖
・西欧文化の流入 … ヨーロッパ人の渡来が増加。このころ宣教師マテオ＝リッチが活躍
〇民衆運動 … 明代後半からは、財政破綻・政治混乱により重税が課されるようになっていた
・[15 抗租] … 佃戸が小作料の軽減を求める運動。鄧茂七の乱(1448、福建省)以降みられる
・民変 … 都市の下層民による反税・反政府運動。16 世紀末から17 世紀初頭までみられる
→ 絹織物業への課税を止めさせた織傭の変(1601)・魏忠賢専横期の開読の変(1626)など
・奴変 … 明末清初(16 世紀前半あたり)に江南を中心に発生した、家内奴隷の解放を求める運動
〇崇禎帝(位1627-44) … 明朝17 代にして最後の皇帝
・政治改革 … 魏忠賢を処罰し自殺に追い込む。キリスト教徒の徐光啓を登用
・袁崇煥 … ヌルハチやホンタイジの後金軍をたびたび撃退したが、謀反の疑いで処刑(1630)
⇒ 反乱が頻発する中で、[16 李自成]の反乱軍が北京占領。崇禎帝自殺、明の滅亡( 1644)
⑥明代の社会・経済 宋代「 」。江蘇省の蘇州と、浙江省の湖州
〇穀倉地帯の移動 … 江南デルタは16 世紀にほぼ開発しつくされる 後期倭寇の原動力
・長江下流域 … 稲作から、桑・綿花などの生産へ。蘇州の絹織物など家内制手工業が発達
→ 商品作物栽培の活性化により商工業が発展。生糸は日本やヨーロッパにも輸出された
