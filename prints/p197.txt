(3) 中華民国の成立
①革命前夜 日本亡命中、宮崎滔天らの支援 清朝打倒・共和政実現など
〇[1 孫文]の登場 … ハワイで[2 華僑]を中心として[3 興中会]を結成(1894)
→ 日清戦争や義和団事件に乗じて武装蜂起するもたびたび失敗。日本で亡命生活を送る
〇革命諸団体の結成 … 長沙で湖南省出身者を中心に華興会が設立(1903)。黄興・宋教仁ら
→ 上海では光復会が設立(1904)。蔡元培・章炳麟・秋瑾らのほか、魯迅も参加していた
〇[4 中国同盟会]( 1905) … 興中会・華興会・光復会などが東京で結集。中国初の本格的政党
・[5 三民主義] … このころはあくまで反清・君主独裁打倒。1917 年には軍閥とも協同
[6 民族] … 漢・満・蒙・回・蔵の五族共和による、統一的な中華民族の形成
[7 民権] … 国民が国家を形成する権利。個人の自由を、自身でなく国家形成に使う
[8 民生] … 貧富の差の解消。とくに土地所有における格差の是正。地権の平等
・四大綱領 … 駆除韃虜・恢復中華・創立民国・平均地権。内容は三民主義に基づく
・『民報』… 機関紙を発行 租界の外国資本と取引し成長。利権回収運動を展開し民族運動と結ぶ
〇[9 鉄道国有令](1911) … 財政難の清朝に対し、英米独仏で四国借款団が結成
→ 民間鉄道を国有化し借款の担保としたため、民族資本家が反発。[10 四川暴動]が発生
②[11 辛亥革命](1911-12) 現在の台湾では双十節(国慶日)とよばれる祝日
〇[12 武昌蜂起](1911.10.10) … 第一革命(≒辛亥革命)の勃発
→ 四川暴動の鎮圧に向かった湖北の新軍が、革命派に鞍替えして挙兵したもの
〇[13 中華民国] の建国(1912.1) … 革命が全国に波及、全省のうち３分の２が独立を宣言
・孫文の帰国 … [14 臨時大総統]に選出され、[15 南京]を都とする
・[16 袁世凱]の暗躍 … 西太后の死後に失脚していたが、清朝から革命鎮圧を命じられる
→ 清朝に背き、皇帝を退位させるかわりに臨時大総統の地位を譲り受けることを孫文と密約
・[17 宣統帝(溥儀)]の退位(1912.2) … 清の滅亡 ※各地の独立：外モンゴル(1911)
→ ３月に袁世凱は臨時大総統就任、[18 北京]に遷都 チベット(1913)
・臨時約法(1912.3) … 宋教仁らが起草した暫定憲法。国会開設、臨時大総統の権限縮小
〇袁世凱の独裁 革命は強力な独裁のもと推進されるべきとし、議会政治にそもそも否定的
・[19 国民党]の結成(1912.8) … 孫文が指導者にたてられたが、実権は宋教仁
→ 選挙で大勝したが、袁世凱が宋教仁を暗殺。国民党はわずか半年で解散
・第二革命 … 袁世凱独裁に対する蜂起。失敗し、孫文は亡命先の東京で中華革命党を結成(1913)
・独裁の強化 … 袁世凱は正式に大総統に就任(1913)。翌年に臨時約法を廃止
・[20 二十一カ条要求]の受け入れ(1915) … 大戦中、日本に山東省の旧ドイツ権益を譲渡
・帝政復活宣言 … 強まる国内の反発をおさえるため
→ 国内で帝政反対の運動がおこる(第三革命)。日・英・露も帝政に反対
⇒ 袁世凱は帝政を取り消し、失意のうちに病死(1916)
