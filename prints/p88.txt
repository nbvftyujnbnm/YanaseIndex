○[1 ガーナ王国](７～12 世紀ごろ) … 詳細不明。イスラームは伝えられたが受容されず
→ サハラ以南の[2 金]・象牙と、サハラ以北の[3 塩]とを交易して栄える(塩金貿易)
○[4 マリ王国](1240-1473) … イスラーム教を受容。都[5 トンブクトゥ]が交易で栄える
→ 14 世紀には旅行家のイブン＝バットゥータが来訪している
※マンサ＝ムーサ … マリ王国最盛期の王。メッカ巡礼の途上で莫大な金を散財したことで有名
○[6 ソンガイ王国](1473-1591) … マリ王国を征服し、サハラ中央にも進出
→ 16 世紀のトンブクトゥにつくられたサンコレイ大学は、黒人最初の大学といわれる
③サハラ以南の東部アフリカ(ザンベジ川流域)
○10 世紀ごろ … ムスリム商人が来航し、ダウ船で金・象牙など交易。イスラームの受容はじまる
・海港都市の発達 … [7 マリンディ]・[8 キルワ]・モンバサなど
・[9 スワヒリ語] … この地域の商業都市で生活したムスリムはスワヒリとよばれた
→ バントゥー語(現地語)に[10 アラビア語]を取り込んだ言語が用いられるようになった
○[11 モノモタパ王国](11～19 世紀) … 金・象牙を交易。積み出し港としてソファラが発達
→ [12 ジンバブエ遺跡](「石の家」の意。現在は国号)に巨大な石造建築群が残される
④インド・東南アジア
○[13 ゴール朝](1148 ごろ～1215) … アフガニスタンにおこり、インドに進出
・成立 … ガズナ朝の領内で勢力を拡大、のち独立してガズナ朝を滅ぼす
・北インドの支配 … ラージプート時代のヒンドゥー教諸勢力を破る(1192)
・滅亡 … 末期には分裂し、北インドの奴隷王朝などが自立。ホラズム朝に滅ぼされた
○[14 デリー＝スルタン朝] … [15 デリー]に都したインドのイスラーム５王朝の総称
・[16 奴隷王朝](1206-90) … マムルークの[17 アイバク]がゴール朝から自立
※クトゥブ＝ミナール … インド最古のイスラーム建築。アイバクが建てたモスクのミナレット
・ハルジー朝(1290-1320) … トルコ系。チャガタイ＝ハン国の侵攻を耐え切る
・トゥグルク朝(1320-1413) … トルコ系。旅行家のイブン＝バットゥータが来訪している
・サイイド朝(1414-51) … トルコ系。ティムール帝国の武将が建国し、その保護を受けた
・ロディー朝(1451-1526) … デリー＝スルタン朝唯一のアフガン系王朝
→ パーニーパットの戦い(1526)でバーブル率いるムガル帝国軍によって滅ぼされる
○[18 マラッカ王国](14 世紀末～1511) … 東南アジア初の本格的なイスラーム国家
・マラッカ海峡 … マレー半島に存在し、中継貿易で栄えた
・明への朝貢 … 鄭和による遠征ののち、永楽帝に朝貢
・滅亡 … [19 ポルトガル]にマラッカを占領され滅亡。1641 年にはオランダ領となる
○[20 マタラム王国](16 世紀後半～19 世紀初め) … ジャワ中部～東部のイスラーム王国
→ マジャパヒト王国にかわり成立。中継貿易で栄えたが、[21 オランダ]の支配下となる
