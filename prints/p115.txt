②[1 フェリペ２世](位1556-98) … スペイン絶対王政の全盛。「2 日の没することなき帝国」
○カトー＝カンブレジ条約(1559) … の終結。西・仏ともに多額の借金を抱えていた
○カトリックの保護 … 新教徒・異教徒を厳しく弾圧。エラスムスの著作も禁書となる
・[3 ユグノー戦争](1562-98)への介入 … フランスのカトリック貴族を支援
・ネーデルラントにおけるカトリック強制 … [4 ゴイセン]など新教徒が多数派であった
→ [5 オランダ独立戦争]勃発(1568)。イギリスの介入もあり最終的にオランダ独立
・天正遣欧使節の歓待(1584) … 日本の少年使節が教皇グレゴリウス13 世のもとを訪れていた
○[6 ポルトガル]併合 … フェリペがポルトガル王位継承。1580 年から1640 年まで同君連合
○[7 レパントの海戦](1571) … [8 オスマン帝国]のセリム２世を破る
→ スペイン海軍は「9 無敵艦隊」と称されたが、[10 アルマダ海戦](1588)で英に敗北
○スペインの衰退
・国内経済の衰退 … 新教徒・異教徒(ユダヤ・イスラーム)には商工業者が多かった
→ カトリックを保護しそれ以外の宗教は弾圧したため、商工業者の多くが国外追放となる
・銀の供給減少 … イギリスなどの私掠船に妨害され、アメリカからの銀を輸送できず
→ 無敵艦隊もイギリスに敗れ、国外での経済活動も制限されるようになる
(3) オランダの独立と繫栄
①オランダ独立戦争(1568-1648) … 八十年戦争ともいわれる
○契機 … フェリペ２世によるカトリック強制に対し、ゴイセンなど新教徒が反発
○[1 1 オラニエ公ウィレム] … 国王から、オランダのうち３州の総督に任命されていた
→ 国王によるカトリック強制に反発、独立戦争の指導者となる。英名オレンジ公ウィリアム
○オランダの内情 … 正確にはネーデルラント。現オランダと現ベルギーの両方を含む地域
・北部７州 … 現オランダ。毛織物業者・自営農が中心で、新教徒が多かった
・ … 現ベルギー。カトリックが多く、1579 年に独立戦争から脱落
○[12 ユトレヒト同盟] … ホラント州(オランダの語源)を中心に北部７州が結成(1579)
→ ウィレムは[13 ネーデルラント連邦共和国]の独立を宣言。初代オランダ総督就任(1581)
○イギリスの介入 … スペインがなおも独立を認めないなか、ウィレムが暗殺される(1584)
・[14 エリザベス１世] … 蘭から支援を要請され、私掠船によるスペイン貿易船の妨害を強化
・[10 アルマダ海戦](1588) … 英上陸をはかった西海軍を、ドレーク率いる英海軍が阻止
○事実上の独立達成(1609) … [15 アントウェルペン]陥落(1585)やアルマダ海戦が背景
→ スペインはオランダと休戦協定を結び、独立問題から手を引いた
