③元代の文化
○特徴 … 民間では、宋代以降の庶民文化が発達
・東西交流の活発化 … 大帝国の建設と交通網の整備による
・儒教の軽視 … (のちに復活はしたものの)科挙の停止も影響。朱子学の隆盛は明代を待つ
○白話小説 … 白話とは口語のこと。 『水滸伝』『三国志演義』『西遊記』の原型が生まれる
○[1 元曲] … 民謡・雑劇のなかに、モンゴル支配への反発を秘めたもの
・『西廂記』… 王実甫作。封建社会に抵抗する自由恋愛を描く
・『琵琶記』… 高明作。上京し出世した夫に、郷里で困窮していた妻が琵琶を弾きつつ再会する
・『漢宮秋』… 馬致遠作。前漢代、匈奴の単于に嫁いだ王昭君の悲劇
○歴史書
・『十八史略』… 曾先之著の通俗史書。室町中期に日本に伝来、江戸～明治のころ広まった
・『元朝秘史』… 口承文学をもとにした元代の史書。成立の詳細な経緯は不明
○元を訪れた旅行家
・[2 マルコ＝ポーロ] … ヴェネツィア商人。大都を訪れ、フビライに17 年仕えた
→ 陸路で大都を訪れ、海路で帰国。 『3 世界の記述(東方見聞録)』を口述した
・[4 イブン＝バットゥータ] … モロッコ出身のムスリム。海路で14 世紀半ばに中国に至る
→ スマトラを経由し、泉州・福州・杭州を訪れたのち大都へ。『5 三大陸周遊記』の著者
○元を訪れたフランチェスコ派宣教師 ワールシュタットの戦いのあと。十字軍協力を要請し失敗
・[6 プラノ＝カルピニ] … 教皇インノケンティウス４世の使者。グユクに親書を提出
・[7 ルブルック] … 仏王ルイ９世の使者(十字軍失敗直後)。カラコルムでモンケに面会
・[8 モンテ＝コルヴィノ] … 教皇ニコラウス４世の手紙をもって大都を訪れる
→ 大都の大司教を務め、中国で初めて[9 カトリック]を布教
○文物・学問の往来 白磁に藍色の下絵を施した陶磁器
・モンゴルに伝来 … イスラーム世界から数学・天文学・コバルト顔料など
→ この影響で、元で[10 授時暦](11 郭守敬が作成)や染付(青花)が生まれる
・モンゴルから伝播 … 火薬・武器・印刷術・陶磁器など
→ イランやインドのミニアチュールに中国画の技法がみられるようになる
※日本への影響 … 授時暦の影響をうけて、江戸時代に渋川春海が貞享暦を作成
