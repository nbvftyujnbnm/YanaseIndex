⑤[1 雍正帝](世宗、位1722-1735) 南西部のミャオ(苗)族がとくに激しく抵抗
〇[2 キリスト教布教の全面禁止](1724) … 文化流入の中断。一部の宣教師は北京残留認可
〇改土帰流 … 元代は現地人を土司として間接統治を行ったが、明清代に中国内地化が進められる
〇[3 キャフタ条約]( 1727) … ロシアと外モンゴル・シベリア方面の国境を画定
〇[4 軍機処]の設置 … ジュンガル部平定の参謀機関として創始。皇帝の諮問機関となる
→ 国政・軍政の最高機関となり、軍機大臣が従来の内閣大学士にかわるようになる
〇[5 地丁銀(制)]の確立・全国化 … [6 丁税を地税に繰り込み、銀で一括納入]させる
⑥[7 乾隆帝](高宗、位1735-95) … 康熙帝・雍正帝に続く清の最盛期 [8 丁税]の消滅
〇ジュンガル部平定 … その支配下であった回部(ウイグル人、イスラーム)も制圧
→ 東トルキスタン(タリム盆地近辺)を、1759 年までには[9 新疆]と称して支配
〇版図の拡大 … 最大版図を現出。ミャオ族の反乱を平定、ビルマ・ベトナムも朝貢させた
〇開港場を[10 広州]のみに限定(1757) … 貿易は[11 公行](特許商人の組合)に独占させた
〇[12 理藩院]の設置 … 藩部の間接統治機関(ホンタイジが設置した蒙古衙門が源流機関)
・直轄領 … 中国東北部(満州および黒竜江以北)・中国本土・[13 台湾] 回部
・[14 藩部] … 内・外モンゴル(チャハル・ハルハ・ジュンガル部)、青海、チベット、新疆
・属国(朝貢国) … 朝鮮王朝・琉球王国・大越(ベトナム)・シャム(タイ)・ビルマ
⑦清代の文化 雍正帝がチベット東北部を切り分けてできた地域
〇イエズス会士の活動 … 順治帝のときアダム＝シャールの時憲暦を実用化
・ (南懐仁、白) … 康熙帝に仕える。三藩の乱で大砲を鋳造、天文学もよくした
・ブーヴェ(白進、仏) … 康熙帝に仕える。『康煕帝伝』でヨーロッパに清の繁栄を伝える
→ レジス(雷孝思、仏)とともに中国全土の実測地図『皇輿全覧図』を作成
・[15 カスティリオーネ](郎世寧、伊) … 雍正帝・乾隆帝に仕え、西洋画で活躍
→ 建築家でもあり、円明園(ヴェルサイユ宮殿を模したバロック風庭園)を設計
〇絵画 … 清代は宮廷画家であっても、董其昌の南宗画の流れをくむ者がほとんどであった
〇文学 明の遺民では、八大山人や石濤らが清初に活躍
・『紅楼夢』… 曹雪芹による自伝的長編小説。乾隆帝のころの社会が舞台
・『16 儒林外史』… 呉敬梓の長編小説。官吏の堕落を風刺
・『聊斎志異』… 妖怪談などの奇談を集めた、蒲松齢による短編集
・『 伝奇』… 純粋な恋愛譚として唐の玄宗と楊貴妃を描いた戯曲
・『桃花扇(伝奇)』… 明朝滅亡期を舞台とした悲劇的恋愛譚。明への追慕がこもった戯曲
〇大編纂事業 … 思想統制の一環。学者の懐柔もかねている
・『17 康煕字典』… 康熙帝が編纂させ、最も完全な漢字字書といわれている
・『18 古今図書集成』… 中国最大の百科事典とされる。康熙帝の命で編纂開始、雍正代に完成
・『19 四庫全書』… 全国の書籍を提出させてまとめたもの
