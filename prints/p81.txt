６章 イスラーム世界の形成と拡大
１．イスラーム文明の誕生
(1) イスラーム教の成立 ６世紀、[2 ビザンツ帝国]と[3 ササン朝]との抗争・衰退に
①アラビア半島の風土 よってシルク＝ロード西端が途絶。かわって交易を担うようになる
○気候 … 乾燥地帯だが、ナイル川・ティグリス川・ユーフラテス川やオアシスが存在
→ 古代より都市が発展し、農耕とともに砂漠に適応した遊牧が発達
○[1 アラブ人](ベドウィン) … セム系。多くはもと多神教で、遊牧や隊商貿易で栄えた
②ムハンマドの登場 インド洋と地中海を結ぶ陸上交易で栄えたが、貧富の差が拡大していた
○[4 メッカ] … 紅海沿岸ヒジャーズ地方の商業都市。商人部族[5 クライシュ族]が支配
○イスラーム教の成立 ※ジャーヒリーヤ(無明) … イスラーム成立以前のベドウィン社会を指す語
・[6 ムハンマド] … クライシュ族ハーシム家出身。40 歳のとき神の啓示を受ける(610 年頃)
→ ムハンマドは最後の「7 預言者」としてイスラームの布教を行った
※「8 啓典の民」… イスラームの世界観で同じ神をもつ、ユダヤ教徒・キリスト教徒のこと
・一神教 … 唯一神[9 アッラー]への絶対的な帰依とともに、人間の平等を説いた
・[10 偶像崇拝の禁止] … ムハンマドの肖像も禁止(彫刻・絵画が発展しなかった要因)
・『11 クルアーン』… イスラーム教の聖典。ムハンマドの啓示を[12 アラビア語]で記す
・『ハディース』… ムハンマドの言行録。これとコーランを[13 ウラマー](知識人)が解釈
→ ウラマーはこれらに基づき[14 シャリーア](イスラーム法)を作成、指導者となった
・六信五行 … コーランに基づく信仰と、厳格な生活規範を遵守することが求められた
六信 … アッラー・天使・啓示・預言者・来世・宿命を信じること
五行 … [15 信迎告白]・礼拝・断食・[16 喜捨]・[17 巡礼]
○イスラーム世界の建造物
・[18 モスク] … イスラームの聖殿。ドーム(円屋根)が特徴的な建築
・スーク(市場) ・キャラバンサライ(隊商宿) ・マドラサ(学院) ・ミナレット(尖塔)
○[19 ヒジュラ](聖遷、 622) … クライシュ族は偶像崇拝を禁じるイスラームを迫害した
→ ムハンマドはメッカを出る。ユダヤ人のオアシス農村であったヤスリブに移住
⇒ ヤスリブは[20 メディナ](預言者の町)と呼ばれた。このころウンマ(共同体)が形成
※[21 イスラーム暦] … 聖遷の年を元年とする[22 太陰暦](１年354 日となる)
○イスラーム勢力のアラビア半島統一
・クライシュ族の追撃 … メッカからの攻撃をすべて撃退し、ムハンマドが指導力を増す
・メッカ征服(630) … クライシュ族は戦意を失い、ムハンマドにメッカを無血開城
※[23 カーバ神殿] … メッカにあった多神教の神殿。聖なる黒石を中心に多くの偶像が存在
→ ムハンマドは偶像をすべて破壊し、黒石のみ残してイスラームの聖殿をたてた
・アラビア半島の統一(632) … アラブ人が次々と帰順。同年にムハンマドが死去
