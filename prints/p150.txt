⑤[1 恐怖政治](1793.6-94.7) … 国民公会において、ジャコバン派が独裁を行った期間
〇[2 公安委員会] … 設置は1793 年４月。６月からジャコバン派が委員を独占
→ 独裁政治の執行機関となる。ほか保安委員会(警察を管轄)・革命裁判所も設置
〇ジャコバン派の指導者たち
・マラー … バスティーユ襲撃にも参加しており、サン＝キュロットから支持されていた
→ ジロンド派支持の町娘シャルロット＝コルデーにより暗殺される(1793.7)
・ダントン … 八月十日事件を扇動し、民衆から爆発的に人気を得る
→ 一方でジロンド派とも接近しており、次第に支持を失っていった
・エベール … ジャコバン派最左派に属し、サン＝キュロットからの支持がとくに強かった
→ キリスト教を否定。[3 革命暦(共和暦)]の制定、理性の崇拝など
・[4 ロベスピエール] … ルソーの思想を理想とした。1793 年７月から公安委員会に参加
→ 反革命派を大量に粛清。ジャコバン派内部でも右派のダントン、左派のエベールを処刑
⇒ 1794 年４月からは、事実上ロベスピエール個人による独裁政治となる
※ダヴィド … フランスの古典主義画家。国民公会議員でもあり、ジャコバン派と親しかった
→ 『球戯場の誓い』『マラーの死』などを描く。政治式典の演出にも携った
〇改革の推進 … 1793 年10 月にマリー＝アントワネットが処刑されている
・[5 1793 年憲法(ジャコバン憲法)] … 共和政・一院制・普通選挙
→ 史上初の徹底的な民主的憲法であったが、結局は実施されずに終わった
・[6 封建的貢租の無償廃止] … 封建制が完全に崩壊。農奴は消滅し、自作農が創出された
→ 土地を入手し安定した農民は、しだいに保守化し革命の進行を嫌うようになる
・[7 最高価格令] … 生活必需品の価格が高騰しないように統制し、下層民の生活を保護
→ 商品・食糧を高く売りたい商工業ブルジョワ・農民の間で不満が高まる
⇒ 最高賃金も同時に統制され、ロベスピエールの代表的政策として非難を浴びた
・[8 徴兵制]の整備 … 1793 年末にヴァンデの反乱がおおよそ鎮圧。近代的国民軍の形成
→ 中世は傭兵が主流であったが、フランス革命以降は徴兵による近代的常備軍が主流に
・メートル法の採用 … 国民議会の時点で提案はされていた。実際に実施されたのは1799 年
→ 言語・教育も全国の均一化がはかられたが、具体的な成果にはつながらなかった
⇒ 国民の均一化や国民軍の形成は、国民国家の成立への道を早めた
・最高存在の祭典 … ロベスピエールはキリスト教に否定的であったが、理性の崇拝も批判
→ 革命の理念を最高存在(神)として崇拝する祭典を1794 年６月に強行。独裁への反発強まる
〇[9 テルミドールの反動](1794.7) … [10 テルミドール(熱月) ９日のクーデタ]とも
→ 穏健共和派を中心にロベスピエールを逮捕、処刑。ジャコバン派独裁・恐怖政治の終焉
※[11 1795 年憲法(共和暦三年憲法)] … 共和政・二院制・制限選挙
→ 独裁への反省がみられる。国民公会はこれを制定ののち解散、総裁政府へ移行
