アメリカの古代文明
①古代アメリカの特徴
○生活 … 紀元前３万年ごろから、モンゴロイド系の人類がベーリング海峡から渡来
→ 人力のみによる原始農耕を行い、 [1 トウモロコシ]を主食としていた
○道具の使用 … 青銅器のほか、金銀製の金属器を使用。[2 鉄器]は存在せず
→ [3 車輪]や犂、ろくろなども使用された形跡なし
○家畜 … アンデスではラクダ科の小型獣であるリャマ(ラマ)・アルパカなどが飼われた
→ 羊は飼われず。また[4 牛・ 馬]のような大型獣も存在せず(牛馬耕の不可能。羊もなし)
②メソアメリカ文明 …「メソ」は「中央」の意
○ [5 メキシコ高原]
・[6 オルメカ文明] … 紀元前1200 年ごろ成立したとされる都市文明。紀元前後まで存続
→ 巨石人頭など大型石造物が残る。ジャガー神を信仰し、石造物のモチーフにもなる
・[7 テオティワカン文明](紀元前２世紀～紀元後７世紀ごろ) …「太陽のピラミッド」が残る
・トルテカ文明 … テオティワカン文明崩壊後、各地の城塞都市がもったとされる文明
・[8 アステカ文明](14 世紀～1521) … [9 テノチティトラン]に都を建設
→ マヤ文明を継承、ピラミッド型神殿・人身御供。アステカ文字(絵文字)を使用
○[10 ユカタン半島] … ４～16Ｃごろ[11 マヤ文明]が発達。灌漑農業を営み都市国家を形成
・天文学の発達 … 火星・金星などを観測、軌道計算も行う。精密なマヤ暦も存在
→ 数学ではゼロの概念が存在し、20 進法が用いられた。マヤ文字(絵文字)も使用
・宗教 … 階段ピラミッド型の石造神殿を建設。人身御供がしばしば行われた
→ 遺跡としてはチチェン＝イッツァが世界遺産に指定されている
③アンデス文明 … 現[12 ペルー]に展開
○[13 チャビン文明] … 紀元前10 世紀ごろに成立、紀元前後まで現ペルー全域に展開
→ 金属器や石造物がみられる。ここでもジャガーがモチーフのものが存在
○各地の芸術文化 … パラカス文化・ナスカ文化(紀元後１世紀～)・モチカ文化
→ ナスカ文化は「ナスカの地上絵」が有名だが、その意味は明らかになっていない
○国家の成立 … ティアワナコ文化・ワリ文化を経て、再び普遍的文化が形成。大都市建設もすすむ
→ 北部にモチカ文化の流れをくむチムー帝国が成立したが、ケチュア人により制圧される
○[14 インカ文明](13 世紀～1533) … 都[15 クスコ]。この地出身のケチュア人が形成
・支配地域 … 15 世紀には、現ペルーのほかエクアドル・ボリビア・チリ近辺も支配
・既存のアンデス文明の継承 … 巨大石造建築・金属工芸
・太陽信仰 … 皇帝は太陽の化身として専制政治を展開。神殿・宮殿も建設される
・[16 キープ](結縄) … 文字が存在せず、紐の結び方によって数字を記録した
※[17 マチュ＝ピチュ] … クスコの北方、断崖に位置する都市。ケチュア語で「老いた蜂」
