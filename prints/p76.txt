③元朝の衰退
○漢化の進行 … 宮廷には貴族文化が形成され、財政悪化と増税を招いた
○[1 紅巾の乱](1351-66) … [2 白蓮教徒](3 弥勒下生信仰などを説く)による反乱
→ はじめ韓山童とその子韓林児が中心。反乱は全土に拡散し混乱状態に陥る
○元の滅亡 … 乱のなかで台頭した[4 朱元璋]が、江南で[5 明]を建国(1368)
→ モンゴル人は中国支配を放棄、モンゴル高原に撤退。以降は北元とよばれる
(3) 元代の社会と文化
①元の中国支配 アラブ人やトルコ人などのムスリム、ヨーロッパ出身者などさまざま
○[6 モンゴル第一主義] …「モンゴル」はモンゴル人だけでなく支配者層全体をさす
・モンゴル人(人口の1.5％) … 主要官職を独占。公用語もモンゴル語
・[7 色目人](人口の1.5％) … 西方の諸民族。高級官僚に登用されるなど優遇をうけた
・[8 漢人](人口の14％) … 金(および遼)・高麗支配下であった漢人・女真人など
・[9 南人](人口の83％) … 南宋支配下であった漢人など
○統治機構 … 官僚制による中央集権体制を採用。近衛兵は遊牧民で固めた
・中央統治 … 中書省・枢密院・御史台などをそのまま用いた
→ 尚書省は廃止。これに伴い六部は中書省の管轄下にうつされた
・行中書省 … 南宋併合後、地方行政機関として中書省から独立。各地に設置された
・ダルガチ(達魯花赤) … 地方の行政一般を監察する役職 現代中国の「省」の由来
○農村支配 … 厳しい徴税を行ったが農耕社会への介入はせず。地主による佃戸支配が残る
○文字 … モンゴル語の表記にはウイグル文字が用いられた
→ フビライが[10 パスパ(パクパ)文字] をつくらせたが、使用は宮廷内に留まる
○宗教 … 厳しい統制はなく、さまざまな宗教が信じられた
・仏教と道教 … 前者は禅宗・浄土宗など。後者は華北の全真教、江南の正一教(天師教)など
・儒教 … 初期には軽視され不振。南宋の滅亡とともに[11 科挙を廃止]
→ 官僚制維持のため江南中心に儒教が再評価。1313 年に科挙も再開
・ネストリウス派キリスト教 … 初期のイル＝ハン国で保護を受けた
・[12 チベット仏教(ラマ教)] … 13 世紀に普及し手厚い保護をうけた
②経済の発展
○交通網の整備 … [13 ジャムチ]のほか、隋代以来の大運河も補修・新築された
○商業 … 従来は都市・港・関所など取引所ごとに徴税されたが、最終取引地のみの課税に変更
・国際貿易の活性化 … [14 泉州]・杭州・広州などが発展。[15 イスラーム商人]が活躍
・[16 交鈔] … 金・銅は不足しており取引には銀を使用したが、なお不足したため紙幣を発行
→ 財政悪化とともに濫発され、インフレを招き経済が混乱した
○政府の財源 … 塩の専売による利益が八割を占めた。塩引(塩の引換券)が高額紙幣として流通
財政悪化の一因にもなった
パスパがフビライの師となり、
モンゴル人貴族層に広く
帰依をうけたことがきっかけ。
