②[1 三十年戦争]( 1618- 48) …「最初のヨーロッパ大戦」 ハプスブルク家
〇[2 ベーメン]での宗教対立 … のち神聖ローマ皇帝となる国王フェルディナントが旧教強制
→ 新教徒が反発、プラハ窓外放出事件(1618)。1620 年に新教貴族も反乱、1623 年に鎮圧された
〇新教国による介入 神聖ローマ皇帝側の傭兵軍団長
・[3 デンマーク] … 英蘭から資金援助をうけていた。国王クリスチャン４世
→ 新教徒支援を唱えドイツに進軍するも、 [4 ヴァレンシュタイン]により撃退(1629)
・[5 スウェーデン] … 仏の資金援助もあり、国王[6 グスタフ＝アドルフ]のもと参戦
→ グスタフ＝アドルフ戦死(1632)、ヴァレンシュタイン暗殺(1634)により和議
〇[7 フランス]の介入 … 旧教国だが、ハプスブルク家と対立していたため新教徒を支援
→ 宰相[8 リシュリュー]が直接派兵。これをうけてスペインが皇帝支援の動きをみせる
③戦争の終結とその影響 仏はカタルーニャの反乱やポルトガル独立を支援し、西を苦しめた
〇[9 ウェストファリア条約]( 1648) …「神聖ローマ帝国の死亡証明書」
・宗教戦争の終結 … アウグスブルクの和議を再確認。さらに帝国内の[10 カルヴァン派]公認
→ 変わらず個人の信仰の自由はなかったが、新旧両派の勢力均衡がはかられた
・フランス … ドイツから[1 1 アルザス地方]などを獲得。スペインとの和平は1659 年を待つ
・[12 スウェーデン] … フランスと共同出兵し、北ドイツの西ポンメルンなどを獲得
→ 最盛期、「バルト帝国」と称される。一方、海への出口を失い[13 ハンザ同盟]は衰退
・神聖ローマ帝国 … [14 帝国内の領邦の独立主権を承認]、それぞれ領邦国家として自立
→ 帝国は事実上解体、これ以降はオーストリアと称される。ドイツ国内の分裂状態が固定化
・イギリス … このころピューリタン革命の最中。条約に参加していない
・各国の独立承認 … 三十年戦争中にはスペインからポルトガルが独立している(1640)
→ スペインから[15 オランダ]、帝国から[16 スイス]の独立が国際的に承認された
〇三十年戦争の結果 戦渦が少なかったオーストリア国内では台頭。ドイツへの影響力はもった
・[17 ハプスブルク家]の弱体化 … かわって[18 ブルボン家]がヨーロッパの覇権を握る
・[19 主権国家体制]の成立 … 皇帝・教皇権の衰退。17～18 世紀の基本的な国際関係が形成
・ドイツの荒廃 … 傭兵による都市・農村の破壊、人口激減。統一・近代化の大幅な遅れ
→ 諸領邦の中からプロイセンが台頭。オーストリアとともにドイツの強国となる

①17 世紀までのプロイセン … [20 ユンカー]が官僚・軍人の中心となった
〇ブランデンブルク＝プロイセン公国 … 下記２国が同君連合(1618)。三十年戦争に新教側で参戦
・[21 ブランデンブルク選帝侯国] … 12 世紀に辺境伯、1356 年の金印勅書で選帝侯となる
→ 15 世紀からドイツ騎士団長(当時)の[22 ホーエンツォレルン家]が支配し強大化
・[23 プロイセン公国] … 16 世紀の宗教改革で、ホーエンツォレルン家はルター派に改宗
→ リトアニア＝ポーランド王国のヤゲウォ朝に臣従、公国として認められる(1525)
