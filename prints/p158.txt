③[1 第二共和政](1848-52)
〇[2 臨時政府]の内部対立
・穏健共和派 … ラマルティーヌが首班。中小資本家など有産市民を代表した
・急進共和派 … 労働者を代表した。社会主義者の[3 ルイ＝ブラン]らがいた
〇国内政策 … いずれも1848 年
・[4 国立作業場]の設立 … ルイ＝ブランが主導。失業者の救済施設
・奴隷制の廃止 … ナポレオンが復活させていたものを再び廃止
・選挙法改正 … 満21 歳以上の男子普通選挙の実施を宣言 鎮圧され、ルイ＝ブラン亡命
〇[5 六月暴動] … 選挙後に成立した新政権が国立作業場を閉鎖。反発したパリ労働者が暴動
〇新憲法の制定 … 1848 年11 月。男子普通選挙のほか、一院制や大統領制を規定
→ 翌月の大統領選挙では、[6 ルイ＝ナポレオン](ナポレオンの弟ルイの子)が圧勝
〇第二帝政へ … 大統領は1851 年にクーデタを決行、議会から共和派を追放。国民投票で追認
→ 翌年、さらに国民投票を経て、大統領は皇帝[7 ナポレオン３世]として即位
４．東方問題とロシア ロシアは[9 南下政策]。イギリスはインドとの接続安定を目論む
(1) [8 東方問題] … 19 世紀、オスマン帝国の衰退を背景とする諸外国の介入・国際問題
①[10 ギリシア独立戦争](1821-31) … 英仏露３国がギリシアを支援
〇[11 ナヴァリノの海戦](1827) … ３国が協同してエジプト・トルコ連合軍を撃破
〇アドリアノープル条約(1829) … ロシアがトルコから黒海北岸を獲得。ギリシア独立
〇ロンドン会議(1827-31) … ギリシア独立承認。バルカン半島の利害をめぐり諸国の対立が明確化
②[12 エジプト＝トルコ戦争]
〇第１次(1831-33) … [13 ムハンマド＝アリー]がオスマン帝国からの正式独立を要求
・キュタヒヤ条約(1833) … トルコがロシアの支援をうけたため、英仏はロシアの進出を警戒
→ 英仏がエジプトとトルコを和解させる。トルコはエジプトにシリアの統治権などを割譲
・[14 ウンキャル＝スケレッシ条約](1833) … 援助を得た代償に、トルコがロシアと締結
→ ロシア軍艦に、[15 ボスフォラス・ダーダネルス両海峡の自由通行権]を認める
⇒ ロシアが黒海からエーゲ海に出られるようになったため、英仏は反発
〇第２次(1839-40) … トルコは[16 ロシア]、エジプトは[17 フランス]と結び再び戦争
→ イギリスがエジプト・フランスの勢力拡大を恐れ、トルコ支援に転換。エジプトを破る
〇戦争の結果
・[18 ロンドン会議](1840) … エジプトの領土をエジプト・スーダンに限定。シリアは返還
→ 一方で太守の世襲を承認。エジプト独立と引き換えに領土的押し込めをはかった形
・五国海峡協定(1841) … 英仏普墺露が東方問題の妥協をはかり、海峡封鎖の原則を確認
→ ウンキャル＝スケレッシ条約を破棄、両海峡を中立化(各国軍艦の通行を禁止)
※パーマストン … 英外相。自由貿易の理念に基づく積極外交を展開。ロンドン会議も主導した
