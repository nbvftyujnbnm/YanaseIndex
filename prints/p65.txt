○[1 冊封体制] … 漢代からはじまる、中国と周辺地域が形成した国際秩序を指す語
・内容 … 周辺国家が中国に遣使し貢ぎ物を献上する(朝貢)。このとき返礼品が下賜される
→ 中国は朝貢する者(周辺国家の長)を家臣と認め、見返りとして官職などを授ける(冊封)
・唐に朝貢し冊封を受けた国 … 渤海・新羅・南詔 ※朝貢貿易… 献上品と下賜品の交換
・唐に朝貢した国 … 日本・チャンパー・アンコール朝・シュリーヴィジャヤ王国
・その他 … 吐蕃(同盟関係)、ヴァルダナ朝(単に遣使)、ウイグル(安史の乱で唐を援助)
○社会と経済 … 隋代に完成した大運河によって、江南への移住も進んだ
・長安の発展 … 都城制を採用。諸外国からの商人・留学生が集った国際都市。人口最大130 万人
城壁、碁盤の目のような街路、北に宮城・中央の南北に朱雀大路。日本・渤海でも採用された
・農業 … 華北では小麦(粟との二毛作)、華中・華南では茶(および稲・綿花)の栽培が増加
・手工業 … 窯業・染織・漆器などが発達 唐代後半には茶を飲む習慣が普及
※[2 唐三彩] … 長安・洛陽付近の貴族文化を背景とした、低温による焼物
→ 人型や器型など造形はさまざま。緑・褐色・白で彩色した。副葬品に使用
・市の発展 … 都市の城壁の内で開かれ、政府は営業時間や売買品目などの市制を厳しく定めた
※飛銭 … 遠隔地取引で用いられた為替手形。中国初の手形だが紙幣として流通するには至らず
→ 唐代には開元通宝などの銅銭も鋳造されたが、流通量は多くはなかった
・陸上交通 … 絹の道ではイラン系[3 ソグド人]やトルコ系ウイグル人が活躍
→ 経済の中心地が江南に移ると、大運河に近い洛陽が国際商業都市となっていった
・海上交通 … 海の道ではアラブ系・イラン系のムスリム商人が活躍
→ 揚州・[4 広州]など華中・華南の海港が栄えた
※[5 市舶司] … 唐宋代、輸入品の販売認可や徴税を行った役所。８世紀の広州が最初

○学問書
・『水経注』… 北魏の酈道元が、黄河・長江など水系ごとの地理をまとめた
・『斉民要術』… 北魏の賈思勰が著した、中国最古の農耕技術書
・ 『傷寒論』… 西晋の王叔和が、後漢の張仲景による医学書を編纂
・『後漢書』… 宋(南朝)の范曄による歴史書
・『三国志』… 西晋の陳寿による歴史書
・『宋書』… 梁の沈約による宋(南朝)の歴史書
○道教 … 後漢末の太平道・五斗米道から発達。農民に普及し、しばしば農民反乱の支柱となる
・老荘思想の広まり … 政治から離れ、文化に親しんで物事を論じる[6 清談]が流行
※[7 竹林の七賢] … ３世紀、魏から西晋の時代、清談に明け暮れた阮籍ら７名のこと
・[8 寇謙之] … 北魏で新天師道の教団を組織し、道教を大成。太武帝の保護を受ける
