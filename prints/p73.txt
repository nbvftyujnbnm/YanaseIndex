④宋代の文化 神宗代の、程顥・程頤・張横渠が理論を拡大 君臣の別、父子の別など
○儒学 … 宋学(朱子学)が、北宋の周敦頤を祖とし、南宋の[1 朱熹]により大成される
・意義 … 訓詁学を批判する形で登場。儒教思想を世界観・哲学・実践倫理まで拡大した
・内容 … 格物致知・理気二元論・[2 性即理]・[3 大義名分論]・[4 華夷の別]
・[5 四書]の重視 … 五経のほかに、『大学』『中庸』『論語』『孟子』を重要経典とした
※[6 陸九淵](陸象山) … 儒学者。心即理を説き、朱熹と論争。明代の陽明学につながる
○仏教 … [7 禅宗]が士大夫に、[8 浄土宗]が民衆に流行。いずれも成立は唐代
○道教 … 金の王重陽が[9 全真教]をひらく。華北で道教の主流となる
○文学 唐宋八大家：韓愈・柳宗元(唐代)、欧陽脩・蘇洵・蘇軾・蘇轍・王安石・曾鞏(宋代)
・散文 … 古文の復興、力強い文体。蘇軾(蘇東坡)「赤壁の賦」
・詞 … 韻文の一種、唐代に楽曲の歌詞として成立した歌唱文学。宋代に宋詩として発達
○史書
・『資治通鑑』… [10 司馬光]が、戦国～五代の中国通史を編年体で著す
・『新唐書』… [11 欧陽脩]が著した唐の一代記
○絵画
・院体画 … 宮廷の画院で発達した画風。徽宗「桃鳩図」など。明末でいう北宗画
・文人画… 絵画を専門としない士大夫による画風。とくに水墨画が流行。明末でいう南宗画
○工芸品 … 景徳鎮の成立により窯業が発展。[12 青磁・白磁]が生産された
○宋の三大発明 … イスラーム世界を経由してヨーロッパに伝わる
・[13 火薬]の製造 … 唐代に黒色火薬が発明され、宋代に火筒が実用化された
・[14 羅針盤]の実用化 … 戦国末期には磁針が南北を指すことが知られていたとされる
→ 漢代にも司南とよばれる方位磁針が存在
・[15 木版印刷]の普及 … 隋唐代に発明されたとされ、宋代には大都市に民間出版社が存在
→ 大蔵経(一切経)の印刷がすすみ、このころの仏典は宋版とよばれる
※北宋の畢昇（ひっしょう）が活字印刷を発明したとされるが、木版印刷ほど普及せず
(5) アジアにおける「国風文化」の動き
①朝鮮半島 … [16 高麗]( 918-1392)が成立
○建国 … [17 王建]が新羅・後百済を滅ぼし半島を統一。都[18 開城(開京)]
○外政 … キタイ帝国(遼)や金の侵入を退け、五代・北宋から冊封をうけた
→ モンゴル帝国のフビライ＝ハンの遠征(1259)をうけ、属国とされる
※三別抄の乱(1270) … 元の支配に反発し、済州島で高麗人が反乱をおこすも失敗
○内政 … 宋の制度にならい、三省六部・郡県制・[19 科挙]を導入
→ 文班と武班とよばれる特権的な官僚階級が形成され、あわせて[20 両班]とよばれた
⇒ 両班は朝鮮王朝(李朝)において特権階級として確立し、支配者的な貴族層となった
