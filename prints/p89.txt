７章 ヨーロッパ世界の形成
１．地中海北方へ広がるキリスト教
(1) ヨーロッパの風土と人々 … 北東はウラル山脈、北西は大西洋、南は地中海に囲まれる
①南ヨーロッパ … アルプス山脈以南。イベリア・イタリア・バルカン・アナトリア半島
→ 地中海性気候。夏は乾燥し、冬は降水量が多い。ギリシア・ローマ文明が生まれた地
②西北ヨーロッパ … アルプス山脈以北。イベリア半島西北部やグレートブリテン島など
→ 大西洋の暖流と偏西風の影響をうけ、高緯度にも関わらず比較的温暖
③東北ヨーロッパ … アルプス以北、おおよそエルベ川以東。たびたび東方から騎馬遊牧民が侵入
→ 亜寒帯気候。とくに内陸は寒さが厳しいが、黒海北岸は肥沃な黒土に恵まれる
(2) ゲルマン人の大移動
①[1 ケルト人] … 東南ドイツが原住地の印欧語族。ローマからはガリア人とよばれた
○拡大 … 前６世紀ごろまでに西北ヨーロッパで小部族国家群を形成。独自の文化が発達
→ 多神教の神話体系をもち、現在もその文化はアイルランドや北フランスの一部に残る
○弱体化 … ゲルマン人の圧迫をうけたのち、カエサルなどローマ帝国に征服される
②[2 ゲルマン人] … [3 バルト海]沿岸が原住地の印欧語族。農耕・牧畜を営む
○部族国家(キヴィタス) … 貴族・平民など自由人による総会(民会)で意思決定。奴隷も存在した
○拡大 … ケルト人を圧迫、[4 ライン川]東部に至る。前５世紀ごろからローマとも接触
→ たびたびローマ領ガリア(フランス～北イタリア)に侵入し、ローマ帝国の強敵であり続けた
※トイトブルクの戦い(後９) … ゲルマニアに進出しようとしたアウグストゥスを破る
○後３～４世紀 … 一部が平和的にローマ帝国内に移住し、傭兵やコロヌスとなる者もいた
→ ローマ帝国北辺では、ゲルマン人からの防衛をゲルマン人傭兵に依存するようになった

③ゲルマン人の大移動
○背景 … 寒冷化・人口増加で耕地不足
・４世紀後半、[9 フン人]がヴォルガ川をこえ西進
→ 東ゴート人(黒海北岸のゲルマン人の一派)を征服し、隣接する[10 西ゴート人]も圧迫
・ゲルマン人諸部族の移動 … ５世紀以降が主。西ゴート人の移動開始( 375)が最初
○フン人 … アジア系騎馬遊牧民。匈奴と同系ともいわれるが詳細不明
・[11 アッティラ](406 ごろ～453) … フン人首長、パンノニア(ハンガリー)に大帝国建設
・カタラウヌムの戦い(451) … 西ローマ帝国残党・西ゴート王国などの連合軍に敗れる
・北イタリア侵入(452) …「大教皇」レオ１世の説得で退却、疫病も蔓延し撤退
・アッティラの死 … 東西ローマ帝国に侵入したが急死。このあとフン人の勢力は衰える
