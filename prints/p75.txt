(2) 元朝の成立
①[1 フビライ＝ハン](位1260-94) … ５代目ハン、元朝初代皇帝としての廟号は世祖
○権力の掌握 … アリクブケと争い勝利。フビライの支配に反対した[2 ハイドゥ]も退ける
○遷都 … 国号を[3 元]に改め、現北京に新都を建設し[4 大都]と命名
⇒ 中国支配を確立したこの帝国を、大元ウルス(元朝)とよぶ
※ウルス … ある人間集団とその領域を指すモンゴル語
→ 正確には「国」とは異なり、ハンの代替わりごとに解散・再構成されるもの
○各地への遠征
・[5 大越] … ベトナム北部、当時の王朝は[6 陳朝]。一時は撃退される
・[7 南宋] … 臨安を占領し滅ぼす(1276)。厓山の戦い(1279)で残存勢力も一掃
・鎌倉幕府 … いわゆる[8 元寇](文永・弘安の役、1274・81)。撃退される
・チャンパー(占城) … ベトナム南部に遠征するも撃退される(1282・84)
・[9 パガン朝] … ビルマ初の統一王朝。1287 年に征服
・シンガサリ朝 … ジャワ島。元への服属を拒否したが、遠征軍が到着するころにはすでに自壊
→ 元軍は結果的に[10 マジャパヒト王国]の建国(1293)に協力した形となった
②モンゴル帝国の内実 … チンギス家一門が形成したいくつかのウルスがゆるやかに連合
○大元ウルス … フビライのウルス。モンゴル高原～中国を支配
→ 元の皇帝は、大ハンとして他のウルスのハンよりも高い権威をもった 都：サライ
○[11 キプチャク＝ハン国] … ジュチ家のバトゥが南ロシアから中央アジアにかけて支配
→ 被支配層はトルコ系遊牧民が中心。14 世紀前半にその影響でイスラーム化。1502 年滅亡
キエフ公国滅亡(1240)からモスクワ大公国自立(1480)まで、モンゴルの支配下(「タタールの軛」)
※クリム＝ハン国 … 15 世紀にトルコ系民族がキプチャク＝ハン国から自立
→ 民族名としてはタタールを自称。クリミア半島に勢力をもち、イスラーム教を奉じた
⇒ オスマン帝国の保護下で黒海貿易を独占。1502 年にキプチャク＝ハン国を滅ぼす
○[12 イル＝ハン国] … フラグがインダス川西岸から現イラン・イラクにかけて支配
・宗教 … 初期はネストリウス派を保護 宰相ラシード＝ウッディーンが、『 』を著す
→ 13 世紀末、指導者[13 ガザン＝ハン]のころにイラン人の影響をうけてイスラーム化
・イラン＝イスラーム文化 … 細密画(ミニアチュール)が、中国絵画をとりいれ独自に発達
・滅亡 … 14 世紀後半、[14 ティムール帝国]に併合される
○[15 チャガタイ＝ハン国] … チャガタイ(チンギスの子)が西トルキスタン一帯を支配
・衰退 … モンケと対立し敗れる。14 世紀後半にイスラーム化、内紛から東西分裂
・ティムールの台頭 … モンゴル系出身。西チャガタイ＝ハン国でトルコ系遊牧民を統合
→ 1370 年には[14 ティムール帝国]を築きあげ、チャガタイ＝ハン国を併合
※オゴタイ＝ハン国 … かつては上記の３ハン国と同列に扱われたが、近年では存在が疑問視
