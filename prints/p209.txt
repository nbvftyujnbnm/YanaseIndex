(2) ソヴィエト連邦の成立 ※首都ペテルブルク … 1914 年にドイツと開戦したためペトログラードと改称
①第二次ロシア革命 → 1924 年レニングラード、1991 年(ソ連消滅時)サンクト＝ペテルブルクに改称
〇第一次世界大戦(1914-18) … セルビアを支援、連合国側として参戦
・[1 タンネンベルクの戦い](1914) … ヒンデンブルク率いるドイツ軍に大敗。守勢となる
→ 独・墺によりポーランド・ウクライナ・白ロシア(現ベラルーシ)などを占領される
・国内の混乱 … 1916 年ごろからストライキ頻発、中央アジア諸民族の蜂起も発生
→ 多数の農民が徴兵され、食糧不足も発生。燃料も不足し厭戦の気運が高まる
〇[2 三月革命](二月革命、 1917.3)
・[3 ペトログラード]での蜂起 … 国際婦人デーにおこった「パンよこせデモ」を皇帝が弾圧
→ 軍隊がしだいにデモ側に回っていき、労働者や兵士によるソヴィエトが結成
・皇帝ニコライ２世の退位 … [4 ロマノフ朝]倒壊。ツァーリズムの終焉
〇政局の混乱
・臨時政府の成立 … 諸党派が集ったもの。立憲民主党中心。第一次世界大戦の継続を唱える
→ 労働者・農民・兵士のソヴィエトは戦争に反対。メンシェヴィキ・社会革命党が指導
⇒ 臨時政府とソヴィエト政権の二重権力状態となる
・レーニンの帰国 … チューリヒから戻り[5 四月テーゼ]を発表
・七月暴動(1917.7) … 労働者が「すべての権力をソヴィエトへ」を掲げて大規模デモ
→ 社会革命党右派の[6 ケレンスキー]が弾圧。臨時政府首相に就任し二重権力状態解消
⇒ ボリシェヴィキ指導者も弾圧。レーニンはフィンランドに亡命、トロツキーは逮捕
・コルニーロフ将軍のクーデタ(1917.9) … 継戦を主張し、ケレンスキー内閣に反乱
→ 臨時政府はこれを鎮圧できず。レーニンやトロツキーが戻り蜂起して反乱軍を撃破
〇[7 十一月革命](十月革命、1917.11) … 世界初の社会主義革命。いわゆる「ロシア革命」
・[8 ボリシェヴィキ]の武装蜂起 … ペトログラードにて。レーニンやトロツキーが主導
→ 社会革命党左派もこれに協力。ケレンスキー亡命
・臨時政府の瓦解 … ほぼ無血のうちに達成。世界初の社会主義政権樹立
②共産党一党独裁へ ※『世界を揺るがした十日間』… ジョン＝リード(米)によるルポ
〇第２回全ロシア＝ソヴィエト大会(1917) … 各地のソヴィエトを結集。事実上の国家最高機関
・[9 人民委員会議]の選出 … 事実上の政府、内閣に相当。議長レーニンが実質の首相
→ 外務人民委員トロツキー、民族人民委員スターリンなど
・「10 平和に関する布告」… レーニンが起草したものを採択
→ [11 無併合・ 無賠償・ 民族自決]の原則。即時停戦・秘密外交の否定など
⇒ 連合国側はこれを無視。ソヴィエトは秘密協定を暴露した上で単独講和へ
・「12 土地に関する布告」… 土地私有制を廃止。地主の土地を無償で没収し、農民に分配
・社会主義政策 … 生産手段の公有化。重要産業・銀行・貿易を[13 国営]化
