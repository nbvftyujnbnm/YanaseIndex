〇[1 考証学] … 清代におこった、古典の実証主義的研究を重視する儒教の学派
・黄宗羲 … 明の遺臣。清朝には仕えず『明夷待訪録』(1663)で批判
・顧炎武 … 明の遺臣、清朝に仕えず。考証学の祖とされ、死後に『日知録』が刊行(1695)
・王夫之 … 明の遺臣。『読通鑑論』などを著すが、反清思想が強く19 世紀後半まで刊行されず
・銭大昕 … 乾隆帝に仕えた清代考証学の大家。勅撰地理書『大清統一志』の編纂に寄与
・戴震 … 考証学の確立者。『四庫全書』の纂修官をつとめた
・段玉裁 … 戴震に師事。後漢の字典に注釈をつけた『説文解字注』は現代でも重要
〇[2 公洋学] …『春秋』の解釈書のひとつ公洋伝を重んじる
・魏源 …『海国図志』(1842)で富国強兵のための近代軍隊の創設・殖産興業などを説く
・康有為 … 考証学は経世致用という本来の目的を失い形式化したとして批判
→ 清末の変法運動を指導したが失敗、1898 年に日本に亡命。『孔子改制考』(1897)
(3) 東アジア世界
①[3 朝鮮王朝]( 1392-1910)
〇内政 … このころ両班が特権的社会階層として確立
・建国 … 前期倭寇に苦しむ高麗にかわり、高麗の武人[4 李成桂](太祖)が倭寇を撃退し建国
→ [5 漢城]が都とされる。1403 年に明の永楽帝から冊封をうける
・明の影響 … [6 朱子学]を官学化、科挙も実施。高麗で保護されていた仏教は排斥される
〇19 世紀以前の対日関係 … 足利義満と交易開始(1404)、対馬の宗氏を介して貿易を統制
・安土桃山時代 … [7 豊臣秀吉]の侵略をうける(壬申・丁酉の倭乱、1592-98)
→ 李舜臣の活躍で撃退するも、全土が荒廃。多くの陶工が捕虜として日本に連れ去られた
・江戸時代 … 対馬藩主となった宗氏と己酉約条を結び国交回復(1609)
→ [8 朱印船]が来訪、交易を実施。また将軍の代替わりごとに[9 朝鮮通信使]を派遣
〇文化の成熟 … 15 世紀、４代国王世宗のころ 両班は漢字を正統とし、公文書などでは使用せず
・[10 訓民正音] … 庶民にも読めるように1446 年に制定された表音文字
→ 19 世紀に民族運動と結びつき、[11 ハングル]と呼ばれるようになる
・金属活字 … 世界初の[12 銅活字]とみられる。国家事業としても多くの書物が出版された
〇女真人との関係 … ホンタイジの侵攻(丙子胡乱、1637)をうけ、清に朝貢
②日本 … 1603 年に徳川家康が征夷大将軍に就任 山田長政がシャム(アユタヤ朝)に渡航(1612)
〇朱印船貿易 … 江戸幕府初期に展開された。台湾・マカオ・東南アジアに進出、日本町の形成
〇鎖国 … 幕府が方針転換、1641 年に完成。日本の対外交易は「四つの口」に限定された
・長崎：長崎奉行が管理。明・清、オランダとの交易 ・対馬：宗氏が管理。朝鮮王朝との交易
・薩摩：島津氏が管理。琉球王国との交易 ・松前：松前氏が管理。蝦夷地との交易
※琉球王国 … 中山王尚巴志が三山を統一して成立(1429)、中継貿易で繁栄
→ 明に朝貢したが、1609 年に島津氏の侵攻をうけ薩摩藩に服属。清代まで日中両属が続く
