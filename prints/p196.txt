☆中国革命と移民
・南洋華僑 … 中国南部海岸地方出身の華僑。明末から清代にかけて東南アジア・ハワイなどへ
→ 移民先の諸地域でも社会集団を形成し影響力をもったほか、中国本国の革命運動も支えた
・苦力 … アメリカへの移民も多かった。人種差別・移民排斥運動をうけて民族運動が刺激
(2) 日本による朝鮮の植民地化
①[1 日露戦争]( 1904-05)
〇朝鮮半島をめぐる日露の対立
・大韓帝国の成立(1897) … 閔妃殺害事件(1895)のあと、朝鮮の親ロシア化が進行
・義和団事件(1900-01) … 事件後、ロシアは朝鮮半島の北方に隣接する満州に軍の駐留を続ける
・[2 日英同盟](1902) … 日本が対露開戦に向けて準備。英はこのころ南ア戦争終結直後
〇経過 … 日本が善戦。旅順を攻略し、奉天会戦や日本海海戦にも勝利
・第１次ロシア革命の勃発(1905) … ロシアは継戦困難となる。一方で日本も財源不足であった
・アメリカ大統領[3 セオドア＝ローズヴェルト]の仲介 … 日本が事前に依頼していた
〇[4 ポーツマス条約](1905.9) … 日本側全権小村寿太郎、ロシア側全権ウィッテ
・韓国、南樺太 … 前者は日本の監督・指導権を承認、後者は北緯50 度以南のサハリンを割譲
・遼東半島南部 … ロシアは関東州の租借権を日本に譲渡。翌年に清朝も了承
・東清鉄道南満州支線 … 長春～旅順間の利権を譲渡。日本は[5 南満州鉄道]を設立
〇日露戦争後の国際関係 … 日英同盟は四カ国条約の締結(1921)まで継続
→ ロシアはアジア方面の南下を放棄、日本との対立軸が消滅。日露協約(1907、10、12、16)
→ イギリスとロシアも、ドイツの世界政策をうけて協調。英露協商(1907)
②韓国併合 西アジア方面への進出(３Ｂ政策)を警戒
〇日韓議定書(1904) … 日露戦争中。大韓帝国内の日本軍の移動を承認させた
〇第１次日韓協約(1904) … 韓国の財政・外交顧問を日本政府が推薦
〇列強からの韓国支配承認
・アメリカ … 桂・タフト協定(1905)。日本は米のフィリピンへの指導権を承認
・イギリス … 第二次日英同盟協約(1905)。日本は英のインド支配に協力
・ロシア … ポーツマス条約(1905)および日露協約。アジア権益を相互に承認
○第２次日韓協約(韓国保護条約、1905) … 韓国の外交権を奪い保護国化
→ 漢城に韓国統監府を設置(1906)、初代統監伊藤博文 ※ハーグ密使事件(1907)
○第３次日韓協約(1907) … 高宗を退位させ内政権を奪う。司法権にも介入し軍隊を解散させた
→ [6 義兵運動](反日武力闘争)が活発化するが、日本は武力でこれを抑圧
※伊藤博文暗殺(1909) … ハルビン駅にて、民族運動家の安重根により暗殺
○韓国併合条約(1910) … 韓国を植民地化。地税の整理・土地調査事業を推進
→ 韓国統監府を改め、朝鮮総督府を設置(初代総督寺内正毅)
