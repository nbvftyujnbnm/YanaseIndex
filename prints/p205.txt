(4) 第一次世界大戦
①大戦の勃発 州都
〇[1 サライェヴォ]事件( 1914.6.28) … 墺皇太子夫妻がセルビア人青年により暗殺
→ オーストリアが対セルビア宣戦。諸国も参戦し、第一次世界大戦勃発(1914-18)
〇大戦の構図
・同盟国 … 三国同盟のうちドイツ・オーストリア。ほか[2 トルコ]・[3 ブルガリア]
・連合国(協商国) … 英・仏・露ほか、インド・中華民国(北京の段祺瑞政府)など計27 カ国
→ [4 日英同盟]を口実に日本も参戦、膠州湾・南洋諸島などドイツ支配地域を侵略
〇イタリアの動き … 当初は中立を宣言するも、1915 年に対墺宣戦。翌年に対独宣戦
→ ロンドン秘密条約(1915)で、「5 未回収のイタリア」を見返りに連合国側となっていた
〇北欧の情勢 … 大戦では中立
・スウェーデン … 19 世紀後半に、鉄鋼・造船業を中心に工業国として成長していた
・ノルウェー … スウェーデンとの同君連合(1814)から、1905 年に分離
・デンマーク … 19 世紀後半、普・墺に奪われた領地の回復は諦め、国土開発を進める
⇒ いずれも議会政治が発達、20 世紀初頭に女性参政権を実現。大戦後は国際連盟を重視
②戦線の展開
〇西部戦線 … 独仏国境線沿いに形成。前線には[6 塹壕]が築かれた
・ドイツ軍の進出 … 中立国[7 ベルギー]を通過(1914.8)して北フランスへ。英開戦の口実
・[8 マルヌの戦い](1914.9) … パリ急襲をはかったドイツ軍を後退させる。戦線膠着
・[9 ヴェルダンの戦い](1916) … フランスの要塞をドイツ軍が攻略しようとするも失敗
→ ２月から12 月まで続き、70 万人に及ぶ死傷者を出した。第一次大戦最大の激戦
・[10 ソンムの戦い](1916) … 北フランスで連合国が反攻に出るも、決定的な成果なし
→ ６月から11 月までの戦闘。ヴェルダンの戦いにおける独軍の戦力低下はもたらした
⇒ このときイギリスは、開発途中であった戦車を実戦投入
・パッシェンデールの戦い(1917) … 連合国がベルギーで反攻に出るも、ほぼ成果なし
→ 沼地の泥に足をとられ、ドイツ軍の猛攻撃もあり連合軍は約３カ月で40 万人以上が死傷
〇東部戦線 … 独・墺の東部に形成。主にロシアとの戦い
・[11 タンネンベルクの戦い](1914) … ドイツ領ポーランドで、ドイツ軍がロシア軍に大勝
→ ドイツの将軍はヒンデンブルク。しかし決定的打撃を与えるに至らず戦線膠着
・ガリポリの戦い(1915) … トルコ領なので東部戦線とは言わない。ムスタファ＝ケマルが活躍
→ 英仏がダーダネルス海峡入口の確保を目指すが失敗。英海軍大臣チャーチル辞任
・イギリスの中東での動き … これも東部戦線とは関係なし。「三枚舌外交」はこのころ
→ 1914 年にエジプト保護国化をトルコに通達、スエズ運河を確保した
⇒ トルコ国内でのフセインの反乱には、「アラビアのロレンス」を送り込んでいる
