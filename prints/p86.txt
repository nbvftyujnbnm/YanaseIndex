(2) トルコ人の進出と王朝建設
①トゥールーン朝(868-905) … アッバース朝のトルコ人奴隷の子がエジプト・シリアを支配
○トルコ人の西アジア流入 … ウイグル帝国滅亡(840)のころ、多くが戦争捕虜や傭兵となった
→ アッバース朝などでマムルークは軍の主力となり、ときには独自の王朝をひらいた
○滅亡 … アッバース朝に再び吸収された
②[1 カラ＝ハン朝](10 世紀半ば-12 世紀) … 中央アジア初の[2 トルコ系]イスラーム王朝
○成立 … 詳細不明。サーマーン朝を滅ぼして中央アジアに進出した
○滅亡 … 西遼により滅ぼされる。西遼にもイスラームは広まったが仏教が保護され続けていた
③[3 ガズナ朝](962-1186) … アフガニスタン、トルコ系
○成立 … サーマーン朝に仕えたマムルークが独立政権を樹立 寺院を破壊し、インド仏教に打撃
○インドへの侵入 … カイバル峠をこえてパンジャーブ地方に侵入、たびたび略奪を行った
○フィルドゥーシー … ガズナ朝の保護をうけ、叙事詩『シャー＝ナーメ』を著した
④[4 セルジューク朝](1038-1194) … スンナ派を再興し、ファイーティマ朝に対抗した
○セルジューク人 … トルコ系遊牧民。カラ＝ハン朝・ガズナ朝の成立に乗じて西方に進出
→ 自らもトルコ系であるが、他王朝と同じくマムルークを用いた
○[5 トゥグリル＝ベク](初代、位1038-63) … ガズナ朝から自立し、王朝をひらく
・[6 バグダード]入城( 1055) … アッバース朝カリフから要請され、ブワイフ朝を滅ぼす
・[7 スルタン] … 世俗指導者としての称号。カリフは宗教的権威、スルタンは政治権力
→ トゥグリル＝ベクは初めて[8 アッバース朝]のカリフからスルタンの称号を与えられた
○アルプ＝アルスラーン(２代、位1063-72) … シリアに進出し地中海に至る 十字軍の一因
→ アナトリア近郊にて、マンジケルトの戦い(1071)でビザンツ皇帝を捕虜とした
※ルーム＝セルジューク朝(1077-1308) … もとローマ帝国支配地のアナトリアに地方政権が成立
→ はじめニケーアに都したが第１回十字軍(1096)で占領され、コンヤに遷都
○マリク＝シャー(３代、位1072-92) … 全盛期
・[9 ニザーム＝アルムルク] … イラン人宰相。先代スルタンにも仕えていた
→ 各地に[10 ニザーミーヤ学院](11 マドラサの一種)を設立、スンナ派を復興
・[12 イクター制]の整備 … ブワイフ朝の制度を改良。マムルーク朝でも採用された
・ウマル＝ハイヤーム …『ルバイヤート』の著者。ジャラーリー暦を作成
○滅亡 … 12 世紀に入るとセルジューク朝は分裂し崩壊した
⑤[13 ホラズム(＝シャー国)](1077-1231) … 西トルキスタン(アム川下流域)、トルコ系
○全盛期 … セルジューク朝から独立、東西交易で繫栄。13 世紀には西遼やゴール朝を破る
○滅亡 … モンゴル帝国のチンギス＝ハンによって滅ぼされる
⑥ザンギー朝(1127-1222) … イラク北部、トルコ系。セルジューク朝から自立
→ 十字軍(第２回)に対して攻勢をとったが、アイユーブ朝に滅ぼされた
