〇新文化運動 … 思想・文化の革新運動がおこる 論文「謹告青年」にて
・[1 陳独秀] … 雑誌『2 新青年』を刊行(1915)。「民主と科学」を掲げる
→ デモクラシーとサイエンスを受容し、旧来の儒教的価値観を克服することを目指す
・[3 胡適] …『新青年』に「文学改良芻議」を掲載(1917)、白話文学を提唱
→ 文学革命のおこり。文語表現・儒教からの脱却と同義であり、新文化運動と不可分の動き
・[4 魯迅] …『新青年』で『狂人日記』を発表(1918)、白話文学の嚆矢。 『阿Ｑ正伝』(1921)
・[5 李大釗] … 1918 年に『新青年』で論文を発表。中国でのマルクス主義の先駆者
※[6 中国共産党]の成立(1921) … 陳独秀・李大釗らがコミンテルンの指導をうけ上海で結成
〇[7 五・四運動]( 1919) 京師大学堂を前身に1912 年設立。新文化運動の拠点でもあった
・パリ講和会議(1919) … 二十一カ条要求の取り消し及び山東省の利権返還要求が却下される
・運動の勃発 … 北京大学の学生デモが発生、中国全土に波及。中国はヴェルサイユ条約調印拒否
→ ワシントン会議を経て、1922 年に山東省の利権が中国に返還された
※[8 中国国民党]の結成(1919) … 五・四運動をうけて、孫文が大衆政党結成
〇[9 カラハン宣言](1919) … シベリア出兵をうけ、レーニンが中華民国との提携をはかる
→ 帝政ロシアが北京議定書(1901)などで中国から得た利権を放棄。一切の秘密条約を破棄
②国民革命と南京政府
〇中国におけるナショナリズム … 第一次世界大戦による世界的なナショナリズム高揚も影響
・資本家 … 租界で外国資本と取引し成長する一方、そのなかで民族意識を高め、民族資本家に
・労働者 … 新文化運動の影響でマルクス主義が広がる。雇用者たる外国資本との階級・民族闘争
〇[10 第１次国共合作]( 1924) … 孫文が国民党を改組、共産党員の国民党入党を認める
・「第一革命」から「国民革命」へ … 軍閥打倒による中国統一をめざす
・新三民主義…[11 連ソ・ 容共・ 扶助工農]。反帝国主義・反軍閥が主軸となっている
→ 1925 年に孫文が死去、国民党内で右派(反共)と左派(容共)との対立が激化
※[12 五・三〇事件](1925) … 上海の在華紡でストライキ。日本人監督が中国人労働者を射殺
→ 市民・労働者・学生によるデモ・大規模ストライキが波及、各国租界当局が弾圧
・[13 広州国民政府]の成立(1925) … 広東国民政府とも。主席汪兆銘(汪精衛) 左派 右派
・[14 北伐](1926-28) … 北京の軍閥打倒をはかる。国民革命軍司令官に[15 蔣介石]就任
・武漢国民政府(1927.2) … 北伐により長江流域まで進出、汪兆銘らが共産党と提携して樹立
・[16 上海クーデタ](1927.4.12) … 浙江財閥と結んだ蔣介石が、市民・労働者を弾圧
→ 上海で労働者を率いていた周恩来は脱出に成功したが、広東・北京にも反共運動が広がった
※南京事件(1927.3) … 北伐軍が迫る中で民衆が暴動、外国人が標的となり死傷者が発生
→ 諸外国は南京市民に報復を実施。上海クーデタの背景のひとつ
・[17 南京国民政府](1927.4) … 共産党を排除した政府が成立(武漢政府も存続)
→ ７月には武漢政府から共産党が離脱、汪兆銘も失脚。第１次国共合作瓦解
