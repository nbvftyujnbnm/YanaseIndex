○文学 「詩仙」とよばれた 「詩聖」とよばれた 作品『長恨歌』『白氏文集』

・伝奇小説 … 唐宋代で流行した短編小説。六朝文化のころの怪異譚である志怪小説から発展
○書道 … 唐代に、書道として楷書が完成。盛唐の顔真卿は、初唐の三大家と比べて力強い書
○絵画 欧陽詢・虞世南・褚遂良。東晋の王羲之から続く、貴族的・繊細な書
・閻立本 … 初唐の太宗のころ、宰相もつとめた画家。肖像画の様式を確立
・呉道玄 … 盛唐の玄宗のころ。人物画・仏画にすぐれ、とくに山水画の基礎を築く
・李思訓 … 唐の皇族。山水画にすぐれ、その画風は宋代の院体画のもととなったとされる
・[4 王維] … 盛唐の玄宗のころの詩人・画家。詩文でも李白や杜甫と並ぶ文筆家

①日本 ４世紀には広開土王碑から朝鮮半島への介入が窺える ５～６世紀、『宋書』倭国伝の記述
○古墳時代 … ３世紀半ばごろからヤマト王権による統一がすすむ。倭の五王が南朝に朝貢
○飛鳥時代 … ６世紀ごろから。このころ日本に仏教伝来。遣隋使および遣唐使の派遣
②朝鮮 推古天皇代(摂政:厩戸皇子)、小野妹子(607、煬帝) 舒明天皇代、犬上御田鍬(630、太宗)
○[5 高句麗](前１世紀～668) … 扶余人(ツングース系)の一支族である貊人が建国
・支配の拡大 … はじめ朝鮮半島北方に成立、209 年には丸都を都とする
→ 313 年に楽浪郡を滅ぼし、427 年に平壌に遷都、５世紀に遼東半島やその北方に勢力を拡大
・好太王(広開土王) … 最盛期の王、半島南部にも勢力を拡大
→ ４世紀末に日本軍を撃破。現中国の吉林省集安(当時の丸都付近)に広開土王碑が残る
・対隋 … [6 煬帝]の３度にわたる侵略を撃退
・対唐 … [7 太宗]の侵略を撃退するも、新羅と結んだ高宗の侵略をうけ滅亡
○三韓の動向 … 朝鮮半島南部では小国家が分立していたが、４世紀ごろから連合
・[8 馬韓] … 朝鮮半島西南部。[9 百済]が成立(346-660)
・[10 辰韓] … 朝鮮半島東南部。[11 新羅]が成立(356-935)
・弁韓 … 朝鮮半島南端、加羅諸国が分立。ヤマト王権が進出し、日本では任那とよばれた
○[11 新羅]による朝鮮半島統一 … 統一達成後の新羅の都は金城(現慶州)
・唐との連携 … 百済を660 年、高句麗を668 年に滅ぼす
→ 663 年の[12 白村江の戦い]では、新羅・唐連合が日本・百済残党連合を破った
・半島統一(676) … 唐勢力を排除(安東都護府の撤退:平壌→遼陽)。のち朝貢し冊封を受けた
・[13 骨品制] … 氏族を出自により５等級に分ける、新羅独自の身分制度が存在
→ 骨品に基づいて位階や官職などが決まったため、新羅は貴族社会であったといえる
・仏教の発展 … 慶州には石窟庵のほか[14 仏国寺](石造の多宝塔が有名)が存在
