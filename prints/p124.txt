(4) 17 世紀の危機と三角貿易の形成
①17 世紀の危機 … 16 世紀のヨーロッパは、宗教改革はあれども大航海時代の展開により繁栄
〇経済の停滞 … 過剰な人口増加や、物価の高騰など スケートが娯楽として広まる
〇疫病の発生 … ペストが14 世紀以来の大流行をみせる。1655 年のロンドンなど
〇気候の寒冷化 … イギリスではテムズ川が凍結。ロシアなど各地で波状的に凶作・飢饉
⇒ ヨーロッパ北部では小麦・ぶどうが栽培できなくなる。人口増加が停滞、地域によっては減少
〇社会の混乱
・イギリス … ピューリタン革命(1642-49)
・フランス … フロンドの乱(1648-53) スペインとは異なる独自の文化を
・ロシア … ステンカ＝ラージンの乱(1670-71) もつ地方。言語もカタルーニャ語
・スペイン … カタルーニャの反乱(1640-52)。バルセロナを中心に展開
・「魔女狩り」… 16 世紀にほぼ消滅していたが復活。17～18 世紀、ヨーロッパ全土にみられる
※セイラム魔女裁判(1692) … ニューイングランドにて。アメリカでは最後の魔女裁判
〇危機への対応 … 各国とも重商主義政策を展開した点では似通っていた
・イギリス … 名誉革命(1688-89)も経て絶対王政が終焉。商業が発展、資本主義の展開を準備
・フランス … 社会情勢を安定させるため、王権が強化。絶対王政の全盛を迎える
・ドイツ … 三十年戦争(1618-48)によって荒廃。国内の分裂状態が固定化
・オランダ … 圧倒的な経済力を背景に、危機の時代にあって唯一繁栄を続けた
②三角貿易の形成 ※17 世紀の文化 … バロック美術・科学革命が特徴的。
〇奴隷貿易の展開 ニュートン・デカルト・グロティウスら
・15 世紀半 … ポルトガルが大西洋で黒人奴隷貿易を開始。スペイン植民地でインディオが酷使
・16 世紀 … インディオの人口激減。スペイン植民地でもアフリカから黒人奴隷を輸入開始
・17 世紀 … オランダ・イギリス・フランスが奴隷貿易参入、西アフリカ海岸などに商館を設置
・18 世紀 … イギリス・フランスを中心として大西洋三角貿易が活発化。とくにイギリスが主導
港湾都市[1 リヴァプール]などが奴隷貿易の中心地として繁栄。資本蓄積、産業革命を準備
※西欧諸国による[2 プランテーション]経営 … 中南米では、食料自給ができないほど徹底
・16 世紀 … ポルトガルがブラジルでサトウキビを栽培。これを原料に[3 砂糖]生産さかん
・17 世紀 … 蘭・英・仏がアメリカ・西インド諸島進出。サトウキビ・タバコ・綿花などを栽培
→ イギリスは、[4 ジャマイカ]にて大規模プランテーション経営
〇黒人王国の繁栄 … ヨーロッパ人とアフリカ現地人で協同し奴隷狩りを行った
・ベニン王国 … 現ナイジェリア。ポルトガルの進出以前から存在、ジョアン２世とも交易
→ 17 世紀にはイギリス商人から鉄砲を購入し、大々的に奴隷狩りを実施 「奴隷海岸」
・ダホメ王国 … 現ベナン。ポルトガル、のちフランスと交易。ベニンと並ぶ奴隷供給地
・アシャンティ王国 … 現ガーナ。イギリスと交易し18～19 世紀にかけて繁栄
