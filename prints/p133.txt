〇[1 ホッブズ](英) … 主著『リヴァイアサン』(1651)
・自然状態 …「万人の万人に対する闘争」の状態であるとした
・自然権 … 自己保存の権利。個人が闘争の恐怖から逃れるために社会契約を結ぶとした
・社会契約 … 個人は統治者に自然権を譲渡し、その代償として安全を保障される
→ この時点で統治者が主権者となり、個人が統治者に抵抗することはできない
・意義 … イギリス王政復古期にあって、絶対王政を擁護する形になった
→ 王権神授説を否定し、国王の主権は社会契約によるものであるとした点が画期的
〇[2 ロック](英) … 主著『統治論二篇(統治二論、市民政府二論)』(1690)
・自然状態 … 万人は不完全ではあるが、自由・平等・独立をもつ比較的平和な状態
・自然権 … 生命・自由・財産を守る権利。これを完全にするために社会契約を結ぶ
・社会契約 … 個人が相互に同意し、自然権の一部を代表者に信託することで国家を形成
→ 主権は市民にあり、政府が信託に反した場合はこれに抵抗できる(抵抗権・革命権)
・意義 … 名誉革命を擁護。フランス啓蒙思想やアメリカ独立革命に影響
→ 代議政体による間接民主制が想定されており、国家内部での権力分立も説かれている
〇[3 ルソー](仏) … 上の２人とは異なり、フランス啓蒙思想の文脈。『社会契約論』(1762)
・自然状態 … 万人が自由・平等・正義をもつ黄金時代。人類は文明とともにこれを失ったとする
・自然権 … 自由を守る権利。現状の社会における奴隷的束縛から逃れるために社会契約
・社会契約 … 個人は自らと一体不可分の共同体を構成し、そこに自然権を譲渡
→ 共同体は一個の人格をもち、社会全体の利益を追求する一般意思に基づき政治を行う
・意義 … フランス革命に影響。直接民主制に基づく徹底した人民主権を主張
④啓蒙思想 科学革命の文脈からはデカルト、社会契約説の文脈からはロックらの影響
〇背景 … 17 世紀の科学革命をうけて、18 世紀に人間・社会・国家のあり方を見直す動きがおこる
・内容 … 学問の広範囲にわたって知的活動を展開し、改革による人間の生活の改善を追求
→ キリスト教的・封建的な世界・社会観を厳しく批判。無知を「理性の光」で照らす
・展開 … 書籍・新聞・貴族サロン・文化人サークル・コーヒーハウス・カフェなどで普及
→ 官吏・医師・法律家など中産階級が基盤。貴族の支持者は少数、民衆にもあまり普及せず
〇[4 ディドロ](仏) … フランスの数学者ダランベールとともに『百科全書』を編纂
→ 1751 年から20 年以上かけて、100 人以上の執筆者によって出版された大辞典
※百科全書派 … ディドロを中心とする、執筆に関わったフランス啓蒙主義者たちをさす語
→ ダランベールのほか、ヴォルテール・ルソー・ケネー・ビュフォンら
〇[5 ヴォルテール](仏) …『哲学書簡』(イギリス便り、1734)。フリードリヒ２世と親交
→ カトリック教会の不寛容や王政の弊害など、既存の体制を厳しく批判 サン＝スーシ宮殿
〇[6 モンテスキュー](仏) …『法の精神』(1748)。絶対王政を批判 にも招かれる
→ ロックの社会契約説を継承し、その権力分立論を三権分立として具体化
