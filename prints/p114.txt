○[1 イタリア戦争](1494-1559) … 火砲が大量に用いられた、近代的戦争の最初
・背景 … 仏王シャルル８世がナポリ王国に遠征、西王フェルナンド５世が反攻しナポリ領有
→ これ以降、断続的にフランスとスペインによるイタリアをめぐる争いが続く
・狭義のイタリア戦争 … 皇帝[2 カール５世]と仏王[3 フランソワ１世]の時期(1521-44)
→ 前者は[4 ハプスブルク家]、後者は[5 ヴァロワ家]。この期間にローマ劫略(1527)
・イギリスの参加 … 英女王メアリ１世(西王フェリペ２世の妻であった)がフランスに出兵
→ 仏王アンリ２世はこれを破り、逆にイギリス最後の大陸領カレーを奪回
・カトー＝カンブレジ条約(1559) … フェリペ２世・アンリ２世・エリザベス１世による和議
→ フランスはイタリア支配を断念したが、ハプスブルク家とヴァロワ家の対立は続く
②絶対王政の成立 … 中世から近世へ(貴族・特権都市・ギルド・教会組織などは残る)
○[6 絶対王政] … 16～18 世紀にみられる、主権国家の初期形態
→ 主権者たる国王が行政・軍事などの権力を独占。絶対君主として国家を強力に統治
⇒ 行政では[7 官僚制]、軍事では[8 常備軍]が整備され、国王の権力を支えた
○[9 王権神授説] … 国王の権力は神聖不可侵で、教皇や神聖ローマ皇帝も介入不可とする
・ジェームズ１世 … 英国王。即位前に論文「自由なる君主国の真の法」を著す(1598)
・フィルマー … 英国王チャールズ１世に仕える。『家父長権論』を著す(1680)
・ボーダン … 16 世紀後半、フランスの思想家。ユグノー戦争期に『国家論』を著す
・[10 ボシュエ] … 17 世紀に仏国王 に仕え、皇太子の教育係となる
○[11 重商主義] … 莫大な費用を要する官僚制・常備軍の財源として採用された
・[12 重金主義] … 金銀の蓄積を第一とする、重商主義の初期段階。16 世紀スペインが典型的
→ 海外からの金銀獲得につとめ、国内からの金銀流出を抑える。略奪的貿易
・[13 貿易差額主義] … 輸出を輸入より多くし、その差額による貨幣獲得につとめる
→ 国内では輸出産業の育成が求められる。輸入は高い関税をかけることで抑制
⇒ 産業革命前のイギリス毛織物産業が該当。17 世紀フランスでは[14 コルベール]が推進
※[15 産業保護主義] … 国内の産業を育成するため、国家が保護を加える
→ 遅れて産業革命を迎えた、19 世紀ドイツ・日本が典型。ドイツではリストが主張

①[16 カルロス１世](位1516-56) … 父はフェリペ１世(ハプスブルク家)、母はフアナ
○ハプスブルク家のスペイン継承 … カルロスの国王即位による。皇帝カール５世と兼位(1519)
○アメリカ征服 … アステカ帝国(1521)、インカ帝国(1533)。金や銀を獲得、重金主義の典型
○オスマン帝国による圧迫 … [17 プレヴェザの海戦](1538)でスレイマン１世に敗北
→ オスマン帝国は地中海の制海権を獲得。バルカン半島への進出を強める
○ハプスブルク家の分裂 … オーストリア＝ハプスブルク家とスペイン＝ハプスブルク家
→ ネーデルラントはスペインが継承。中継貿易・毛織物業で繁栄し、世界中の商人が集った
