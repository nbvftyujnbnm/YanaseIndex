○仏教 … 後漢代に伝来。初期は中国古典(とくに老荘思想)に基づいて解釈された(格義仏教)
・[1 仏図澄] … 中央アジアの亀茲の僧。後趙(羯の国)に来訪
・[2 鳩摩羅什] … 亀茲の僧、後秦(羌の国)に来訪
→ 漢訳仏典の編纂に活躍、大乗仏教を紹介。格義的性格が薄れる転機となった
・[3 法顕] … 東晋の僧。 陸路でインド(グプタ朝)に渡り、海路で戻る。『仏国記』を著す
・慧遠 … 東晋の僧、阿弥陀仏像の前で念仏する修行をはじめる
→ 402 年に白蓮社を結成、浄土教の祖とされる。唐代に大成され浄土宗となる
・達磨 … インドの僧、６世紀に海路で北魏の洛陽に来訪。禅宗の祖とされる
※[4 石窟寺院]の造営
・敦煌… 甘粛地方。４世紀(五胡十六国時代)～14 世紀。莫高窟などが有名
・雲崗… 平城郊外。北魏(文成帝、５世紀後半)。インドの影響(ガンーラ美術やグプ様式)
・竜門… 洛陽郊外。北魏(孝文帝～玄宗、５世紀末～６世紀初)。中国独自の様式
○六朝文化 … 呉～東晋～宋～斉～梁～陳で栄えた貴族文化。漢詩文では四六駢儷体が流行
・[5 陶潜(陶淵明)] … 東晋から宋にかけての田園詩人。『帰去来辞』『桃花源記』
・謝霊運 … 宋の詩人。東晋を擁護し宋を批判したため処刑された。『山居賦』
・[6 顧愷之] … 東晋の画家。『女史箴図』
・王羲之 … 書聖とよばれる東晋の書家。 『蘭亭序』『喪乱帖』
・昭明太子 … 梁の皇太子、『7 文選』を編纂
②唐代の文化
○儒教 … 後漢代からの訓詁学を整理する動きがおこる。下級貴族の科挙対策として再興
→ 唐代に孔穎達らが太宗の命をうけ、儒教経典注釈書『8 五経正義』を編纂
○仏教 … 隋唐代においても基本的には皇帝の保護を受けた
・新宗派の成立 … 隋代に天台宗、唐代に浄土宗・禅宗などが成立
・玄奘 … 陸路でインド(ヴァルダナ朝)まで往復、『大唐西域記』を著す
→ 高宗に願い出て、インドから持ち帰った経典を保存する大雁塔を建立(652)
・義浄 … 海路でインド(ラージプート時代)まで往復、『南海寄帰内法伝』を著す
・会昌の廃仏(845) … 18 代の武宗による。「三武一宗の法難」の一
→ 日本から唐を訪れていた円仁がその様子を『入唐求法巡礼行記』に著している
○道教 … 隋唐代において皇帝から保護される。全国に老子廟が建立された
○外来宗教
・[9 祆教] … ゾロアスター教。伝来は北魏のころ
・景教 … ネストリウス派キリスト教。７世紀前半に伝来、長安に大秦景教流行中国碑(781)
・摩尼教 … マニ教。７世紀末に伝来
・回教 … イスラーム教。ムスリム商人経由で伝来。現代中国では清真教という
