(5) ローマ帝国の衰退
①「３世紀の危機」 増税政策の一環であったが、市民の特権意識が薄れる
○[1 カラカラ帝] … アントニヌス勅令(212)で[2 帝国内の全自由民に市民権を付与]
○「３世紀の危機」 セウェルス帝(193)ないしマクシミヌス＝トラヌス帝(235)以降
・[3 軍人皇帝]時代 … 各地の軍団が擁立した皇帝が20 人以上続き、政治的・軍事的に混乱
・対外的危機 … 軍事費の増大により、貨幣の改悪と都市への課税が繰り返された
ゲルマン人 … たびたびライン川・ドナウ川を越えて帝国に侵入 エデッサの戦い(260)
ササン朝ペルシア … [4 ヴァレリアヌス帝]が[5 シャープール１世]に捕虜とされる
・国内の危機 … ガリアの農民反乱、増税による上層市民の都市からの流出、慢性的疫病の流行
→ 人口減少などによりラティフンディアが衰退。ローマ兵も各駐屯地で募集するようになる
征服戦争の終結とともに奴隷供給が減少(２～４世紀) 没落自由農民、解放奴隷、ゲルマン移住民など
○[6 コロナートゥス]の普及 … 小作人(7 コロヌス)を用いる大土地経営
→ 生産物を地主と小作人で折半。農民の隷属性が強く、中世農奴制の先駆とされる
②専制君主政へ
○[8 ディオクレティアヌス帝](位284-305) … 軍人皇帝時代を収拾、帝国の再建へ
・専制君主政(9 ドミナートゥス) … 元首政から移行。共和政の伝統を無視した個人独裁
官僚制の整備 … 元老院はローマ参事会(市役所のようなもの)となる。中央集権の強化
[10 皇帝崇拝]の強要 … 皇帝(ドミヌス)は神々の代理として神聖化
ペルシア宮廷の跪拝礼を導入。反発したキリスト教徒に史上最大の弾圧(303)
・[11 四分割統治](テトラルキア) … 国土防衛を分割、支配を効率化。自身は東方正帝に
東方正帝：小アジア・シリア・エジプト 東方副帝：バルカン半島
西方正帝：イタリア・アフリカ北岸 西方副帝：ブリタニア・ガリア・ヒスパニア
○[12 コンスタンティヌス帝] … 306 年から西方副帝。統一皇帝在位は324 年～337 年
・[13 ミラノ勅令]( 313) … キリスト教を公認 ※ゲルマン人傭兵を増員
・帝政の強化 … 皇帝権の神聖化や官僚制の強化を推進。四分割統治を終了し、帝国再統一(324)
・[14 ニケーア公会議](325) … [15 三位一体]を説く[16 アタナシウス派]を正統に
→ [17 アリウス派]は異端として追放され、ゲルマン人に広まっていった
・ソリドゥス金貨の鋳造 … 帝国内で広く流通。国際交易の発展にも寄与
→ 東ローマ帝国でも継承され、ノミスマと呼ばれるようになった オリエント風
・ビザンティウム遷都(330) … [18 コンスタンティノープル]と改称 専制政治の継承
・土地緊縛令(323) … 小作人の移動禁止。コロヌスとして身分を固定し、税収の安定をはかる
○ユリアヌス帝(位360-363) … ギリシア古典やミトラ教に傾倒、一時的に異教復活。「背教者」
※ゲルマン人の大移動(４～６世紀) … 西ゴート人の移動(375)が最初
→ 西ゴート人はドナウ川をこえてローマ帝国領に侵入。ローマ帝国の衰退がすすむ
