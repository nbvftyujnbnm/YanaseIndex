〇[1 インド大反乱]( 1857-59) … インド人傭兵(2 シパーヒー、セポイ)が蜂起
・背景 … インド人の宗教的慣習を無視。従来からの藩王国とりつぶし政策による旧支配層の反発
・経緯 … 北インド全域に拡大。デリー占領、ムガル皇帝による統治の復活を宣言
→ 東インド会社は、ヒンドゥー教徒(ムガル皇帝に反発)やシク教徒を味方につけこれを鎮圧
・ (1858) … 皇帝バハードゥール＝シャー２世を廃位
・[3 東インド会社の解散](1858) … インドを本国政府の直接統治下に。乱も翌年に完全鎮圧
②社会・経済 ※ラクシュミー＝バーイー … 乱の指導者のひとり、西部の藩王国の王妃
〇経済の変容 … 英の[4 機械綿布]が大量に流入、インド主要産業の[5 手織り綿布]が壊滅
→ 中国向けアヘンや、綿花・藍などの商品作物の栽培が主流となっていく
⇒ インドは、イギリス木綿工業の原料供給地・製品市場に転落 地主・農民ともに没落
〇社会の変容 … 過酷な地租制度や手織り綿布産業の壊滅により、自給自足的な村落社会が崩壊
③[6 インド帝国]の成立( 1877)
〇支配体制 … [7 ヴィクトリア女王]がインド皇帝を兼任(インド総督が副王を務めた)
→ とりつぶし政策をやめ、保守的な藩王国を懐柔して分割統治。直轄領も存在
〇改革の推進 … 綿花・茶・藍のプランテーションは変わらず発達
・アジア初の鉄道建設 … 輸出入品の効率的な運搬のため。電信や運河も建設、利益は本国へ
・英語教育の推進 … 統一言語がなくペルシア語などを用いていたインドで急速に普及
・大学の設立 … 植民地の行政官僚としてインド人を養成。ロンドンへの留学受け入れも
〇第１回インド[8 国民会議](1885) … ボンベイにて。英がインド人の不満解消を目的に開催
→ ナオロジー(1886 議長)らが登場、イギリスの思惑を外れてインド独立運動の中心勢力を形成
※インド＝ナショナリズムの背景 … 鉄道・英語の普及により国民の連帯が強まる
→ 大学の設立による知識人層の創出、留学による議会制度の流入、国民会議の開催など
