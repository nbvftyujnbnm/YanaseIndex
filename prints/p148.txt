(2) フランス革命のおこりと国民国家の形成
①[1 三部会](1789.5-6)
〇議員の召集 … 開催地[2 ヴェルサイユ]。定員は聖職者300 人、貴族300 人、平民600 人
〇議論の紛糾 … 第一・第二身分は身分別議決方式を、第三身分は合同議決方式(一人一票)を主張
②[3 国民議会](のちに4 憲法制定議会、1789.6-91.9) … 第三身分を中心に自立
〇[5 球戯場の誓い](1789.6.20)
・背景 … 国王ルイ16 世が保守貴族に圧迫され、国民議会に議場の使用を禁じた
・内容 … 国民議会は球戯場に移動、憲法制定まで解散しないことを宣言。三部会消滅
〇[6 バスティーユ牢獄]襲撃( 1789.7.14) の勃発
・背景 … 国王が、ネッケルを罷免のうえ国民議会を圧迫するための軍を準備。パリ市民が憤激
→ 市民らは牢獄襲撃を成功させ、自由主義貴族ラ＝ファイエットを革命軍の指揮官とした
・大恐怖(グランド＝プール) … 社会の変動を見聞きした農民たちによるパニック現象
→ 全国で暴動が発生、貴族・官吏・富豪の邸宅を襲撃。革命の原動力のひとつ
〇[7 封建的特権の廃止](1789.8.4)
・無償廃止 … 身分的隷属関係・領主裁判権など。教会の十分の一税も廃止
・有償廃止 … 土地にかかる貢租の徴収権。しかしとても貧農に買い取れる価格ではなかった
⇒ 不十分(上記の、8 封建地代が有償廃止であったこと)ながら、農奴の消滅に大きく前進
〇[9 フランス人権宣言](1789.8.26)
・起草者 … [10 ラ＝ファイエット]ら。内容にはルソーの啓蒙思想の影響がみられる
・ロックの影響 … 立憲君主政をめざし、主権在民や[11 私有財産の不可侵]などを規定
・アメリカ独立宣言の影響 … 大いにみられるが、[12 幸福追求権については規定なし]
〇[13 ヴェルサイユ行進](十月事件、1789.10) … 秋に凶作。パリで物価高騰・食糧難が発生
→ パリの女性たちが、国王一家が暮らす[13 ヴェルサイユ]に向けて徒歩で行進
⇒ 国王一家を[14 パリ]のテュイルリー宮殿に連行。国民議会もパリに移動した
〇諸制度の整備 … 国民議会では、有産市民の立場からの法整備が進められた
・教会財産の没収・国有化、度量衡の統一(メートル法の提案)
・ギルドの廃止 … 同業組合だけでなく、労働組合も禁じられた
・三色旗(トリコロール) … ラ＝ファイエット考案、左から青・白・赤。現在のフランス国旗
〇国王の権威失墜 著名な革命家だったが、国王派によって買収されていたことが死後に発覚
・自由主義貴族[15 ミラボー]の死(1791.4) … ラ＝ファイエットと同様、立憲君主政を説く
・[16 ヴァレンヌ逃亡事件](1791.6) … 国王一家がパリ脱出をはかり失敗
→ 王妃[17 マリー＝アントワネット]の実家オーストリアからの支援を期待したとされる
・[18 ピルニッツ宣言](1791.8) … 普墺２国が、フランス王政復興のため介入すると警告
〇[19 1791 年憲法]制定 … フランス初の成文憲法。立憲君主政・一院制議会・財産制限選挙
