②インド洋交易圏 … ダウ船によるムスリム商人の交易が、10～11 世紀に最盛を迎える
○ムスリム商人の活動範囲拡大 … インド洋沿岸に居住地を形成
・言語 … アラビア語・ペルシア語が商業における共通語となる
・貨幣 … ディーナール金貨が15 世紀初めまで流通
→ ７世紀にウマイヤ朝カリフのアブド＝アルマリクがダマスクスで鋳造したもの
○[1 カーリミー商人] … カイロを拠点とし、紅海ルートで活躍したムスリム商人のこと
→ アイユーブ朝・マムルーク朝の保護をうけ、地中海～インド洋の交易を担った
○東アフリカ沿岸 … ８世紀ごろから、ムスリム商人が金・象牙・奴隷などを求めて渡来
・港市の成立 … キルワ・モンバサ・マリンディなど。12 世紀にはムスリム居住地が増大
・[2 スワヒリ語] … アラビア語とアフリカ東岸の言語が混交して成立
○ (14～18 世紀) … 南インドのヒンドゥー文化を継承
→ 北インドのデリー＝スルタン朝に対抗。インド交易には参加し、馬などを得た
③南シナ海交易圏 … 宋代に中国の経済成長がすすみ、中国商人が海洋に進出
○宋代 … 唐代に市舶司が広州に設置されていた。のち泉州・杭州・明州(寧波)などにも設置
→ 景徳鎮を中心に生産された青磁・白磁が、ムスリム商人などに輸出された
※[3 ジャンク船] … 宋代に開発された。角型・折りたたみ式・三本の帆が特徴的
○元代 … 大帝国の成立と交通網の整備により東西交流が活性化
→ 泉州・杭州・広州などの港市が発展(明に衰退)。宋元代の泉州市舶司はムスリムが務めた
※南インドとの交易 … 宋代にはチョーラ朝・元代にはパーンディヤ朝が記録に残る
○明代 … 13～14 世紀の東アジアでは、[4 前期倭寇]による海賊行為が横行
・[5 洪武帝](14 世紀後半) … 倭寇対策で、私貿易を禁止し朝貢貿易に限定(海禁政策)
・[6 永楽帝](15 世紀) … [7 鄭和]の南海遠征により、朝貢貿易圏が拡大
→ 死後は海上統制が緩和。15 世紀後半～16 世紀に密貿易を中心とする後期倭寇が活発化
※[8 マラッカ王国] … 明に朝貢。海禁政策をうけムスリム・中国商人との中継貿易で繁栄
→ 鄭和の拠点にもなった。遠征終了後は東南アジアのイスラーム化の拠点となった
