(4) 統一王朝の形成
①分立期(前６～前４世紀半ば) … アーリヤ人の第２次移動後、農業が発展
○都市国家の発展 … 前６世紀ごろには、北インドで16 大国が相争ったとされる
・[1 コーサラ国] … ガンジス川中流。後期ヴェーダ時代に成立、のち強大化
・[2 マガダ国] … ガンジス川下流。前４世紀前半にコーサラ国を併合、最有力となる
※商工業の発展 … クシャトリヤやヴァイシャなど、新興階級が成長
○アレクサンドロス大王の侵入(前327-前325) … カイバル峠をこえて北西インドへ
→ インドにおける統一王朝形成の契機となる。前317 年ごろにマウリヤ朝が成立
②新思想の形成 ブラフマン(梵／宇宙の根源)とアートマン(我／個々の生命の根源)は同一とする
○[3 ウパニシャッド哲学] … 世界最古の哲学思想といわれる。「奥義書」と訳される
・背景 … 後期ヴェーダ時代の末(前７世紀ごろ)、バラモン教内部で祭式偏重を見直す動き
・思想 … 梵我一如の宇宙観を真理として自覚することで、[4 輪廻転生]の業から解脱する
○[5 仏教] 尊称：釈迦牟尼世尊＝シャーキャ(釈迦)族の聖者、ブッダ(仏陀)＝悟りを開いた者
・開祖 … [6 ガウタマ＝シッダールタ](前500 ごろ)。カピラ城(コーサラ国支配下)の王子
・教義 … 主に[7 クシャトリヤ]層の支持を受ける。マガダ国からも保護
ウパニシャッド哲学の影響 … 思索の重視、業や輪廻の思想など
→ 人生は苦(生老病死)であり、悟りに至りそこから解脱する。そのために八正道を実践
バラモン教の批判 … ヴェーダに基づく儀礼やヴァルナ制を否定。一切平等や慈悲を説く
○[8 ジャイナ教] 尊称：マハーヴィーラ(大勇)、ジナ(勝者)など
・開祖 … [9 ヴァルダマーナ](前500 ごろとされる)。バラモンの権威を否定
・教義 … [10 ヴァイシャ]層、とくに商人から支持された
→ [11 苦行]による救済や、[12 不殺生]・真実語・不盗・不淫・無所有の五戒を説く
③[13 マウリヤ朝](前317-前180 ごろ) … 都[14 パータリプトラ] ギリシア人勢力を一掃
○[15 チャンドラグプタ] … [16 マガダ国]のナンダ朝を倒し建国。西北インドにも遠征
・支配領域 … ガンジス川～デカン高原～インダス川～アフガニスタン東部。インド初の統一王朝
・統治機構 … 属州制を採用(ペルシアの影響)。官僚制・常備軍・徴税制を整備、中央集権支配
○[17 アショーカ(阿育王)](３代、全盛期。位前268-前232 ごろ) 王子マヒンダを派遣
・[18 ダルマ(法)]の理念による支配 … これを刻んだ[19 石柱碑]・磨崖碑を各地に建立
・仏教の保護 … 各地に釈迦の遺骨(仏舎利)を納める仏塔(20 ストゥーパ)を建立
→ 第３回[21 仏典結集]をパーリー語で実施。また[22 セイロン島]で布教活動
※カリンガ遠征 … 半島東北部、ベンガル湾岸。多くの戦死者をだし、悔いた王が仏教に帰依
○衰退 … インド北西部にたびたび異民族が侵入
・前２世紀ごろ … バクトリア・匈奴・大月氏 ※南印にはチョーラ朝・パーンディヤ朝
・前１世紀ごろ … サカ族(イラン系)・パルティア やチェーラ朝が後３世紀ごろまで存在
