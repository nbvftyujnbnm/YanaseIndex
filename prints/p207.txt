２．ヴェルサイユ体制とソ連の成立
(1) ヴェルサイユ体制の成立
①[1 パリ講和会議]( 1919) … 革命の最中であったロシアは招かれず
〇会議の方針 民主党 大戦末期に提唱していた
・アメリカ … 大統領[2 ウィルソン]の[3 平和原則十四カ条]が会議の基本精神
→ [4 秘密外交の廃止]、[5 海洋の自由]、平等な通商関係、[6 軍備縮小]
植民地における住民の利益尊重(7 民族自決)、国際平和機関の設立
・英仏の反発 … 英首相[8 ロイド＝ジョージ]・仏首相[9 クレマンソー]
→ 植民地支配や秘密条約にこだわり、「十四カ条」の理想の一部は実現せず
⇒ アジア・アフリカに民族自決の原則は適用されず。対独復讐の色が極めて濃い内容となる
※「スペイン風邪」… インフルエンザ。1918 年３月に米から拡大、全世界で約15 万人の死者
〇ドイツ … [10 ヴェルサイユ条約]で講和。内容は以下
・[11 国際連盟]の設立
・全海外植民地の没収 … 英・仏・日などの[12 委任統治領]とされた
・領土の割譲 … [13 アルザス・ロレーヌ]などを仏に、[14 ポーランド回廊]などを波に
→ 港市[15 ダンツィヒ]は自由都市とし、[16 ザール地方]も国際連盟が15 年間管理
・軍備の縮小 … 徴兵制を廃止、陸軍10 万・海軍１万6500 に制限
→ 潜水艦・航空機の所有も禁止。実質的に空軍の全面禁止
・[17 ラインラント非武装] … ライン川右岸にはドイツ軍配備禁止
・巨額の賠償金 … 1921 年に具体化、1320 億金マルクという天文学的な数字となる
〇オーストリア＝ハンガリー 国家主席ピウスツキ
・大戦末期 … 1918 年末に、諸民族が相次いでオーストリア＝ハプスブルク家の支配から自立
→ ポーランド共和国・[18 チェコ＝スロヴァキア]・[19 ハンガリー]など
→ セルブ＝クロアート＝スロヴェーン王国も自立(のちの[20 ユーゴスラヴィア])
・ハプスブルク帝国の崩壊 … 諸民族の自立により二重帝国は解体
→ 皇帝カール１世(フランツ＝ヨーゼフ１世の甥)退位、オーストリアの帝政倒壊
・[21 サンジェルマン条約] … 墺の講和条約。諸民族の独立承認・ドイツとの合併禁止
→「22 未回収のイタリア」(南チロル・トリエステ)などをイタリアに割譲
・[23 トリアノン条約](1920) … 洪の講和条約。ハンガリー革命(1919)により遅れた
→ 周辺諸国に領土の３分の２を割譲。マジャール人が各国で少数民族となる
※ハンガリー革命 … クン＝ベラらが共産主義革命を起こすが失敗
→ 仏支援のルーマニア軍介入を経て、反革命の国民軍司令官ホルティが権力を奪取
⇒ ハンガリー王国成立するも国王なし。ホルティが摂政として独裁(1920-44)
〇ブルガリア … [24 ヌイイ条約]。ギリシア・ユーゴへの領土割譲など
