○中国の支配
・前漢 … [1 武帝]が大軍を派遣し、[2 衛氏朝鮮]を滅ぼす(前108)
→ 半島に[3 楽浪郡]など朝鮮４郡を設置。半島北方では前１世紀に[4 高句麗]が成立
・後漢 … 光武帝は大幅に自治を認め、中国の影響力後退 ツングース系。313 年に楽浪を滅ぼす
→ ２世紀末に公孫氏(遼東の豪族)が楽浪郡を領有、のちにその南半分を帯方郡とする
※三韓 … 韓民族が１世紀から４世紀ごろに部族連合を形成。部族内でも小国家が分立
→ 半島西南・東南・南端にそれぞれ[5 馬韓]・[6 辰韓]・[7 弁韓]が存在
・魏 … ３世紀前半に公孫氏を滅ぼし、楽浪・帯方二郡を支配下とする。西晋もこれを継ぐ
②日本
○先史時代 … 約１万年前、氷河時代の終わりとともに朝鮮半島と陸続きではなくなる
・稲作の伝来 … 農耕は前1000 年紀に開始。稲作(長江下流域発祥)は前５世紀ごろに伝来
→ 前４～前３世紀に全国普及、同時期に国家形成。なお弥生時代は前３～後３世紀ごろ
・青銅器と鉄器の伝来 … 前３世紀ごろ、朝鮮半島からいずれも同時期に伝来
○中国史書における記述 … 日本は「倭」と記される
・『漢書』地理誌 … 前１世紀ごろ、倭人は百余国に分かれ、定期的に楽浪郡に遣使
・『後漢書』東夷伝 … 後１世紀、奴国の王が[8 光武帝]に遣使。「漢委奴国王」の金印
→ ２世紀前半、「倭国王帥升」が安帝(位106-125)に遣使。２世紀後半、「倭国大乱」
・『魏志』倭人伝 … ３世紀、邪馬台国の卑弥呼が帯方郡に遣使、「親魏倭王」の金印を授かる
・「空白の４世紀」… 中国史書に記述なく、朝鮮に広開土王碑が残る。ヤマト王権による統一期
・『宋書』倭国伝 … ５～６世紀、「倭の五王」が南朝に遣使
３．中央ユーラシア世界の形成と展開
(1) 中央ユーラシア世界の環境と社会 生産手段に乏しい遊牧民に協力し、その保護を受ける
①植生 … ヨーロッパとアジアをつなぐ広大な乾燥地帯。北から順に下記の３つの帯に分かれる
○森林地帯 … シベリアなど寒冷な地域を含む。狩猟・採集民が毛皮などを交易品として生活
○草原地帯 … 前９世紀ごろから遊牧民が出現。騎馬技術による軍事力をもとに広域国家を形成
○オアシス地帯 … 農耕民が定住し都市国家を形成。遠距離交易を行う
②１～２世紀の東西交流 キャラバン。商人が一団でラクダなどに荷を積み砂漠を移動する
○「草原の道」… 草原地帯を通じた交易路。イラン系・トルコ系・モンゴル系などの遊牧民が活動
○「オアシスの道」… オアシス地帯をイラン系[9 ソグド人]などが隊商を組み中継貿易で繁栄
→ ローマ帝国～パルティア(イラン高原)～クシャーナ朝(タリム盆地)～後漢(甘粛地方)を結ぶ
※「絹の道(シルク＝ロード)」… 19 世紀ドイツの地理学者リヒトホーフェンが初めて用いた言葉
→ 主にオアシスの道、時には草原の道をあわせて称する。中国の絹が取引され西方へ伝わる
○「海の道」… 造船・航海技術は未熟であったものの、さかんに交易 166 年(後漢代)
→ サータヴァーハナ朝とローマ帝国との季節風貿易や、日南郡への大秦王安敦の使者来訪など
