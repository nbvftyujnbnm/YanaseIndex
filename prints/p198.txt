→ 安徽派の段祺瑞(長江下流域)、直隷派の馮国璋(北京が中心)など

〇[2 青年トルコ革命](1908)
・エンヴェル＝パシャ … 青年トルコを指導しサロニカで挙兵。ミドハト憲法を復活させる
→ 軍事政権を樹立。翌年にはアブデュル＝ハミト２世を退位させ、立憲君主制へ
・領土の縮小 … 混乱に乗じ、自治領[3 ブルガリア]が自立を宣言(1908)
→ [4 オーストリア]は1908 年にボスニア・ヘルツェゴヴィナを併合
※セルビアの反発 … ボスニアにはセルビア人が多く居住したため。露もセルビアを支援
→ 墺露開戦が懸念されたが、露が第１次ロシア革命直後だったなどの理由により回避
・イタリア＝トルコ戦争(1911-12) … トルコの混乱と、第２次モロッコ事件(1911)に乗じて
→ イタリアはオスマン帝国から北アフリカのトリポリ・キレナイカ(現5 リビア)を獲得
・第１次バルカン戦争(1912) … アルバニアが独立を求めて蜂起、トルコは自治を認める
→ バルカン同盟(セルビア・モンテネグロ・ブルガリア・ギリシア)が対トルコ宣戦
⇒ ロシアも同盟を支援しトルコ敗北、バルカン半島の支配が崩壊。アルバニア独立
・第一次世界大戦(1914-18) … 露との対立から、同盟国側として参戦(独墺との協力)
→ 劣勢となり、スルタンが単独講和を決定(1918)。エンヴェル＝パシャは亡命
・[6 セーヴル条約](1920) … 連合国側とトルコとの講和条約。領土の大部分を割譲
→ スルタンが一身上の安全を代償に締結、国内の反発が高まる。青年トルコ政権は崩壊
〇トルコ革命(1920-23) 大戦中、ガリポリの戦い(1915)で英仏連合軍を撤退させ名声をあげていた
・[7 ムスタファ＝ケマル] … 青年トルコ革命に参加、のちトルコ国民党を指導
・ギリシア＝トルコ戦争(1919-22) … 英仏の支援をうけたギリシアがイズミルに上陸
・トルコ大国民議会 … ケマルがアンカラで開催(第１回、1920)。現在のトルコ国会
→ イスタンブルのスルタン政府を無視、アンカラ政府を樹立
⇒ 国民軍として活動を開始し、ギリシアを小アジアから撃退。1922 年にイズミルを奪回
・[8 スルタン制の廃止](1922) … オスマン＝トルコの滅亡。翌年に共和政を宣言
・[9 トルコ共和国]の成立(1923) … ケマルが初代大統領に就任、首都[10 アンカラ]
→ 1934 年にケマルは議会から「11 アタテュルク(トルコの父)」の尊称を授与される
〇トルコの世俗的近代国家化 … イスラーム圏初の世俗国家 ギリシア＝トルコ戦争の講和の一環
・[12 ローザンヌ条約](1923) … 英仏と交渉し、セーヴル条約を破棄
→ 国境を再画定し、これまで認めていたカピチュレーション・治外法権も廃止
