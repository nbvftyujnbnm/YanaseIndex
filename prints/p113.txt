③イギリス国王の動き … バラ戦争終結後、テューダー朝が続く
○[1 ヘンリ８世](位1509-47) クレメンス７世。カール５世の圧力に屈した
・離婚問題 … 婚姻無効の特別赦免を教皇に求めるが拒否される。イギリス宗教改革の開始
・[2 首長法(国王至上法)]( 1534) … イギリスの教会では、国王を唯一最高の首長とする
→ [3 イギリス国教会]の成立。修道院は廃止され、所領・財産は王室に没収された
・[4 トマス＝モア]の処刑(1535) … カトリック教義から離婚に反対。反逆罪とされる
・アイルランド国王を兼任(1541) … イングランドとアイルランドは同君連合となる
→ 国教の布教を目論んだが浸透せず、アイルランドではカトリックが多数のまま
○[5 エドワード６世](位1547-53) … 一般祈祷書を発布 教義：プロテスタント
○[6 メアリ１世](位1553-58) … カトリックに復帰、反対者を弾圧 儀礼：カトリック的
→ 夫は西王フェリペ２世で、イタリア戦争に巻き込まれ、唯一の大陸領[7 カレー]を喪失
○[8 エリザベス１世](位1558-1603) … 子をなさず、テューダー朝最後の王となる
→ [9 統一法](1559)を定め、一般祈祷書に基づき礼拝法を統一。イギリス国教会の確立
④[10 対抗宗教改革] … カトリック教会が自ら腐敗を改め、勢力回復をはかる
○[11 トリエント公会議](1545-63) … 教皇パウロ３世(ヘンリ８世を破門)が召集
・教皇の至上権を確認 … 同時に、福音主義(聖書主義)および信仰義認説も否定された
・[12 宗教裁判]や異端尋問の強化 … [13 禁書目録]も作成され、思想統制が強められた
・カトリック教会の改革 … 聖職売買や贖宥状の販売などを禁止
○[14 イエズス会] … スペイン貴族[15 イグナティウス＝ロヨラ]がパリで設立(1534)
→ 厳格な規律のもと、ローマ教皇への服従を誓う。海外で熱心な布教を行った
※[16 フランシスコ＝ザビエル] … 1549 年、日本にキリスト教を伝えたイエズス会士
○[17 魔女裁判] … 15～18 世紀。とくに宗教改革の時期、新教・旧教双方で苛烈な魔女狩り
３．スペインの盛衰とオランダの覇権
(1) 主権国家への道
①主権国家の形成 … 主権・領土・国民をもつ。統一的な国家権力が、排他的に領土内を統治
○中世封建社会における国家 … 国王も封建領主の一人にすぎず、統一的支配なし。国境も流動的
→ 11 世紀後半から商業活動が活性化。国王と都市・商人が結び、王権が伸長
※ルネサンス期のイタリア … 分裂期ゆえに都市国家間の外交は成熟。相互に使節を常駐した
○主権国家体制へ … イタリア戦争と、スレイマン１世ひきいるオスマン帝国の圧力による
・国家権力の成長 … 対外的脅威に対応するため、安定した国内からの徴税と兵士の募集が必須
・[18 勢力均衡] … バランス＝オブ＝パワー。イタリア戦争のとき生まれた概念といわれる
→ 複数国家で均衡を保ち、一国の強大化を防ぐことが、それぞれ自国の安全につながる
・主権国家体制に基づく国際秩序 … 安全を保障した上での外交使節の相互常駐など
→ 相互に主権を承認し共存するなかで、主要な慣例は国際法(条約と慣習法)となる
