１．大航海時代 15～17 世紀、ヨーロッパ諸国が世界に進出した時期
(1) 東方への関心とポルトガルの進出
①大航海時代の背景 ※イタリア商人を通じ東方の知識や造船技術・学問が伝わり、航海ブーム
○[1 香辛料]の需要 … 肉食が普及。保存用の塩漬け肉のにおい消しに[2 胡椒]など使用
○[3 オスマン帝国]の拡大 … 紅海ルートや東方貿易に打撃 ⇒ アジアとの直接交易の必要性
○スペイン・ポルトガルの海外進出 … レコンキスタの延長。潤沢な国家財政も背景のひとつ
○東方への関心と学問・技術の発展 … 造船技術や[4 羅針盤]の改良、地球儀の発明(1492)
・『5 世界の記述(東方見聞録)』… [6 マルコ＝ポーロ]著。「黄金の国」ジパングなど紹介
・[7 地球球体説] … フィレンツェの天文学者[8 トスカネリ]が提唱 ⇒ コロンブスに影響
※大型船の開発 … カラベル船(15 世紀ポルトガル) → カラック船 → ガレオン船(16 世紀スペイン)
②大航海時代の結果 … 経済活動が活性化、「繫栄の16 世紀」をもたらす
○[9 商業革命](15 世紀) … ヨーロッパからアジア・アメリカを含む世界的な商業圏の形成
・背景 … ポルトガルによるインド航路の開拓。インドの香辛料は直接リスボンなどへ
・結果 … 経済の中心地が地中海周辺から[10 大西洋]沿岸へ移動 アントウェルペンも繁栄
イタリア諸都市や南ドイツ諸都市の衰退 イベリア半島・ブリテン島・イギリス海峡の経済の活性化
○[11 価格革命](16 世紀) … ヨーロッパにおいて[12 貨幣価値が下落、 物価が上昇]
・背景 … スペインによる「新大陸」征服。[13 銀]の大量流入や、西欧の人口増加など
・結果 … 定額の貨幣地代で生活する封建貴族に決定的打撃。西欧で[14 封建制の崩壊]を促進
→ 西ヨーロッパ向けの輸出用穀物の需要が増大。東欧では[15 再版農奴制]につながる
○国際的分業体制の成立 … 西欧は発展し工業中心へ。東欧は農業中心のまま、後背地に
・農場領主制(16 グーツヘルシャフト) … エルベ川以東では大地主による農奴制が残る
→ 15 世紀以降も農民は自立しておらず、領主に支配され、賦役労働を課された
・再版農奴制 … 東欧では、領主がグーツヘルシャフトなどにおける農奴支配を強化
③ポルトガルの東方航海 現モロッコ 現セネガル、アフリカ最西端 アフリカ南端
○[17 エンリケ] …「航海王子」 。1415 年にセウタ征服、1445 年にヴェルデ岬を越える
○[18 バルトロメウ＝ディアス] … 国王ジョアン２世のもと。1488 年に[19 喜望峰]到達
○スペインとの植民地分界線 南アメリカ東部がポルトガル領に。のちブラジル領有の根拠
・[20 教皇子午線](1493) … アレクサンデル６世の調停、ヴェルデ岬の西方に境界線を示す
・[21 トルデシリャス条約](1494) … ジョアン２世が交渉、教皇子午線より西方に境界を移動
○[22 ヴァスコ＝ダ＝ガマ] … 国王マヌエル１世代。アラブ人の案内でインド洋横断( 1498)
