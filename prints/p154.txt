(2) ラテンアメリカの独立運動
①フランス領[1 ハイチ] … 1791 年から黒人奴隷の解放運動がおこる
〇フランス本国の対応 「黒いジャコバン」と称された
・国民公会 … フランス革命の精神に基づき、1794 年に奴隷制の廃止を宣言
・ナポレオン … 黒人奴隷を復活させ、現地には反乱鎮圧のため軍を派遣
〇[2 トゥサン＝ルヴェルチュール] … 黒人奴隷の子。英軍の介入を撃退、1801 年全島掌握
→ ナポレオンが派遣した軍も撃退したが、謀略により捕縛され1803 年に獄死
〇独立の達成(1804) … デサリーヌがフランス軍を撃破、世界初の黒人共和国となる
②スペイン領 … 1780 年にペルーで反乱を起こしたトゥパク＝アマルが、独立運動の先駆とされる
〇[3 シモン＝ボリバル] … ベネズエラの富裕クリオーリョ出身。南米大陸北部を中心に活動
→ 大コロンビア共和国の独立を宣言(1819)。現コロンビア・ベネズエラ・ボリビアなど解放
〇[4 サン＝マルティン] … アルゼンチンのスペイン軍人出身。南米大陸南部を中心に活動
→ アルゼンチンの独立運動に身を投じ、ペルー・チリも解放。ボリバルとは協力失敗
〇イダルゴ … メキシコの貧しいクリオーリョ出身の司祭。現独立記念日(1810.9.6)に蜂起
→ イダルゴは敗れて処刑されたが、その後に[5 メキシコ]独立(1821)
③ポルトガル領 … [6 ブラジル]の富裕クリオーリョが、本国の王子を皇帝に擁し独立を宣言
→ ブラジル帝国(1822-89)成立。大きな混乱もなく、既存の体制がおおよそ維持された
※南米大陸の住人たち
・[7 クリオーリョ] … 植民地生まれの白人。スペイン領における独立運動の中心
・メスティーソ … 白人とインディオの混血 ※ペニンスラール
・ムラート … 白人と黒人の混血 … 本国出身の白人
④諸国の独立に関する動き
〇ヨーロッパの対応
・墺 … ウィーン体制に基づき、メッテルニヒはスペインを支援。独立運動の弾圧をはかる
・米 … ５代大統領モンローが[8 モンロー宣言](1823)でヨーロッパからの干渉を牽制
→ [9 ヨーロッパとアメリカとの相互不干渉]。これ以降、アメリカ外交の基本方針となる
・英 … 外相[10 カニング]がスペインの影響力後退による市場開拓を狙い、独立運動を支援
→ スペイン立憲革命から続くイギリスの独自路線により、五国同盟は機能停止
〇独立後のラテンアメリカ諸国
・政治 … カウディーリョ(クリオーリョ軍人)独裁や、大地主による寡頭政治など。不安定
・経済 … 特定の農産物・鉱物の輸出に依存。コーヒー・砂糖・バナナなど
・国際関係 … 独立政府は存在しても、イギリスやアメリカに従属する立場となる
※諸国における奴隷制廃止 … イギリスは1833 年に奴隷制廃止。フランスは1848 年(第二共和政)に
改めて奴隷制廃止。アメリカは南北戦争後、1863 年に奴隷解放宣言
