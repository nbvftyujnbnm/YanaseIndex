③大陸の科学者
〇フェルマー(仏、1607-65) … 数学者、デカルト・パスカルと交流。「最終定理」で有名
〇ホイヘンス(蘭、1629-95) … 土星の環および衛星タイタンを発見。振り子時計を製作
〇レーヴェンフック(蘭、1632-1723) … 自作の顕微鏡で微生物を観察。赤血球の発見
〇ベンジャミン＝フランクリン(米、1706-90) … 凧あげの実験で雷が電気であることを証明
→ 避雷針の発明のほか、アメリカ独立宣言を起草した政治家の一人でもある
〇[1 リンネ](瑞典、1707-78) … 植物学者。二名法(動植物を属名と種名で表す)を確立
〇オイラー(瑞西、1707-83) … 数学の幅広い分野において膨大な功績を残す
→ フリードリヒ２世期のプロイセンや、エカチェリーナ２世期のロシアなどを拠点とした
〇ビュフォン(仏、1707-88) …『博物誌』で生物進化の可能性を指摘、進化論の先駆
〇ラグランジュ(仏、1736-1813) … 数学者だが、著書『解析力学』などその応用に功績が多い
→ マリー＝アントワネットの教師を務めた。メートル法の制定に関わった科学者のひとり
〇ラヴォアジェ(仏、1743-94) …「近代化学の父」、質量保存の法則を発見
→ 燃焼現象を酸素との結合と説明。なお「酸素」の命名は「酸を生むもの」との誤解による
⇒ フランス革命のあと、市民から過剰な徴税を行った者のひとりとして処刑された
〇ボルタ(伊、1745-1827) … 正極を銅板、負極を亜鉛版、電解液を硫酸とするボルタ電池を発明
→ 起電力・電圧などの単位ボルトは、ボルタの名をとって1874 年に命名されたもの
〇ラプラス(仏、1749-1827) … 太陽系の起源について星雲説を唱える。メートル法制定にも関わる
→ 確率論におけるベイズの定理を体系化した。決定論者であり、「ラプラスの悪魔」が有名
※ラプラスの悪魔 … 一定時点において、世界のすべての物質とその力学的状態を知る存在
→ この存在は未来の現象もすべて予測できるとされたが、20 世紀に量子力学によって否定
(4) 政治・社会思想
①自然法 … 実定法に優越する、神や自然などから与えられた永久不変の法
〇近代自然法 … 自然法は人間の理性に内在するとされた。その成文化により19 世紀に衰退
〇[2 グロティウス](蘭) … 自然法を国際関係に適用した「近代自然法の父」、「国際法の父」
・『海洋自由論』(1609) … オランダ東インド会社の進出を擁護。公海自由の原則の基礎とされる
・『戦争と平和の法』(1625) … 三十年戦争のなかで、自然法理念に基づく国際法の必要性を説く
②[3 王権神授説] … ・フィルマー(英)、ボーダン・ボシュエ(仏)ら
⇒ 絶対王政を支えたこの理論に対し、17～18 世紀の市民革命の時代に[4 社会契約説]が登場
③社会契約説 … 近代自然法の理念から、社会や国家の起源について説明
〇概要
・自然状態 … 社会が成立する以前の、無数の個人だけが存在すると仮定された状態
・自然権 … 個人が、人間として生まれた時点で先天的に保有する権利。日本では天賦人権とも
・社会 … 個人という無数の主体が、それぞれ社会契約を結ぶことで成立するとされた
