③[1 七月王政](1830-48) 労働者や中小資本家は不満を強め、普通選挙を求めて政府に反発
〇内政 … 極端な制限選挙に基づく立憲君主政。有権者は上層ブルジョワジーのみ
→ 銀行家・資本家が実権掌握、金融資本や大資本を中心として[2 産業革命が進行]
〇外政 … シャルル10 世がはじめたアルジェリア出兵を継続、植民地化をすすめる
→ 東方問題の一環である二度のエジプト＝トルコ戦争にも介入している ※七月の円柱
(2) 二月革命 同年のできごと：米墨戦争の終結・カリフォルニアで金鉱発見、バーブ教徒の乱(イラン)
①[3 二月革命]( 1848) … フランス国内では選挙法改正運動が高まっていた ※改革宴会
〇経緯 … ギゾー内閣は法改正を拒否、集会も禁止。パリ市民が禁を破りデモ行進、のち蜂起
→ ルイ＝フィリップ亡命・退位。共和政が宣言され臨時政府が成立(第二共和政の開始)
○影響 …「4 諸国民の春」をもたらす(国民主義が高揚、ヨーロッパ各地に波及)
※ドイツ出身の[5 マルクス]と[6 エンゲルス]が『7 共産党宣言』を刊行(1848)
②1848 年革命(「諸国民の春」) ウィーン体制の完全な崩壊
〇[8 三月革命](1848) … 二月革命をうけ、ドイツ連邦の各地で民衆が次々と蜂起
・[9 ウィーン]三月革命 … オーストリア首都で暴動、[10 メッテルニヒ]が亡命
→ 一時は墺皇帝が憲法制定を約束したが、フランス六月暴動の鎮圧をうけて反撃・鎮圧
・[11 ベルリン]三月革命 … プロイセン首都で暴動、国王が憲法制定を約束
→ 普通選挙に基づく[12 フランクフルト国民議会]を開催(1848)、統一ドイツ憲法を制定
⇒ 普王は憲法を拒否、軍を派遣してプロイセンやドイツ諸邦の革命運動を次々と鎮圧(1849)
〇ハンガリー … [13 コッシュート]が独立運動指導、マジャール人自治権を獲得(1848)
→ 10 月からオーストリアが反撃、ロシアの協力も得て運動を完全に鎮圧
〇ベーメン(ボヘミア) … オーストリア領。1848 年に一時的に政治的自由を獲得
・内部対立 … [14 チェック人]とドイツ人。とくにズデーテン地方(ドイツ人居住地)
・パラツキー … チェック人指導者。フランクフルト国民議会への参加を拒否
→ 対抗する形でスラヴ民族会議を召集したが、あくまでチェック人中心にとどまる
・運動の衰退 … プラハで急進派が蜂起、オーストリア軍が鎮圧。このとき民族会議も解散
〇ポーランド … 北部のポーランド立憲王国ではロシアが強力な支配を続けた
・南部 … 墺領。クラクフで1846 年に蜂起、ガリツィアで1848 年に蜂起。いずれも鎮圧
・西部 … 普領。ポズナニで1848 年に蜂起。プロイセン軍により鎮圧
〇イタリア … 北部がオーストリア領となっていた
・ミラノ … オーストリア領ロンバルディアの中心都市。1848 年に市民が蜂起
→ これに乗じてサルデーニャ王カルロ＝アルベルトがオーストリアに宣戦するも大敗
・[15 ローマ共和国]の建国(1849) … 青年イタリアを率いるマッツィーニが教皇領で宣言
→ 仏大統領ルイ＝ナポレオンが介入・派兵。ガリバルディらが抗戦するも敗北、共和国崩壊
〇イギリス … [16 チャーティスト運動]が最後かつ最大の盛り上がり(普選達成はならず)
