⑤中世都市の自治 中世都市は平均人口約5000 人。パリやヴェネツィアのような10 万都市は稀
○自治権の獲得 … 武力・経済力を背景に領主(国王・諸侯・司教など)から[1 特許状]を取得
・都市の自治 … 都市生活は参事会によって管理され、独自の都市法を形成
→ [2 農奴解放権]・裁判権・自衛権などをもつ。当初の参政権は大商人層に限られた
※「3 都市の空気は自由にする」… 中世都市の自治権をさす、19 世紀のことば
→ 農奴であっても、都市に移住し１年と１日経過すれば自由民となれた
・[4 コムーネ(自治都市)] … イタリア中心。農村を支配し事実上の都市国家となるものも
・[5 帝国都市(自由都市)] … ドイツ中心。納税により皇帝直属となり、諸侯と同格に
○ギルドの形成 … 都市で結成された商人の同業者組合
・都市内での統制 … ギルド同士の利益保護のため自由競争は禁じられた
→ 厳しい規則が設けられ、生産統制や技術・品質の保持がはかられた
⇒ 組合に所属しない商人は商業活動を禁じられ、ギルドが市場を独占
・[6 商人ギルド] … 商人中心の組合。当初から市政を支配した
・同職ギルド(ツンフト) … 手工業者中心の組合。ツンフト闘争を経て市政に参加
→ 親方のもとで職人・徒弟が働く[7 徒弟制]がとられ、親方のみギルドに加入
※[8 ツンフト闘争] … 13 世紀ごろから、各地の同職ギルドが大商人に市政改革を要求
→ フィレンツェでは、毛すき職人(チョンピ)が羊毛商人組合に対して反乱(1378)
コラム ユダヤ迫害史
・ヘレニズム時代(前４～前１世紀) … セレウコス朝がユダヤ人の信仰を弾圧 離散(ディアスポラ)
→ ユダヤ人が反乱(マカベア戦争、前２世紀)。独立を獲得しユダヤ教国家ハスモン朝を開く
・ローマ帝国時代 … ユダヤ人は二度にわたりローマに反乱を起こす(ユダヤ戦争。後66 年、132 年)
→ 鎮圧され、ローマ軍によりイェルサレムとその神殿は破壊。ユダヤ人の立ち入りは禁じられた
・キリスト教世界へ … ニケーア公会議(４世紀)でキリスト教と明確に区別。「異教」とされる
・十字軍運動(11 世紀末～13 世紀) … 異教弾圧運動の側面もあり、ユダヤ教も迫害の対象となった
・ラテラノ公会議(1215) … ユダヤ教徒の、キリスト教徒との通婚や荘園経営を禁止
→ 聖職者・騎士・農民になれないため、商人・金融業者・学者になるユダヤ人が多かった
このとき、ドミニコ会・フランチェスコ会が公認された 一部は東欧に逃れた
・ペスト大流行(14 世紀半ば) … 多数のユダヤ人がデマ・中傷を向けられ殺害された
・レコンキスタの完了(1492) … ユダヤ人が住みにくい社会が拡大していくことも意味した
→ 同年に、スペイン王国でユダヤ教徒追放令。改宗者も「マラーノ(スペイン語で豚)」と蔑視
・対抗宗教改革(16 世紀) … カトリック教会の保守改革が進行 啓蒙思想・フランス革命
→ キリスト教徒とユダヤ教徒との居住地分離政策を復活・強化。街区(ゲットー)の形成
※ゲットー … ヨーロッパ都市における、ユダヤ人の隔離居住地。18 世紀ごろから減少
→ 差別は継続。また20 世紀、ナチス＝ドイツによるユダヤ人の強制収容所もゲットーと呼称
