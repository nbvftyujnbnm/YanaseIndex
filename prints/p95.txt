２．西ヨーロッパの成長と拡大
(1) 大開墾時代 ※[1 シトー修道会]
①農業の発展 … 開墾を積極的に主導。清貧と献身を説き、12 世紀に教会改革を唱えた
○大開墾時代 … 11 世紀から13 世紀まで、荒れ地の開墾が進む。同時に農法の改良も進展
○中世農業革命 牛や馬に農具を繋ぐ方法 重い粘土質の土壌も耕作可能となる
・農具の発達 … 水車・鉄製農具の普及、繋駕法の改良。[2 重量有輪犂]の利用
・[3 三圃制] … 農地を三分する農法。春耕地(大麦など)・秋耕地(小麦など)・休耕地(放牧)
②村落共同体の形成 … 農民が連帯し、領主への立場をしだいに強めていった 聖人・聖遺物崇拝
○荘園の農地 … 重量有輪犂の使用など主要な農作業は共同で行われ、村落社会の結合が強まった
○キリスト教の布教 … 教区ごとに教会がおかれ、民俗信仰と結びつく形でキリスト教が広まる
→ [4 巡礼]も流行(三大巡礼地イェルサレム・ローマ・サンティアゴ＝デ＝コンポステラ)
(2) カトリック教会の改革
①ローマ教会の発展 … 11 世紀ごろ、王権の弱体や農民への布教を背景に、教皇の権威拡大
○聖職者の[5 階層制(ヒエラルキー)] … ピラミッド状の組織形態
→ 上位から教皇、[6 大司教]、[7 司教]、[8 司祭]、修道院長など
※聖職叙任権 …カロリング朝以来、聖職者の任命権はしばしば国王や皇帝の側に存在した
○教会の世俗化 … 寄進を受けた教会は荘園領主に。聖職者の妻帯や[9 聖職売買]も行われた
○教会の改革運動 … とくに11 世紀ごろ、修道院からおこった動き
・[10 クリュニー修道院] … ベネディクト派、南フランスに成立(910)。改革運動の先陣
・[11 シトー修道会] … ベネディクト派、中部フランスに成立(11 世紀末)
→ 大開墾時代を支え、クリュニー修道院にかわる改革運動の中心地となる
・[12 托鉢修道会](13 世紀ごろ) … 徹底した清貧・労働を説き、異端の懐柔などで成果
→ [13 ドミニコ修道会](南仏)・[14 フランチェスコ修道会](伊)など
②教皇[15 グレゴリウス７世]の改革(11 世紀) … 聖職者の妻帯や聖職売買を厳しく取り締まる
○[16 聖職叙任権闘争] … 神聖ローマ帝国は聖職叙任権を国内統治に活用(帝国教会政策)
→ 皇帝による叙任権行使は聖職売買にあたるとして教皇が批判
○[17 カノッサの屈辱]( 1077) … 教皇は独王(のち皇帝)[18 ハインリヒ４世]を破門
→ 国王が教皇を訪ねて謝罪し、破門を解かれる。教皇権の優越が明らかとなる
※[19 ヴォルムス協約](1122) … 教皇の死後、政教分離の形で叙任権闘争は妥協をみる
③[20 ウルバヌス２世] … [21 クレルモン公会議](1095)で[22 十字軍]派遣をよびかけ
④[23 インノケンティウス３世](13 世紀) … 教皇権の絶頂期。托鉢修道会を認可
○「24 教皇は太陽、皇帝は月」… 皇帝や国王を破門、あるいは破門をちらつかせて屈服させた
→ 皇帝オットー４世、イングランド国王[25 ジョン]、カペー朝[26 フィリップ２世]
○[27 第四回十字軍]を提唱 … 十字軍本来の目的から逸脱し、一時的に教皇が十字軍を破門
