162
〇東アジア方面への進出
・東シベリア総督の設置 … 初代ムラヴィヨフ(任1847-61)が次代皇帝のころ活躍
・日本との関係 … アメリカがペリーを日本に派遣するとの報をうけて、ロシアも対抗
→ プチャーチンを派遣、1855 年(和暦では1854 年にあたる)に日露和親条約を締結
②[1 アレクサンドル２世](位1855-81)
〇東方問題
・クリミア戦争 … パリ条約(1856)で黒海の制海権を失う。皇帝はロシアの後進性を痛感
・露土戦争(1877-78) … サン＝ステファノ条約(1878)で南下政策は大きく前進
・ベルリン会議(1878) … ビスマルクの外交的介入により、バルカン方面の南下は再び挫折
〇清との国境画定 … 東シベリア総督(先帝ニコライ１世が設置)ムラヴィヨフが活躍
・アイグン条約(愛琿、1858) … アムール川(黒竜江)以北をロシアが領有、沿海州は共同管理
・北京条約(1860) … 沿海州もロシアが領有。清露国境はアムール川とウスリー川となる
→ 沿海州にウラジヴォストーク港を建設(1871)、日本海方面の不凍港を獲得 沿海州西端の川
〇内政 … クリミア戦争での敗北をうけて、国内で「上からの改革」を推進
・[2 農奴解放令]( 1861) … 農奴の人格的自由と土地所有を承認。農地分与は有償
→ 農民の生活はさほど変わらなかったが、このあとロシアでは1890 年代に産業革命が進行
※買戻し金 … 農地分与の代償として農民に賦課。高額で、約50 年かけての返済を強いられた
→ 分与地は返済満了まで農村共同体(ミール)の所有とされ、満了できる農民は稀であった
・ポーランドの反乱(1863) … 翌年には鎮圧され、ロシア領化がいっそう進行
〇外政
・[3 アラスカ] … 財政難から、比較的安価でアメリカに売却(1867)
・ウズベク人国家の支配 … 1870 年前後に、ヒヴァ・ブハラ・コーカンドの３ハン国を制圧
・イリ事件(1871) … ロシアがイギリスの中央アジア進出を警戒、清朝支配下の新疆に介入
→ 1881 年のイリ条約で、ロシアはイリ地方を清朝に返還したが賠償金を得た
・樺太・千島交換条約(1875) … 日露和親条約のときの国境を再画定。ロシアは樺太を領有
〇[4 ナロードニキ運動] … 1860～70 年代におこった運動。推進者たちをナロードニキと呼ぶ
・内容 … 資本主義を通過することなく、ミールを基盤とした社会主義の実現をはかる
→ ナロードニキはツァーリズム打倒のため、農村の中に入り革命思想を広めようとした
・[5 インテリゲンツィア] … ロシアの知識人階級。自由主義運動の中心にいた
→ ポーランド反乱を機に皇帝専制が強化されると、多くが反発してナロードニキとなった
・運動の展開 … スローガン「人民の中へ(6 ヴ＝ナロード)」。農民の理解を得られず
→ 絶望したナロードニキはテロリズムやニヒリズム(虚無主義。あらゆる価値を否定)に傾斜
・アレクサンドル２世の暗殺(1881) … テロリストのソフィア＝ペロフスカヤらによる
※ポグロム(ユダヤ人迫害)の発生 … 貧困などの社会的不満が、皇帝暗殺を機にユダヤ人に向けられた
