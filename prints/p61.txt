(3) オアシスの道 パミール高原以西のアム川・シル川流域を、古くは「ソグディアナ」と称した
①形成と発展 … 中央アジアのオアシス民が、大陸西方と大陸東方を結ぶ
○西方 … 前４世紀、アレクサンドロス大王の東方遠征でギリシア人の入植がすすみ発展
○中部 … [1 ソグド人](イラン系。ゾロアスター教を信仰)がサマルカンドを拠点に隊商交易
○東方 … 漢代に西域進出がすすみ、絹や仏典の往来路として重視される。「シルク＝ロード」
②中国西域の動き 月氏が匈奴・烏孫に圧迫されアム川北岸へ移動
○武帝の時代 … 前漢、前２世紀 イリ川上流の遊牧民。鮮卑や柔然におされ５世紀に衰退
・対[2 匈奴] … 衛青・霍去病を将軍として派遣。オアシス地帯から一掃
→ 甘粛地方(河西回廊)に河西４郡を設置。最西端は[3 敦煌郡]
・対[4 大月氏] … 張騫を派遣、協力に失敗。烏孫にも向かわせたがこれも失敗
・対[5 大宛(フェルガナ)] … シル川上流。李広利を派遣、遠征により服従させる
○[6 西域都護府]の設置 … 前漢、前１世紀。一時断絶するが、後漢代に復活(91-107)
→ 後漢代、後１世紀に[7 班超]が西域都護に就任。部下の甘英を大秦国(ローマ)へ派遣
⇒ 漢による西域の支配や交易などを通じ、大乗仏教・ガンダーラ美術・イラン文化が中国に伝播
○僧の往来 … ４～７世紀(魏晋南北朝～唐代)。このころソグド人がとくに活躍
・西域⇒中国 … [8 仏図澄](４世紀)・[9 鳩摩羅什](５世紀)が、仏典の漢訳などに活躍
・中国⇒西域 … [10 法顕](４Ｃ末、東晋⇒グプタ)・[11 玄奘](７Ｃ、唐⇒ヴァルダナ)
③中央アジアのトルコ化とイスラーム化 高昌国(トルファン盆地の漢人オアシス国家)を経由
○トルコ化 … 東方(中国西域方面)からの影響。「ソグディアナ」から「トルキスタン」へ
・[12 突厥] … ６世紀に中央アジア進出。ソグド人を商業・外交に利用した
・[13 ウイグル] … ９世紀、キルギスにより国家滅亡。一派が中央アジアに逃れて定住
○イスラーム化 … 西方(中東方面)からの影響 マムルーク(トルコ人などの奴隷兵士)の登用
・ウマイヤ朝 … 704 年、アム川をこえて中央アジアに進出
・アッバース朝 … ウマイヤ朝にかわって、北アフリカから中央アジアまで支配
→ [14 タラス河畔の戦い](751)で唐に勝利。唐の西域進出停止、唐から製紙法が西伝
・サーマーン朝(イラン系) … ９世紀、西トルキスタンでアッバース朝から自立
・カラ＝ハン朝(トルコ系) … 10 世紀にイスラーム受容、トルキスタン進出。12 世紀まで存続
→ トルコ人のイスラーム化のはじまり、そして中央アジアのトルコ化のさらなる進展
④12 世紀以降 トルコ系。イラン方面も支配 契丹人の耶律大石が、カラ＝ハン朝を倒して建国
○12 世紀 … アム川流域にホラズム(イスラーム王朝)、シル川流域に[15 西遼](カラ＝キタイ)
○モンゴルの支配 … 13 世紀、[16 チンギス＝ハン]が遠征。ホラズムやナイマンを滅ぼす
→ チャガタイ＝ハン国が成立。14 世紀半ばにイスラームを受容、同時期から内紛・分裂
○ (1370-1507) … チャガタイ＝ハン国から自立、都[17 サマルカンド]
○[18 ウズベク人] … ティムールを滅ぼす。ブハラ・ヒヴァ・コーカンド３ハン国を建設
