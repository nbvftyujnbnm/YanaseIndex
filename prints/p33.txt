２．古代イラン・中央アジア・アフリカ アレクサンドロスが前330 年にアケメネス朝を滅ぼす。
(1) パルティアとササン朝ペルシア 前324 に大王が没すると後継者(ディアドコイ)が抗争。
①[1 パルティア](前248-後226) ディアドコイのひとりが前312 年に建国、前63 年滅亡
○[2 アルサケス] … 建国者。イラン系遊牧民を率いて[3 セレウコス朝]から独立
○ミトラダテス１世 … 前２世紀、最盛期。セレウコス朝を攻め、メソポタミアまで版図拡大
→ のちに都となる[4 クテシフォン]を建設(従来はイラン北部のヘカトンピュロスが都)
○中継貿易 … 中国・インドとローマを結び、西アジアで覇権を握る ギリシア風の貨幣を鋳造
『史記』の記述：後１世紀、甘英(西域都護班超の部下)が来訪。「5 安息」国として中国に紹介
○パルティアの文化 … 当初はヘレニズムの影響。前１世紀ごろからはイラン文化も重視
○ローマ帝国との戦い 前63 年にセレウコス朝を滅ぼし、小アジアに勢力を拡大していた
・クラッスス … 第１回三頭政治の一角。パルティアに遠征するも戦死(前１世紀)
・トラヤヌス … ローマ皇帝(五賢帝のひとり)。後２世紀にクテシフォンを占領
②[6 ササン朝ペルシア]( 226～651) を滅ぼし、イラン系農耕民の王朝をひらく
○[7 アルデシール１世] … 建国者、都[8 クテシフォン]。[9 ゾロアスター教]国教化
○[10 シャープール１世] … ２代国王、対外進出。 「イラン人および非イラン人の諸王の王」
・西方 … ローマ帝国と争い、シリアを攻略。皇帝[11 ヴァレリアヌス]を捕虜とした(260)
・東方 … クシャーナ朝に侵攻。一時はインダス川まで勢力を拡大 エデッサの戦い
○衰退期 … ５世紀、遊牧民[12 エフタル]の侵入を受ける アム川・シル川流域を根拠とする
○[13 ホスロー１世] … ６世紀、ササン朝の再興者 イラン系遊牧民。グプタ朝も攻撃
・西方 … 東ローマ帝国皇帝[14 ユスティニアヌス]と争い、優勢を維持 正統カリフ時代
・東方 … トルコ系遊牧民の[15 突厥]と結び、[12 エフタル]を挟撃して滅ぼす
○滅亡 … ビザンツ帝国と抗争続く。 [16 ニハーヴァンドの戦い](642)でイスラーム勢力に敗北
○ササン朝ペルシアの文化 … 東西交易により宗教・工芸品が多数往来
・ゾロアスター教 … 国教とされる。ホスロー１世が聖典『17 アヴェスター』の編纂を開始
・[18 マニ教] … [19 ゾロアスター教・ 仏教・ キリスト教]などを融合
→ 迫害され、北アフリカ・南フランス・中央アジア・インド・中国などに伝播
・[20 ネストリウス派キリスト教] … 東ローマで異端とされ、イランに流入。独自に発展
→ ササン朝を通じて中国に伝わり、[21 景教]とよばれた エフェソス公会議(431)
・マズダク教 … ５世紀、ゾロアスター教祭司がマニ教に影響され創始。大弾圧をうけほぼ根絶
・工芸品 … 職人を保護、精巧なガラス器・絨毯など生産。[22 シルク＝ロード]経由で日本へ
雑学：前２世紀ごろの交易国家 羊皮紙の発明 法隆寺の獅子狩文錦、正倉院の漆胡瓶・白瑠璃碗など
・ペルガモン … 小アジア西岸。前３世紀にセレウコス朝から独立、前133 年にローマ帝国の支配
・ナバテア … アラビア半島西北部、前２世紀、首都ペトラ。後106 年にローマに併合。宝石・香料
・ヒムヤル … 半島南西部(イエメン地方)、前２世紀。後４世紀にキリスト教国化、５世紀に滅亡
