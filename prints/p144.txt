４章 環大西洋革命～工業文明と国民国家の誕生～
１． イギリス産業革命
(1) 産業革命の背景
①植民地戦争での勝利
〇広大な植民地の獲得 … 北米・インドなど。 [1 安定した海外市場]となる
〇三角貿易の進展 … [2 資本の蓄積] がすすむ。英の輸出品である[3 綿布]の需要増大
→ [4 キャラコ](インド産綿布)にも対抗し、イギリス企業家たちが生産の効率化をはかる
⇒ 産業革命は[5 綿工業]からはじまることとなった
②イギリス国内の要因
〇 豊富な国内資源 … 石炭・鉄鉱石など
〇封建的制約からの離脱 … 産業革命期は、政治的には市民革命がおこった時期
→ [6 毛織物]工業で、ギルドに縛られない問屋制や[7 工場制手工業]が発達
③[8 農業革命] … 18 世紀後半、産業革命と並行しておこる
〇[9 第２次囲い込み] 議会も協力して囲い込み。第１次囲い込みよりもはるかに大規模
・背景 … 産業革命による人口増加で穀物需要が増大。対ナポレオン戦争も一因
・内容 … 穀物生産の効率化のため、[10 ノーフォーク法]を導入し大農場経営をめざす
※17 世紀半ばから英東部のノーフォーク地方で始まった四輪作法
→ 従来の三圃制と異なり休耕地がない(家畜は舎飼い)。大麦・クローバー・小麦・カブの順
〇農業革命の内容と影響 … ジェントリによる農場経営から、資本主義的大農場経営へと変化
→ 大地主や地主に経営を任された借地農が、安価な給料で農業労働者を支配
⇒ 大量の失地農民・離村農民が発生。その一部は 工場賃労働者となり産業革命を支えた
(2) 技術革新のおこり
①綿工業 … 綿花から綿糸をつくるのが紡績業、綿糸から綿布をつくるのが綿織物業
〇[1 1 ジョン＝ケイ] … 飛び杼を発明(1733)、綿織物業における糸通し作業を効率化
→ 原料となる綿糸の生産がおいつかず「糸飢饉」が発生。 紡績機の改良を促進
〇[12 ハーグリーヴズ] … ジェニー紡績機(多軸紡績機、1764)を発明。産業革命の始まり
→ 人力ではあるが多軸。すなわちひとつの機械で複数の糸を紡ぐことができた
〇[13 アークライト] … 水力紡績機(1768)。人力よりも効率的かつ強度の高い綿糸を生産
→ ランカシャー地方(中心地マンチェスター)に紡績工場を設立、本格的な紡績業の産業化を達成
〇[14 クロンプトン] … ミュール紡績機(1779)を発明し、綿糸の大量生産に成功
〇[15 カートライト] … 力織機を発明(1785)。動力は人力・馬力・水力など徐々に改良
→ 1787 年に蒸気機関が動力として導入されると、綿織物業でも大量生産が可能となる
※[16 ラッダイト運動] … 失職した手工業者による、機械打ちこわし運動。1810 年代に最盛
→ 発明家たちの住居・工場なども襲撃されたが、政府の弾圧をうけて鎮静化していく
