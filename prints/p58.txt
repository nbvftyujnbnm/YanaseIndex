○[1 黄巾の乱]( 184) … 華北で宗教結社[2 太平道]をおこした[3 張角]が反乱
→ 鎮圧されたが後漢の衰退を決定づけ、群雄割拠の状態となり220 年に後漢は滅亡
※[4 五斗米道](天師道) … 江南で[5 張陵]がおこした宗教結社。黄巾の乱に参加
→ 乱の鎮圧後も後漢末の混乱期に宗教勢力のひとつとして残る。太平道は廃れた
④漢代の社会と文化 いずれも、のちの道教の源流となる
○農業の重視 … 商工業も発展したものの、国家はしばしば商人を統制。農業中心を崩さず
○豪族による大規模経営 … 大土地を所有し農地経営。私兵を率い、地方で勢力を拡大
→ 国家による豪族抑制はうまくいかず、[6 郷挙里選]が実施されると中央政治にも進出
○儒教の保護 … 前漢～新代にかけて定着。以後二千年近く中国王朝の権威を支える思想となった
・武帝による国教化 … 前漢代、[7 董仲舒]の献策。五経を定め[8 五経博士]を設置
・[9 訓詁学] … 後漢代に発達し、経典の整理・注釈を行う。[10 鄭玄]や馬融らが活躍
○宗教
・仏教伝来 … 後漢代とされる(初の漢訳経典とされる『四十二章経』の伝来説話による)
・道教の源流 … 後漢代、張角による[11 太平道]と、張陵による五斗米道
○歴史書の成立 本紀と列伝からなる記述方式。以降、中国正史の標準的な方式として採用
・『史記』… 前漢の[12 司馬遷]が、五帝の時代から前漢代までを[13 紀伝体]で記す
・『漢書』… 後漢の[14 班固](西域都護を務めた班超の兄)が、前漢代を記述
○暦法 殷代からみられる。太陰暦を基軸とし、太陽暦との差は閏月をはさむことで解消
・太初暦 … 前漢の武帝が初の改暦を行い、前104 年から前84 まで使用。太陰太陽暦に基づく
・張衡 … 後漢の政治家・科学者。天球儀や地震計をつくる
○文字 … 漢代に篆書から隷書に移行、後漢末には行書や楷書のはしりがみられはじめる
・許愼 … 後漢代、『説文解字』を著す。中国最古の漢字字書
・[15 蔡倫] … 後漢代、製紙法を改良。従来は木簡・竹簡・帛(絹布)に記入していた

①朝鮮古代史
○先史時代 北京原人の活動時期と一致するが、人骨は出土せず
・黒隅里遺跡 … 平壌近郊、約50 万年前。打製石器・獣骨などが出土
・櫛目文土器 … 狩猟民に多くみられる土器。前5000 年ごろから河川の下流域で出土
・支石墓 … 前1000 年ごろからみられるようになる。支配者の存在を示唆するもの
・考古年代 … 前1000 年ごろから青銅器時代。前４世紀ごろから鉄器が普及、鉄器時代へ
○国家の成立 紂王を諫め、うとんぜられ朝鮮に亡命。その地を武王に封じられたとする伝説
・檀君朝鮮 … 建国神話。檀君が前3000 年紀に建国、1908 歳の長寿をもって支配したとされる
・箕子朝鮮 … 前12 世紀に殷の箕子が亡命し建国、41 代準王(位前220-前195)のとき滅亡
・衛氏朝鮮 … 朝鮮に実在した最初の王朝とされる。前２世紀、燕の衛満が準王を追放して建国
