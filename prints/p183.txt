③普仏戦争の敗戦とその後
〇臨時政府 … 皇帝の敗北をうけパリ市民が蜂起、共和政を宣言(1870)。ブルジョワ共和派中心
〇[1 ティエール] … 政府首班となる。政府内でも継戦・講和両論あったが講和条約を締結
→ ドイツ帝国に[2 アルザス・ ロレーヌ]の割譲および50 億フランの賠償金支払い
〇[3 パリ＝コミューン] … 屈辱的な講和に反発したパリ民衆が樹立 ※画家クールベも参加
→ 世界初の労働者・市民による自治政府。第１インターの支援を受けるも鎮圧された(1871)
④[4 第三共和政](1875-1940) 政府軍やドイツ軍により鎮圧
〇共和国憲法の制定(1875) … このとき国旗・国歌も制定(三色旗、 「ラ＝マルセイエーズ」)
・政体 … 三権分立、二院制。男子普通選挙が実施された
・大統領制 … 両院の合同選挙で選出。権限は弱く、行政権は事実上内閣が握った
〇政情の不安定 … 共和派と反共和派(軍部・王党派・カトリックなど)が対立
→ 小党が分立し、短命な内閣が続く。民衆の対独復讐心から軍部・右派を支持する動き
〇[5 ブーランジェ事件](1887-89) … 国民の中から前陸相ブーランジェを支持する声が高まる
→ 共和政転覆・軍事政権樹立が叫ばれたが、ブーランジェが突如亡命しクーデタには至らず
〇[6 ドレフュス事件](1894) … [7 ユダヤ系]軍人が、ドイツのスパイとして告発される
・経過 … 軍法会議で有罪とされ、反共和派はこれを支持。共和派はユダヤ人差別として批判
→ ドレフュスの有罪・無罪をめぐり、世論を二分する大論争に発展
・「余は弾劾する」(1898) … 作家[8 ゾラ]が無罪を主張 オーストリアのヘルツルの指導で、
・結果 … 1906 年にドレフュスの無罪判決が下される 世界シオニズム機構設立(1897)
→ 反ユダヤ主義の顕在化をうけて、ユダヤ人の中ではシオニズムが高まる契機にもなった
〇フランスの労働運動 ゼネラル＝ストライキによる社会革命を目指す、過激な方針
・フランス労働総同盟(ＣＧＴ、1895) … サンディカリズムを基本とする
・[9 フランス社会党](統一社会党、1905) … ジャン＝ジョレスが指導。社会改良を主張
※急進社会党(1901) … クレマンソーが指導、ブルジョワ中道政党。ドレフュス事件が設立の背景
☆19 世紀の国際協力 国際電気通信連合(ＩＴＵ、1947)の源流機関のひとつ
〇 (1864) … マルクス(独)らの指導でロンドンに設立。1876 年に解散
〇[10 国際赤十字社](1864) … デュナン(スイス)により設立、本部ジュネーヴ
〇国際電信連合(1865) … パリで設立、本部はベルン(スイス)。料金体系の統一など
〇万国郵便連合(ＵＰＵ、1874) … ＩＴＵを除けば、最古の国連機関
〇国際オリンピック委員会(ＩＯＣ、1894) … クーベルタン(仏)の提唱、ソルボンヌで設立
→ 本部はスイスのローザンヌ。1896 年にアテネで第１回オリンピック大会が開催される
〇[11 万国平和会議] … ニコライ２世の提唱で、第１回がオランダのハーグで開催(1899)
→ 軍縮には至らなかったが、国際仲裁裁判所(常設仲裁裁判所、1901)の設置などの成果
⇒ 現在は国連による国際司法裁判所(1946)にその役割を譲っている。いずれも本部はハーグ
