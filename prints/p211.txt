〇新経済政策(1 ネップ、1921-) … 戦時共産主義の失敗をうけて方針転換
・資本主義の一部導入を許容 … 私的所有・市場経済を認める
→ 共産党一党独裁や、国家による銀行・大企業・貿易の管理は継続
・内容 … 穀物の強制徴収を廃止、余剰生産物の自由販売を許可。中小企業は私的営業を許可
・結果 … 国内では農村にクラーク(富農)、都市にネップマン(小資本家)が登場
→ 国外では資本主義国との貿易開始や、ソ連の国際的承認などにつながった
※ソ連の承認 … 敗戦国ドイツが最初(2 ラパロ条約、1922)。英仏伊は1924 年に続く
→ 日本は1925 年、[3 アメリカ]は1933 年(世界恐慌を経て善隣外交)
④ソ連の成立 ロシア・ザカフカース・ (白ロシア)・ウクライナ
〇[4 ソヴィエト社会主義共和国連邦]の成立(1922-91) … ４国のソヴィエト政府が連合
→ のちモルダヴィア、カザフ・トルクメン・ウズベク・タジク、キルギス、バルト三国も参加
〇レーニンの死(1924) … 後継者争いが発生。1929 年にトロツキーが追放されスターリン勝利
・[5 トロツキー] … [6 世界革命論]。世界全体を社会主義化するまでの永久革命を指向
→ ソ連一国のみでの社会主義の維持は不可能である、ということが前提。レーニンから継承
・[7 スターリン] … [8 一国社会主義論]。ソ連のみで社会主義維持が可能とする
〇スターリン独裁(1929-63)
・[9 第１次五カ年計画](1928-32) … ２つの経済的政策による急激な社会主義国家建設計画
* 重化学工業の発展 … 鉄鉱・機械・石炭などの重工業分野は300％超の成長が目指された
* 農業の発展 … 農業の集団化。集団農場(10 コルホーズ)・国営農場(11 ソフホーズ)
⇒ 計画経済を実施したことにより、世界恐慌(1929)の影響を免れる。スターリンの権力強化
・大粛清(大量テロル) … 1933 年のキーロフ暗殺を皮切りに、反対派を処刑・シベリア流刑
・第２次五カ年計画(1933-37) … 餓死者を出さぬよう消費財などの軽工業分野の発展をめざす
→ ファシズム国家出現をうけて、結局は重工業・軍需産業分野に注力された
・スターリン憲法(1936) … 独裁の確立。民主的規定はあるが社会主義社会の保全が大前提

〇段祺瑞 … 安徽派(北洋軍閥のひとつ)の指導者。袁世凱の死後に実権を握り、北京も支配
・西原借款(1917) … 日本(寺内正毅内閣)の支援を受ける
・第一次世界大戦への参戦(1917) … 対独宣戦布告。欧米製品が届かなくなり、工業化の契機
・安直戦争(1919-20) … 直隷派 (馮国璋の死後、曹錕が指導)は米英・奉天派の支援を受ける
