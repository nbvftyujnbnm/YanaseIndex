(4) 宋代の社会と文化
①社会階層の変化
○[1 門閥貴族の没落] … 魏晋南北朝～唐代に権勢を誇った。節度使の台頭を機に勢力を失う
○[2 形勢戸] … 唐代後半、均田制の崩壊とともに出現した新興地主。宋代にかけて成長
→ 科挙に合格して中央官僚(3 官戸)となり、労役の免除など特権を得た
※[4 士大夫]・読書人 … 裕福な地主層が教養を身につけ科挙に合格し、知識人として活躍
○[5 佃戸] … 唐代後半、均田制の崩壊とともに出現した小作人。地主の支配を受けた
②農業・工業の発展 華北 … 乾燥地帯、畑作が主。粟・麦・黍などで二毛作が行われた
○[6 江南]開発の進展 … 後漢代にはじまり、南北朝・隋唐宋代も継続
・低湿地の開発 … 南宋の漢人により進展。干拓を進め、囲田・圩田・湖田など水田化
・[7 占城稲( チャンパ米)]の導入 … 日照りに強い。早稲であり、二毛作・二期作が進展
⇒ [8 長江 下流域]が農業の中心地に。「9 蘇湖(江浙)熟すれば天下足る」
※農業技術の発展 揚水装置。漢代に始まり、唐宋代に普及したとされる
→ 苗代・正条植え・施肥・秧馬(田植え具)・客土法(土壌の改良法)・竜骨車など
⇒ 日本には鎌倉時代に大唐米(占城稲のこと)が、室町時代に竜骨車が伝わる
○特産品の生産 … 茶・絹・麻・サトウキビなど。茶を飲む習慣は遊牧民にも広まった
→ 絹や茶は中国から周辺諸国への主要な輸出品となり、遊牧民からは馬などが輸入された
○織物業 … 綿織物・絹織物が南宋のころから発展。明代に本格化
○窯業 … 陶磁器の主要産地として江西省の[10 景徳鎮]が登場。[11 青磁・白磁]を生産
③商業の発展 瓦市(繁華街)・酒楼(料理店)・勾蘭(劇場)・茶館など、夜間営業も開始
○[12 草市] … 都城の外におかれた市。唐代の厳しい市制が崩壊し、民間主導で繁栄
→ 中には鎮・市とよばれる新興都市に成長するものもあった
※都市の役割 … 唐代までは軍事的・政治的な中心地
→ 宋代には、草市の発展とともに、城壁をもたない商業の中心地へと変化
○[13 行・ 作] … 前者は商人、後者は手工業者の同業組合。市制の緩和を機に営業権を独占
○貨幣経済の発達 … [14 銅銭]の使用が普及し、平安時代末期以降の日本にも輸出
→ 紙幣も発行され、宋代には[15 交子]、南宋代には[16 会子]とよばれた
○[17 市舶司] … 唐末から宋代にかけて設置された、海外貿易を監査する役所
→ 唐の６代玄宗が[18 広州]に設置。のち[19 泉州]・杭州・明州(寧波)などにも設置
○中国商人の活躍(10 世紀) ※ムスリム商人はアッバース朝解体に伴い勢力を弱める
→ 陶磁器や絹織物を主力商品として、[20 ジャンク船]で南シナ海・インド洋に進出
※蕃坊 … 宋代の外国人居留地。イスラーム教徒の大食(アラブ人)商人が住んだ
○私貿易の活発化 … 中国・朝鮮・日本の間で、朝貢貿易とは別に行われた
※平清盛 … 日本の太政大臣(1167 就任)、南宋の商人と日宋貿易を行う。神戸の大輪田泊を改修
