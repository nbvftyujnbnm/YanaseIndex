・衆生救済 … [8 菩薩]信仰に基づき、広く大衆を救済することをめざす

⑤[12 グプタ朝](320-550 ごろ) … 都[13 パータリプトラ]
○[14 チャンドラグプタ１世] … 建国者、初代国王。ガンジス川流域を統一
○サムドラグプタ … ２代国王。南インドを除くインドのほぼ全域を統一
○[15 チャンドラグプタ２世(超日王)](３代、位376-415 ごろ) … サカ族を滅ぼし最盛期
→ マウリヤ朝とは異なり、属領では従来の支配者を残す分権的な統治体制がとられた
○衰退 … ５世紀半ば、[16 エフタル]の侵入。５～６世紀、ローマ帝国の混乱で貿易が停滞
※インド古典文化の成熟
○[17 ヒンドゥー教]の浸透 … グプタ朝において、支配体制にとりこまれた
・成立 … バラモン教を母体とする多神教。バラモン教が、非アーリヤ的な民間信仰を吸収し変化
→ 紀元前から徐々に形成。開祖や特定の教義・経典なし。バラモンが司祭をつとめる
※三大神 … 創造神ブラフマー、維持神[18 ヴィシュヌ]、破壊神[19 シヴァ]
・『20 マヌ法典』の定着 … ヴァルナごとの慣習や生活規範を規定。バラモンの優位を強調
・インドの伝統的社会規範の固定 … カースト制や、食料・慣例・衣服など広範な浄・不浄観
○仏教 … 民衆の信仰はヒンドゥー教へと移行していったものの、宮廷の保護を受けて繁栄
・[21 法顕] … 東晋の僧。５世紀初めに陸路で来印、海路で中国に戻る。『仏国記』を著す
・[22 ナーランダー僧院] … ５世紀に創建。仏教以外にもバラモン教や天文学などを研究
→ グプタ朝滅亡後、唐から玄奘(ヴァルダナ朝時代)や義浄(ラージプート時代)が来印し学んだ
・[23 グプタ様式] … ヘレニズムの影響を脱した、純インド的な仏教美術
→ [24 アジャンター]やエローラの石窟寺院にみられる
○[25 サンスクリット語] … 公用語とされ、文学も発展
・叙事詩 …『26 マハーバーラタ』『27 ラーマーヤナ』(いずれもヒンドゥー教の聖典)
・戯曲 …『28 シャクンタラー』([29 カーリダーサ]の作。チャンドラグプタ２世の時代)
○数学 … 十進法や円周率のほか、前２世紀ごろには[30 ゼロの概念]が存在
→ インド数字はアラビア数字の直接の起源となった
