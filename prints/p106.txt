④教会建築
○ビザンツ様式 … ６世紀ごろ全盛。 (ドーム)やモザイク壁画(小片をうめこみ描画)
・[1 ハギア＝ソフィア大聖堂] … ユスティニアヌス帝が都コンスタンティノープルに建立
→ ドームが有名。その周囲の尖塔(ミナレット)はイスラーム支配下で追加されたもの
・[2 サン＝ヴィターレ聖堂] … ラヴェンナに存在。ユスティニアヌスを描くモザイクが有名
○[3 ロマネスク様式] … フランス中南部でおこり、11～12 世紀に広がる
・特徴 … 半円アーチと厚い石壁。このため装飾は控えめで、窓は小さい
→ ローマ・ケルト・ゲルマン・東方の諸伝統が、キリスト教美術として総合された
・[4 ピサ大聖堂] … イタリア。土台が傾いたが建設続行された「ピサの斜塔」が有名
・[5 マインツ大聖堂] … ドイツ。度々改築、ロマネスク・ゴシック・バロック様式が混在
・クリュニー修道院 … フランス。11 世紀ごろ、教会の改革運動の中心地となった
・ヴォルムス大聖堂 … ドイツ。ヴォルムス協約や、のちのヴォルムス帝国議会の舞台
○[6 ゴシック様式] … カペー王権の強化に伴い、13 世紀前半に北フランス中心に最盛
・特徴 … 尖頭アーチ。壁が薄く、広い窓に多彩な[7 ステンドグラス]、豊富な装飾
・[8 ノートルダム大聖堂] … フランス、パリ大学の起源。2019 年にステンドグラスなど焼損
・[9 シャルトル大聖堂] … フランス、「美しき絵ガラスの聖母子」が有名
・アミアン大聖堂 … フランス。こちらも正式にはノートルダム(聖母マリアの意)大聖堂という
・ランス大聖堂 … フランス、クローヴィスの洗礼地といわれる。シャルル７世戴冠の地
・[10 ケルン大聖堂] … ドイツ。ケルン大司教は選帝侯のひとりであった
・ウェストミンスター寺院 … イギリス。ノルマン朝期のものを13 世紀に改築。背後には宮殿
⑤騎士道文学 … ラテン語ではなく、俗語で口承された
○『11 ローランの歌』… 仏、カール大帝の臣下(十二勇士)のひとりローランが主人公
○『12 ニーベルンゲンの歌』… 独、ジークフリートの生涯・ブルグント人の滅亡の二部構成
○『13 アーサー王物語』… 英、サクソン人を撃退したブルトゥン人の伝説。円卓の騎士
※[14 吟遊詩人] … 南仏ではトゥルバドゥール、独ではミンネジンガーとよばれた
→ 宮廷や貴族の館などで、騎士道的な恋愛叙情詩を創作し歌い上げた
