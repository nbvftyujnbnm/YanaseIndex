○[1 渤海](698-926) … とくに８～９世紀に栄え、「海東の盛国」とよばれた
・建国 … ツングース系靺鞨人の大祚栄が建国。貊人と同系統であり、高句麗の遺民
→ 朝鮮半島北部から中国東北部、沿海州まで支配。[2 契丹( 遼)]によって滅亡
※上京竜泉府(東京城)… 最も栄えたときの渤海の都。現在の中国の黒竜江省に存在
・外交 … 唐からは冊封を受け、新羅と対抗。日本とは奈良～平安時代にかけて友好関係
→ 渤海から日本、日本から渤海への使者はそれぞれ日本では渤海使・遣渤海使とよばれた
③雲南 … 唐と吐蕃の対立に乗じて、８世紀にチベット＝ビルマ系の南詔が独立
→ タラス河畔の戦い(751)を契機に勢力拡大、902 年滅亡。のち大理(一説にタイ系)がおこる
⇒ 唐の周縁諸国はいずれも律令・都城制・儒教・中国仏教などをとりいれ、漢字文化圏を形成
(6) 唐とユーラシアの変動
①武韋の禍 … ７世紀末から８世紀初にかけて、政治が混乱
○[3 則天武后](武則天、位690-705) … 高宗の皇后。中国史上唯一の女帝
・即位 … ４代中宗・５代睿宗を相次ぎ廃位。自ら「聖神皇帝」を称し即位、反対派を次々と排除
・武周革命 … 国号を[4 周]と改め、唐は中断。『周礼』を手本とし律令制維持に尽力
→ 科挙官僚を積極的に登用。仏教を手厚く保護し、このころ義浄が唐に戻る
○韋后 … 中宗の皇后。則天武后の死後に中宗が復位すると、中宗を暗殺。政権掌握をはかる
②社会の変化 睿宗の子(のちの６代玄宗)が韋后を殺害し、睿宗が復位
○律令制の軋み … 唐の混乱期にあって、貧富の差が拡大
・農民の疲弊 … 逃亡を行い、均田制による給田や兵役・重税から逃れる
→ 貴族や仏教寺院・道教寺院(道観)による私有地が拡大(5 荘園の形成)
・自作農の解体 … 小作人に転落、荘園をもつ有力者に囲い込まれた(宋代でいう6 佃戸)
→ 国家による給田が行えなくなり、均田制・府兵制が崩壊。税収・国防力が減衰
○軍制の再整備 … 農民の逃亡により府兵制が崩壊していた 河西が最初。 742 年には全国10 カ所
・[7 節度使]の設置(710) … 軍団を指揮し、辺境防備を担当。都護府の役割にかわる
・[8 募兵制]の導入(749)… 府兵制を廃し、傭兵制度を採用 ※玄宗の在位期間:712-756
→ 722 年にはすでに兵の募集を開始しており、12 万人がつどった
○税制の再整備 … 農民の逃亡により、均田制が崩壊していた 原則は銭納であった
・[9 両税法]の導入(780) … 徳宗代の宰相[10 楊炎]による献策。土地公有制を廃止
→ 個人が[11 現住地]に所有する[12 財産や土地]に応じて、[13 春秋２期]に課税
⇒ 租調庸制の崩壊、土地私有の公認。自作農の解体を加速させ、宋代には佃戸が形成
・主戸客戸制 … 両税法とともに導入された戸籍制度。主戸が土地所有者、客戸は無産者
→ 主戸は資産に応じていくつかの等級に分けられ、これに応じて両税を負担
・[14 塩の専売制](761) … 生活必需品の塩を国が専売し、困窮する国家財政の改善をはかる
→ 両税法とともに財政安定に寄与したが、のちに黄巣の乱による唐滅亡の一因となる
