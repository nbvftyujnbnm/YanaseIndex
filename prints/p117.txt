２章 西ヨーロッパにおける覇権争い
１．イギリスの絶対王政と市民革命 16 世紀前半にかけて、官僚制が整備。郷紳(ジェントリ)中心
(1) エリザベス１世(位1558-1603)の支配 … テューダー朝最後の王
①国内政治 「悪貨は良貨を駆逐する」 ※シェークスピアの活躍はこのころ
○貨幣改鋳の実施 … グレシャムの献策。通貨取引の安定化
○[1 統一法](1559) … メアリ１世が廃止した首長法も復活。イギリス国教会の確立
○[2 第１次囲い込み] … 16 世紀を通して展開され、エリザベス１世のころ顕著
・背景 … ヘンリ８世のころ、アントウェルペンへの[3 羊毛]輸出が増加
が『4 ユートピア』を著し、 「5 羊が人間を食う」として囲い込みを批判
・内容 … 地主やジェントリが、農民から耕地・共有地を奪取し、[6 牧羊地]を拡大
→ イギリス議会は反対したが効果なし。第１次囲い込みは暴力的かつ非合法的なもの
・影響 … 毛織物業の発達。エリザベス１世のころにはオスマン帝国にも輸出された
→ 土地を奪われた農民は賃金労働者となり、工場制手工業(マニュファクチュア)が進展
○[7 救貧法](1601) … 囲い込みによる貧富の差の拡大をうけて。社会保障制度のはじめ
②対外政策 都市の商工業者が力をつけジェントリ層に参入、下院の多数派を占める
○カトー＝カンブレジ条約(1559) … ハプスブルク・ヴァロワ家によるイタリア戦争終結に立会う
○重商主義政策 … 積極的な海外進出。国内では囲い込みにより輸出用の毛織物生産が拡大
・レヴァント会社 … 1581 年にオスマン帝国との独占交易を認められ、毛織物を主に輸出
・私拿捕船(私掠船) … 16～17 世紀、イギリスを中心に西欧諸国でみられる。19 世紀に禁止
→ 個人や団体に、国家が他国船を拿捕・略奪する特許を与えた。[8 ドレーク]が有名
・[9 オランダ独立戦争]への介入 … アルマダ海戦(1588)でスペインに勝利、大西洋を制圧
・北米植民の失敗 … ヴァージニア植民地の成立は次代のジェームズ１世を待つ
・[10 イギリス東インド会社]( 1600-1858)の設立 … アジア貿易を独占
※インド拠点 … [11 マドラス](1639)・[12 ボンベイ](1661)・[13 カルカッタ](1690)
(2) イギリス市民革命
①国王による国教強制 メアリ＝ステュアートの子。[16 スコットランド]王ジェームズ６世
○[14 ジェームズ１世](位1603-25) … [15 ステュアート朝]を創始
・ブリテン島の一時的統合 … イングランド・アイルランド・スコットランド同君連合が成立
・[17 王権神授説] … 1598 年(即位前)に「自由なる君主国の真の法」を著す。議会と対立
・[18 イギリス国教会]の保護 … カトリックのほかピューリタン・プレスビテリアンも否定
・[19 ヴァージニア植民地] … [20 ジェームズタウン]建設(1607)。米最初の英植民地
・[21 ニューイングランド]の形成 … 弾圧を逃れたピューリタンがヴァージニアを目指す
→ [22 メイフラワー号]でヴァージニア北方に到達。プリマス建設(1620)
⇒ このときのピューリタンたちはピルグリム＝ファーザーズ(巡礼始祖)といわれる
