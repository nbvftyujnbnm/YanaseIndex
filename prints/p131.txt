(2) 哲学 経験論と合理論の対立は、カントによって
①イギリス経験論 18 世紀に近代観念論哲学に止揚される
〇[1 フランシス＝ベーコン] … ルネサンス末期に帰納法を唱える
→ 実験と観察をくりかえし、事実を積み重ねて一般法則を導き出す
〇[2 ロック] … 経験論哲学を確立、『人間悟性論』。社会契約説の分野でも重要な人物
→ 観念とは生得のものではなく、「白紙(タブラ＝ラサ)」の心に後天的に書き込まれたもの
〇バークリー … 命題「存在するとは知覚されることである(エッセ＝エスト＝ペルキピ)」
〇[3 ヒューム] … 懐疑論(人間の認識力は不完全で、客観的・普遍的真理は正しく認知しえない)
②大陸合理論 物心二元論、力学的世界観の最初(後述)
〇[4 デカルト](仏) … 演繹法を提唱。普遍的真理に論理的推論を重ね、個別的な結論を導出
→ 『方法序説』で命題「我思う、ゆえに我あり(コギト＝エルゴ＝スム)」を示す
〇パスカル(仏) … 遺稿集『パンセ(瞑想録)』に「人間は考える葦である」との言葉が残る
〇[5 スピノザ](蘭) … デカルトの物心二元論に対し、一元論的な汎神論を提唱
→ 主著『エティカ』。神のみが世界で唯一の実体で、他のすべてはその属性や様態のひとつ
〇[6 ライプニッツ](独) … 多元論的な単子論により、機械論と目的論の調和をはかる
→ モナドとは万物を構成する最小実体であり、それぞれ独立しており相互に影響しない
⇒ 自発的に動くそれぞれモナドが世界を構成しているのは、神による予定調和であるとする
(3) 自然科学
①科学革命 … 17 世紀、スコラ哲学・宗教的世界観を打破。自然科学分野の革新、大きな進展
〇フランシス＝ベーコン(1561-1626) … 帰納法を提唱、イギリス経験論のはじめ
→ 観察・実験などの経験を重んじる点が、近代科学の形成における大前提となった
〇デカルト(1596-1650) … 演繹法を提唱。「自己」という唯一の真理から、理性的な推論を重ねる
・物心二元論 … 自己の意識のみが確実な存在で、これは純粋に精神的なもの
→ これ以外の、外界の一切のものは生命的・心的なものを含まないただの物質
⇒ 物質であれば数量化が可能であり、数学的・物理学的に理解することができる
・機械論的自然観 … デカルトの物心二元論を出発点とし、ニュートンのころ確立
→ 自然界の変化はすべて物質の必然的運動で、機械と同じく目的や意図はそこに存在しない
②イギリスの科学者
〇王立協会 … チャールズ２世の勅許のもと1662 年ロンドンに設立。現存最古の学会
→ ボイルや、バネの法則で有名なフックらが参加。ニュートンが会長を務めた時期もあった
〇[7 ハーヴェー](1578-1657) … 血液循環を発見。ジェームズ１世・チャールズ１世の侍医
〇[8 ボイル](1627-91) … ボイルの法則(一定温度で気体の体積・圧力が反比例)
〇[9 ニュートン](1643-1727) …『プリンキピア』に万有引力の法則など示す。力学体系樹立
〇[10 ジェンナー](1749-1823) … 天然痘予防のため種痘法を発明。免疫学・予防接種の原点
