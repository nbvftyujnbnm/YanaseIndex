②アイルランド問題 … 1801 年の併合後、イギリス人地主の搾取により貧窮化
〇ジャガイモ飢饉(1845-49) … アメリカへの移民増加、大陸横断鉄道の労働力となる
〇土地戦争 … 地主に対して小作人が暴力やボイコットを行うもの
・３Ｆ運動 … 1850 年代から、小作権の安定・適正な地代・小作権売買の自由を要求
・アイルランド土地法(1870) … 第１次グラッドストン内閣が３Ｆを認める
→ 不徹底であり、1880 年代にかけて土地戦争が激化
・小作農の自作農化の進展 … グラッドストンは1881 年に３Ｆを完全に認める(新土地法)
→ 次いで土地国有化運動がおこるが、土地戦争はしだいに鎮静化
〇自治獲得への動き … 1867 年のカナダ自治法をうけて、アイルランドでも自治を求める
・[1 シン＝フェイン党]の結成(1905) … 完全独立を求めて激しい運動を展開
・[2 アイルランド自治法]の成立(1914) … 自由党アスキス内閣のとき
→ イギリス人が多い北部の[3 アルスター地方]が反対、アイルランドからの分離を要求
→ シン＝フェイン党は完全独立を求め対立激化。政府は第一次大戦を理由に自治法実施を延期
・イースター蜂起(1916) … シン＝フェイン党は自治延期に反発、ダブリンで武装蜂起。失敗
☆第一次世界大戦後のアイルランド
・シン＝フェイン党による独立宣言(1919) … イギリスと戦争状態に突入。1921 年に停戦
→ 北アイルランドを除外し、形式的に自治領アイルランド自由国が成立(1922)
・デ＝ヴァレラ … 元シン＝フェイン党員。新憲法を制定、国号をエールとする(1937)
・アイルランド共和国の成立(1949) … イギリス連邦から正式に脱退
☆第二次世界大戦後(北アイルランド紛争) … ＩＲＡを中心に、1970～80 年代に激しいテロ活動

→ 1969 年に武力闘争の開始を宣言。このとき英は第１次ウィルソン内閣(労働党、1964-70)

③労働運動の成立と社会改革の推進
〇労働党の成立 理論的指導者ウェッブ夫妻。設立者バーナード＝ショウ(ダブリン生まれの文学者)
・[4 フェビアン協会](1844) … 漸進的な社会主義の実現をめざす
・独立労働党(1893) … 漸進的社会主義。炭鉱夫ケア＝ハーディが設立
・社会民主連盟(1884) … マルクス主義に傾倒したハインドマンが設立
・[5 労働代表委員会](1900) … フェビアン協会・独立労働党・社会民主連盟が合同
→ 1906 年に[6 労働党]と改称。このとき社会民主連盟は離脱
〇自由党の躍進(1906) … 労働党の協力を得て社会改革を推進。労働者の地位向上
・社会福祉の進展 … 国民保健法や老齢年金法が定められる
・[7 議会法](1911) … アスキス内閣のとき。上院に対する下院の優越が定められる
