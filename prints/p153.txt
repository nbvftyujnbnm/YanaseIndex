５章 イギリスの覇権と欧米の国民国家建設
１．ウィーン会議と各地の独立運動 ※「3 会議は踊る、されど進まず」
(1) ナポレオン以後のヨーロッパ … ナポレオンの「百日天下」を経て諸国がようやく妥協
①[1 ウィーン会議](1814-15) … オーストリア外相[2 メッテルニヒ]が主宰
〇[4 タレーラン] … 仏外相。[5 正統主義]を提唱(仏革命以前の国際秩序への復古)
○[6 勢力均衡]の原則 … 強国の出現を防ぎ、ナポレオン以前の国際秩序の復帰をはかる
〇ウィーン議定書(1815) … ヨーロッパの新たな国際秩序が形成(ウィーン体制)
・イギリス … 蘭から[7 セイロン]・[8 ケープ植民地]、仏からマルタ島を獲得
・フランス … ブルボン家の復位を承認。タレーランはこの政権の外相
・プロイセン … 北ザクセン・ラインラントなどを確保 ※スペイン＝ブルボン家も復位
・オーストリア … 北イタリアのロンバルディア・南チロル・ヴェネツィアを併合
→ ベルギー(南ネーデルラント)および南ドイツの飛び地は放棄
※[9 ドイツ連邦] … ライン同盟にかわり成立。普ふくむほぼ全ドイツの領邦・都市が参加
→ 連邦議会の議長はオーストリアが務めた。君主国の集合体で、統一国家とは程遠かった
・ロシア … スウェーデンからフィンランドを獲得。事実上ポーランドも獲得
→ ロシア皇帝が[10 ポーランド立憲王国](ワルシャワ公国にかわり成立)の王を兼ねる形
・[11 スイス] … 永世中立国となる
・オランダ … [12 ベルギー]を獲得し、[13 オランダ立憲王国]が成立
・スウェーデン … ノルウェーを併合 ※両シチリアでもスペイン＝ブルボン朝が復活
・イタリア諸地域 … ローマ教皇領は旧領が復活、サルデーニャはジェノヴァを併合
②ウィーン体制 すでに産業革命を終えているイギリスのみ、基本的に自由主義を支持
〇特徴 … フランス革命・ナポレオン戦争をうけて保守反動。復古主義がとられた
〇[14 神聖同盟](1815) … ロシア皇帝[15 アレクサンドル１世]の提唱
→ キリスト教精神による平和維持。オスマン帝国・ローマ教皇・イギリスは不参加
〇[16 四国同盟](1815) … 英・普・墺・露による、ウィーン体制維持のための国際組織
→ 1818 年にはフランスも参加し、[17 五国同盟]となる メッテルニヒにより抑圧、解散
③体制打破の動き … 各国の自由主義・国民主義(ナショナリズム)運動が、体制と対立
〇ドイツ学生同盟(18 ブルシェンシャフト、1815-19) … ドイツ統一・立憲君主政を目指す
〇炭焼党(19 カルボナリ) … 成立時期不詳、イタリアの秘密結社。19 世紀前半を通じて展開
→ 1820 年ナポリ、1821 年ピエモンテで蜂起するが、オーストリア軍により鎮圧
〇[20 スペイン立憲革命](1820) … 自由主義勢力に軍が同調、一時的に憲法制定に成功
→ 1823 年にフランス軍が介入、憲法撤廃。イギリスは革命を支持、五国同盟に足並みの乱れ
〇[21 デカブリストの乱](1825) … 12 月(デカーブリ)に発生した 指導者リエゴも処刑
→ 貴族出身、自由主義のロシア青年将校がツァーリズム打倒を唱えるも、ロシア政府により鎮圧
