②イギリス
〇ナポレオン戦争中 … オランダ本国がフランスに占領される
→ イギリスはジャワなどオランダ勢力圏を次々と占領。ウィーン会議(1814-15)で返還
〇英領マレー … 英蘭協定(1824)などを経て海峡植民地を形成(1826)
→ さらにマレー半島の４小国を保護国化、マレー連合州を形成(1895)。両者を合わせて成立
〇[1 クック] … イギリス人航海者。1769 年にニュージーランドの領有を宣言
→ 翌年にはオーストラリアが大陸であることを調査(こちらの領有宣言は1829 年)
※[2 タスマン] … 1642 年に、スマニア島・ニュージーランドに到達したオラン人
〇オーストラリア … 原住民[3 アボリジニ]。移民による虐殺で人口激減
・流刑植民地 … アメリカの独立をうけて、オーストラリアがこれに指定される
→ 囚人の移民がすすむ。19 世紀にかけて[4 牧羊業]が発達
・ (1851) … 各国からの自由移民が激増。ヨーロッパ系移民が反発
・オーストラリア連邦の成立(1901) … 自治が認められる
→ 華僑などのアジア系移民を制限(白豪主義)。第二次大戦後まで白人優位社会
〇ニュージーランド … 先住民[5 マオリ]。病原菌や鉄砲が伝来、部族間抗争などで人口減少
→ 1893 年に世界初の女性参政権を実現。1907 年に自治領となる
〇各諸島の支配 … ニューギニア島は英独蘭で分割。英は東南部を支配
・赤道以南 … ソロモン・ニューヘブリデス(現バヌアツ)・フィジー・トンガ・クック諸島
・赤道近辺 … ギルバート諸島・キリティマティ島。フェニックス諸島はアメリカと分割
③フランス … [6 ニューカレドニア](およびロイヤルティ諸島)は現在も領有
→ ソシエテ(7 タヒチを含む)・ツアモツ(パウモツ)・マルケサス諸島を支配
※[8 ゴーガン] … フランスの画家、晩年をヒで過ごす。『ヒの女』(1891)
④ドイツ … ニューギニア島は東北部を支配。残る西部がオランダ領
