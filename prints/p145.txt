②綿花の生産 … [1 ホイットニー](米)が綿繰機を発明(1793)。綿と種などを分離する機械
→ アメリカ南部における綿花プランテーションが強化、生産増大。奴隷制強化も意味した
③動力革命 … 人力・馬力・水力などにかわる、蒸気機関の実用化
〇[2 ニューコメン] … 炭鉱排水用に蒸気ポンプを発明(1708)。熱効率が悪く一般化せず
〇[3 ワット] … 蒸気機関を改良(1769)。力織機に採用されるなど、一般的な動力として普及
④交通革命 … 蒸気機関による、交通網の飛躍的な拡大
〇[4 フルトン](米) … 蒸気船クレアモント号を試作(1807)、ハドソン川に定期航路を開く
→ 死後、サヴァンナ号が英米間を30 日で航行(1819)。蒸気船による初の大西洋横断
※汽船の発達による影響 … 19 世紀後半、ムスリムのメッカ巡礼が活発化。華僑・印僑も増加
〇[5 スティーヴンソン] … トレヴィシックが発明(1804)した蒸気機関車を実用化(1825)
→ ストックトン～ダーリントン間をロコモーション号が時速約18 キロメートルで走った
⇒ 初の営業用鉄道を[6 リヴァプール～マンチェスター間]に開通(1830)。鉄道が普及
〇交通革命の意義 … 18 世紀後半から19 世紀初めまでは運河網が整備され、交通の主力であった
→ 革命後は、鉄道の普及により建設業・炭鉱業・鉄鋼業が飛躍的に発展。通勤圏の拡大にも寄与
☆電気鉄道の普及 … ジーメンス(独)が1879 年に実験成功。ベルリン地下鉄は1902 年を待つ
→ 1890 年、ロンドンに地下鉄が登場。のちパリ(1900)、ニューヨーク(1904)
⑤製鉄業
〇第１次エネルギー革命 … 木炭にかわる燃料として石炭が家庭燃料から徐々に浸透
〇[7 ダービー父子] … コークス製鉄法の実用化(1735)。コークスは石炭からつくる燃料
→ 石炭が製鉄業の燃料としても用いられはじめ、第１次エネルギー革命が急速に進行
〇コート … パドル法の考案(1785)。銑鉄(炭素・硫黄などを含む粗鉄)の精錬にも石炭を使用
⑥通信 過程に反射炉を採用。燃料と金属が接触せず、不純物が混ざりにくい
〇[8 モールス](米) … 1837 年に電信機を発明、翌年にモールス信号を考案している
〇[9 ベル](米) … 電話の発明(1876) 合衆国独立100 年記念のフィラデルフィア万博に展示
〇[10 マルコーニ](伊) … 無線電信を発明(1895)
→ 1901 年には大西洋横断無線通信に成功。1909 年にはノーベル物理学賞を受賞している
※ヘルツ(独) …電磁波の存在を実証(1888)。無線技術の誕生と発展の前提となった
※スラム … 都市部で貧困層が居住する過密地区。産業革命期のロンドンでもみられた
※産業革命の進展
第一次産業革命 … 18 世紀 イギリス 、軽工業（木綿など）中心
第二次産業革命 … 19 世紀後半ドイツ・アメリカ、重工業（機械・造船など）中心
→ 第二次産業革命期に入っていくと、イギリスは「世界の工場」の地位から後退
