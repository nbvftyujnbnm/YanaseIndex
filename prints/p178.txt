②南北戦争
〇北部と南部の対立 … 西部開拓とともに、新たな州を自由州・奴隷州のいずれとするかで対立
・北部 … 23 州。木綿工業を中心とした資本主義的工業がさかん [3 ホイットニー]が
→ 自由な労働力確保のため奴隷制に反対。[1 保護貿易]論 綿繰機を発明(1793)
⇒ 共和党の基盤。かつての連邦派(フェデラリスト)に近い中央集権主義
・南部 … 11 州。黒人奴隷によるプランテーション(タバコ・2 綿花をイギリスに販売)
→ 自由貿易論、民主党の基盤。かつての反連邦派(アンチ＝フェデラリスト)に近い州権主義
〇奴隷制をめぐる南北の対立
・[4 ミズーリ協定](1820) … 南北の妥協案。北緯36 度30 分以上では奴隷制を禁止
→ このラインが中央を通るカリフォルニアは、1850 年に自由州となった
・『5 アンクル＝トムの小屋』… [6 ストウ]が著し、1852 年出版。黒人奴隷が主人公
→ 一年にして30 万部を売り上げ、奴隷制反対の世論が高まる 両州ともに北緯36 度30 分以上
・[7 カンザス＝ネブラスカ法](1854) … ミズーリ協定を否定。帰属は住民投票により決定
→ 奴隷制反対派は、北部資本家・西部農民を基盤とする[8 共和党]を結成(1854)
〇[9 南北戦争]( 1861-65)
・[10 リンカン](任1861-65、16 代) … 北部出身、[11 共和党]。奴隷制拡大阻止を主張
・[12 アメリカ連合国]の成立(1861) … 首都[13 リッチモンド]。南部11 州が分離独立
→ リンカンに対抗し、民主党から[14 ジェファソン＝デヴィス]を大統領とする
⇒ 南部と北部の対立は決定的となり、南北戦争勃発
・緒戦の動向 … リー将軍がリッチモンド防衛戦でたびたび活躍し、南部が優勢
・[15 ホームステッド法](1862) … リンカンが制定。西部の支持をとりつける
→ 西部に５年間定住・耕作したものに160 エーカーを無償で払い下げる
⇒ 北部に敵対する行為をとったものは除外するとした。西部への移民増加
・[16 奴隷解放宣言](1863) … リンカンが発表、内外の世論の支持を得る
→ これをうけてイギリスも南部を支援しにくくなった(英は南部の自由貿易を支持していた)
・[17 ゲティスバーグの戦い](1863) … リー将軍が北部攻撃をはかるが失敗。北部が攻勢へ
・リッチモンド陥落(1865) … 率いる北部軍の攻撃をうけ南部が降伏。南北戦争終結
③南北戦争後のアメリカ 同年の慰霊演説「18 人民の、人民による、人民のための政治」
〇 (1865) … 憲法が修正され、黒人にも市民権・選挙権が与えられる
・解放された黒人奴隷 … 経済的自立は困難 南部では、1890 年代から州法によって骨抜きに
→ 多くは分益小作人(19 シェア＝クロッパー)となり、貧困が続いた
・差別の継続 … 財産による制限選挙、公共施設における黒人隔離など
※クー＝クラックス＝クラン(ＫＫＫ) … 南部諸州で結成された秘密結社
→ 白人至上主義のもと、黒人やアジア人を激しく攻撃。現在も存続
