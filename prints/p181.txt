⑤メキシコの情勢 米によるテキサス併合(1845)が背景 カリフォルニアも米に併合
〇 (1846-48) … 敗北し、国土の半分以上を喪失
〇独裁の打破(1855) … 軍事力をもつクリオーリョ出身者による独裁政権が倒れる
→ メスティーソ出身のフアレスが自由主義的改革を主導。1857 年には憲法を制定
〇[1 ナポレオン３世]による (1861-67) ハプスブルク家出身
・メキシコ内乱 … フランスによってメキシコ皇帝に祭り上げられたマクシミリアンへの反発
・アメリカの反発 … 南北戦争の終結(1865)後、フランスを非難
・フランスの撤兵(1867) … 財政状況の悪化が背景。マネ『マクシミリアンの処刑』(1867)
・フアレスの大統領正式就任(1867) …「建国の父」と称えられる。晩年は独裁
〇[2 ディアス] … フアレス独裁を批判、その死後1877 年に大統領となる。しかし自らも独裁
・産業の近代化 … 一方でアメリカ・イギリスなど外国資本に依存。半植民地化がすすむ
・民衆の窮乏 … 労働運動の弾圧、大土地所有制の復活など。内政は地主階級に依存した
〇メキシコ革命(1910-17)
・[3 マデロ] … 自由主義者。ディアス独裁を批判し大統領選に出馬
→ 農民指導者[4 サパタ]らも独裁に反発し蜂起。ディアス追放、マデロが大統領に(1911)
⇒ 保守派官僚・軍部はマデロに従わず、マデロはサパタら左派を弾圧。混乱が続く
・アメリカの軍事介入 … ウィルソンによる宣教師外交の一環。メキシコ民衆の反米感情が高まる
・憲法の制定(1917) … 新たな共和制憲法を定める。現行のメキシコ憲法
〇カルデナス … 1934 年から1940 年まで大統領を務める。メキシコ革命の定着期
・アメリカの方針転換 … このころはフランクリン＝ローズヴェルトによる善隣外交
・資源ナショナリズム … 自国の資源を自国で管理する。石油産業国有化。鉄道も国有化した
・農地改革の実施 … 従来から革命の目標のひとつであった
・経済の自立化 … 労働環境の改善、中小企業の保護育成など
(4) ドイツの「世界政策」
①ビスマルク外交の終焉 [8 ベルリン]・[9 ビザンティウム]・[10 バグダード]
〇[5 ヴィルヘルム２世](位1888-1918)
・ビスマルクとの対立 … 皇帝は社会主義者鎮圧法の延長・ロシアとの再保障条約更新に反対
→ 1890 年にビスマルクを辞任させ、皇帝親政を開始 [6 クルップ社]の大砲工場が有名
・「新航路(Neuer Kurs)」… ビスマルクからの内政・外政の大転換を皇帝が自称したもの
〇内政 … 労働運動・社会運動を弾圧したビスマルクとは異なり、国内融和をはかる
→ 国内産業や貿易が進展。1910 年代にはイギリスを抜き、アメリカに次ぐ世界２位の生産力
〇外政 …「7 世界政策」と称される、積極的な帝国主義政策
・３Ｂ政策 … 中東方面への海外進出をはかる。イギリスの３Ｃ政策に対抗するもの
・イギリスとの建艦競争 … 世界政策を支えるため、海軍力の増強をはかる
