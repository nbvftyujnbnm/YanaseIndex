④[1 ドイツ帝国] … 初代皇帝ヴィルヘルム１世(位1871-88)。ドイツ初の近代的統一国家
〇政体 … ドイツ帝国憲法に基づく。北ドイツ連邦を継承し、プロイセンがその中心
・[2 外見的立憲主義] … 議会の権限が弱く、国民主権の規定がなく、責任内閣制もない
→ 見かけだけの立憲君主政。皇帝および宰相[3 ビスマルク]に強大な権限が集中
・連邦参議院 … 上院に相当、法案の批准権をもつ。プロイセンの議席数が多く、宰相が議長
・帝国議会 … 下院に相当、25 歳以上の男子普通選挙で選出。法案の審議権のみで決定権なし
〇内政 … 国内対立に苦しむ
・[4 文化闘争] … プロテスタント(ルター派)が多数派、南ドイツ中心に少数派カトリック
→ ビスマルクの抑圧に対し、カトリック勢力は[5 中央党]を結成して抵抗
・「6 アメとムチ」… 労働運動・社会運動を抑圧しようとするが失敗
[7 社会主義者鎮圧法](1878)でドイツ社会主義労働者党を弾圧
医療保険・災害保険・養老保険など[8 社会保険制度を創設]。世界でも最先端
・「鉄と穀物の同盟」… [9 保護関税法](1879)で、資本家とユンカーを結束させた
→ 当初は、産業資本家は保護貿易、ユンカー(地主層)は自由貿易を唱えて対立していた
⇒ ロシアから安価な穀物が流入したため、ユンカーが保護貿易に転換。結束がなった
〇外政 … 露仏同盟の全力阻止によるフランス封じ込めが基本軸 ※岩倉使節団(1871- 1873)の来訪
→ ベルリン会議(1878)でロシアの南下を阻止するなど、巧妙なビスマルク外交を展開

〇対オーストリア 神聖ローマ皇帝レオポルト１世。一時は避難するも、周辺諸国の援助を得た
・[10 第２次ウィーン包囲](1683) … オーストリアに撃退され、以降は連戦連敗
・[11 カルロヴィッツ条約]( 1699)… オーストリアと講和。ハンガリーなどを失う
〇対ロシア 同時代のスルタンはアフメト３世。政治的衰退期、文化面は「チューリップ時代」
・対ピョートル１世 … 1700 年にアゾフ海をロシア領とされる
・対エカチェリーナ２世 … 1768 年から戦争状態、キュチュク＝カイナルジ条約(1774)
→ クリム＝ハン国の宗主権を放棄。クリミア半島および黒海の制海権を失う
〇東方問題 実質ロシアの保護下。ベルリン会議(1878)でトルコ管理下におかれる
・ギリシア独立戦争(1821-31) … アドリアノープル条約(1829)で黒海北岸を露に奪われる
・エジプト＝トルコ戦争(1831-33、39-40) … ロンドン会議(1840)でエジプト総督世襲承認
・クリミア戦争(1853-56) … ナポレオン３世に聖地イェルサレムの管理権を与えたことが契機
・露土戦争(1877-78) … ロシアに敗北、[12 サン＝ステファノ条約](1878)
→ [13 ルーマニア]・[14 セルビア]・[15 モンテネグロ]独立、ブルガリアの自治
