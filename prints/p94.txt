③イギリスへの侵入
○[1 七王国]時代 … ５世紀ごろ、ゲルマン系[2 アングロ＝サクソン人]による国家建設
→ ９世紀からノルマン系[3 デーン人]の侵入に悩まされ、対応のため統一の機運
○[4 イングランド王国]の成立(829) … ウェセックス王[5 エグバート]のが七王国を統一
→ ９世紀末に孫の[6 アルフレッド]がデーン人を撃退するが、10 世紀末には侵入が再開
○[7 クヌート](位1016-35) … デーン人。イングランド征服、デーン朝をひらく(一代限り)
○[8 ノルマン＝コンクェスト]( 1066) 1042 年にはアングロ＝サクソン人王家が復活
・ノルマンディー公[9 ウィリアム]が[10 ヘースティングズの戦い]でイングランド征服
→ イングランド王[11 ウィリアム１世]として即位、[12 ノルマン朝](-1154)を創始
・土地台帳(ドゥームズディ＝ブック)の作成 … 強力な王権を展開、国内の諸侯に提出させた
(8) 封建社会の形成 政治的には封建的主従関係を、経済的には荘園を基盤とする
①ヨーロッパにおける … [13 土地]を媒介とする 中国周代は[14 血縁]を媒介
○背景 … 王権が弱体化し(10 世紀、各地でのカロリング朝断絶など)、個人間の主従関係が形成
→ 国王と諸侯、あるいは諸侯同士で封建的な契約関係が成立。複数の主君に仕える家臣もいた
○特徴 … ゲルマン社会の[15 従士制]とローマ帝政末期の[16 恩貸地制]が結合したとの説
○主従間の双務契約 … 主君は家臣の所領(17 封土)を保護。家臣は主君に[18 軍役義務]
○主従関係の重層化 …「19 臣の臣は臣にあらず」 。国王の影響力も直接契約の範囲まで
○影響 … 一代限りの契約が多かったが、イスラーム勢力やマジャール人に対抗するため契約持続
→ 分権的社会が固定化。王権の弱体が維持され、各地の諸侯が力をもつようになった
※騎士道精神 … 叙任式(オマージュ)で騎士は主君から剣を授けられ、主君への忠誠を誓った
→ キリスト教や宮廷儀礼の影響をうけ、武勇と忠誠・神への奉仕・弱者の保護などを強調
②[20 荘園] … 封建制のもと、領主が独自の裁判権・課税権をもち経営した私有地
○荘園の経営 … 荘園領主(国王・諸侯・騎士・教会・修道院など)が自給自足的な農業生産を行う
・土地の種別 … 領主[21 直営地]、農民貸与の[22 保有地]、牧草地など[23 共有地]
・[24 農奴]の形成 … 農民は保有地の耕作権を得る代償に、領主から厳しい支配を受けた
→ 結婚・職業選択・保有地処分・移住の自由なし。家族の形成・道具や住居の所有などはあり
○荘園領主の特権 ※[25 十分の一税] … 教会が、聖書の記述をもとに農民から広く徴収
・[26 不輸不入権] … 主君からの課税や公使を拒否する権利。地方分権が進み、王権弱体化
・[27 領主裁判権] … 荘園ごとの慣習法に基づき、領主が農民にさまざまな取締りを行った
・[28 賦役] … 農奴に、直営地での一定日数の労働を義務づける。領主の経営の要
・[29 貢納] … 農奴に、保有地での収穫の一部を領主に納めさせる
・農奴への課税 … [30 結婚税]・[31 死亡税]や道具使用料(水車・焼き窯など)を課した
○背景 … 農業生産・交通路(とくに内陸)が未発達で、商工業や[32 貨幣経済]は広まらず
→ 自給自足の[33 交換経済]が主流であったことが、封建社会(荘園制)の成立につながった
