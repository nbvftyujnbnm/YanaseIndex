(2) 草原の道
①[1 スキタイ] … 前７世紀、南ロシア草原(黒海北岸)に成立。遊牧民などの諸集団の連合体
○オリエント世界への侵入 … ギリシア・イランなどと交流。南方のアケメネス朝ペルシアと対立
○文化 … 史上初の遊牧国家として、のちの遊牧騎馬文化の基礎を形成。ユーラシア東方にも伝播
→ 金属製の装身具(武具・馬具には動物の意匠がみられる)や、大型の円墳などが特徴
②[2 匈奴] … 前３世紀末、モンゴル高原に成立。族長の称号は単于(鮮卑・氐・羌でも使用)
○漢民族との対立 … 秦の始皇帝は蒙恬を将軍として遠征軍を派遣、戦国時代の長城も修築
○[3 冒頓単于] … 匈奴の指導者。白登山の戦い(前200)で前漢の高祖を破り最盛期
→ トルコ系民族(東胡/丁零/鉄勒)を併合。[4 月氏]も圧迫(一派が西遷、大月氏となる)
○衰退 … 一時はオアシス地帯も制圧したが、前漢の武帝に攻撃され奪われる。武帝の死後は和睦
・東西分裂(前１世紀) … 東匈奴は前漢に服属、西匈奴は消滅 ４世紀後半、ユーラシア西方に
・南北分裂(後１世紀) … 南匈奴は後漢に服属、北匈奴は西進 出現したフン人との関連(諸説)
③[5 鮮卑] … モンゴル高原東方の遊牧民。もと匈奴に服属、２世紀ごろ自立し有力となる
→ 五胡十六国時代には中国本土に南下。４世紀末には[6 北魏]を建国、漢民族と同化
④[7 柔然](モンゴル系) … ５～６世紀。族長の称号は可汗(突厥・ウイグルでも使用)
→ 鮮卑の南下後にモンゴル高原で台頭。北魏の太武帝に敗れ衰退、のち突厥に滅ぼされる
※５世紀の遊牧民 … ユーラシア西方にはエフタル(イラン系)、中部には高車(トルコ系)
⑤[8 突厥](トルコ系) … ６～８世紀。モンゴル高原を中心に帝国を形成
○成立と滅亡 国王[9 ホスロー１世]
・「第一帝国」… ６世紀、柔然から自立。ササン朝と協力し[10 エフタル]を滅ぼす
→ ６世紀末に隋の攻撃をうけ東西分裂。東突厥は唐の太宗、西突厥は唐の高宗により服属
・「第二帝国」… ７世紀に東突厥が唐から自立。８世紀にウイグルにより滅亡
○特徴 … 北方民族で初の独自文字として[11 突厥文字]を作成(アラム・ソグド文字の影響)
→ また、ソグド人を保護して商業・外交に活用。ササン朝のほかビザンツ帝国とも交流した
⑥[12 ウイグル](トルコ系) … ８～９世紀にモンゴル高原 以降、[14 吐蕃]とも抗争
○国家建設 … ８世紀に突厥を滅ぼし建国。[13 安史の乱]に際して唐を支援し優位にたつ
→ ９世紀に[15 キルギス](トルコ系)の攻撃をうけ国家滅亡。一派が西走しトルキスタンへ
○特徴 … マニ教が国教とされた。仏教も受容。ウイグル文字を作成(ソグド文字の影響)
⑦９世紀以降 … ９世紀にキルギス、10～12 世紀に契丹・タングート、13 世紀にモンゴルが台頭
※チベット高原 … オアシスの道よりも南方に位置する。チベット系民族が居住
○[16 ソンツェン＝ガンポ] … ７世紀半ば、チベットを統一。[17 吐蕃(チベット)]の建国
・[18 チベット文字]の制定 … インド系の表音文字 都ラサ
・チベット仏教の形成 … 大乗仏教が保護をうけ、独自に発展したもの
○最盛期 … ８世紀。唐から甘粛地方を奪って中央アジアに進出、ウイグルと争う
