(5) フランク王国の発展(481-843)
①建国と王朝の交代
○[1 クローヴィス](位 481-511) … 初代国王
・建国 … フランク人諸部族を統一し、フランク王国を建国。[2 メロヴィング朝]を創始
・[3 アタナシウス派]への改宗(496) … 旧ローマ系住民との融和をはかる
・ガリア統一(534) … 他のゲルマン人諸国家と争う。ブルグント王国の征服をもって統一達成
・王国の分裂 … ゲルマン人共通の分割相続制を採用しており、クローヴィスの死後に王国は分裂
○[4 カール＝マルテル] … 宮宰(マヨル＝ドムス。分国ごとにおかれた行政・財務の最高官)
・カロリング家の台頭 … 抗争に勝利し、宮宰の地位を世襲。宮廷の実権をにぎるようになる
・[5 トゥール・ポワティエ間の戦い]( 732) … 軍を率いて勝利。カロリング家の権威上昇
②西ヨーロッパ支配の確立 [6 ウマイヤ朝]がイベリア半島からピレネー山脈をこえて侵入
○[7 ピピン３世(小ピピン)](位751-68) … カール＝マルテルの子
・[8 カロリング朝]の創始 … ローマ教皇の承認を得たうえで王位を奪った
・ピピンの寄進(756) … [9 ランゴバルド人](イタリアに侵入し教皇を悩ませていた)を討伐
→ 遠征により獲得した[10 ラヴェンナ地方]をそのまま教皇に寄進。教皇領のはじまり
○[11 カール大帝(シャルルマーニュ)](位768-814) … ピピン３世の子 モンゴル系
・版図の拡大 … ランゴバルド人を征服(774)。[12 アヴァール人]も破りパンノニア進出
→ ザクセン人も攻撃。イベリア半島のイスラーム勢力と交戦し、ピレネー以西にも進出
ドイツ北部に残ったアングロ＝サクソン人 『ローランの歌』の題材 スペイン辺境伯を設置
・アッバース朝との関係 … ハールーン＝アッラシードと使節を交換 中心都市アーヘン
・中央集権の強化 … 全土を伯管轄区に分割し、地方の有力者を[13 伯(コムス)]に任命
→ 伯および辺境伯(国境防衛のための地方行政官)の監視のため、[14 巡察使]も設置
・[15 カロリング＝ルネサンス] … ラテン語を普及させ、古典文芸の復興につとめた
中心都市アーヘン(現ドイツ、ベルギー国境付近) 神学者[16 アルクィン]による教育
・[17 カールの戴冠]( 800) … ローマ教皇[18 レオ３世]からローマ皇帝の冠を授かる
* フランク王国 … 名目上「西ローマ帝国」が復活し、ビザンツ帝国に対抗しうる勢力となる
* ローマ＝カトリック教会 … ビザンツ皇帝の支配から逃れ、東方教会に対抗しうる勢力となる
⇒ 西ヨーロッパ文化圏の成立。ゲルマン人によるキリスト教中心の文化。古典文化を復興・継承
※[19 スラヴ人]の拡大 … アヴァール人に圧迫され、カルパティア山脈から拡大(６世紀)
・西スラヴ人 … バルト海沿岸からエルベ川流域に定着 ９世紀後半、東方正教に改宗
→ のちのポーランド人・チェコ(チェック)人・スロヴァキア人
・南スラヴ人 … バルカン半島に定着。ブルガール人は南スラヴ人と混血しスラヴ化
→ のちのセルビア人・クロアティア人・スロヴェニア人
・東スラヴ人 … ドニエプル川流域から北方の平原地帯。ロシア人・ウクライナ人
