(3) 十字軍・モンゴルとの対立
①[1 アイユーブ朝](1169-1250) … 都カイロ。エジプトのスンナ派支配を回復
○[2 サラディン](サッラーフ＝アッディーン、位1169-1193) …
→ ファーティマ朝のカリフにとってかわり、自らはスルタンを称する。イクター制を採用
○十字軍との戦い 第１回十字軍が占領に成功していた
・ヒッティーンの戦い(1187) … サラディンが十字軍を破り、[3 イェルサレム]を奪取
・[4 第３回十字軍](1189) … サラディンがリチャード１世(英)らを撃退
・第６回十字軍(1250) … マムルーク主体の軍で迎撃し、ルイ９世(仏)を捕虜とした
○滅亡 … マムルークによるクーデターが発生、スルタンが殺害される
②[5 マムルーク朝](1250-1517) … 都カイロ。滅亡はオスマン帝国による
○バイバルス … アイユーブ朝のマムルークの一人。クーデターを起こし５代君主となる
・アインジャールートの戦い(1260) … パレスチナで、フラグ率いるモンゴル軍を撃退
・カリフの保護 … アッバース家の生き残りをカイロに招き、カリフとして推戴(1261)
→ [6 メッカ・メディナ]の保護権を獲得。歴代君主はカリフからスルタンに任命された
・国家制度 … イクター制の採用、駅伝制の整備。スルタンの世襲はほぼなし
○カイロの発展 … ファーティマ朝の成立で、イスラーム世界の中心地がバグダードから移る
※カーリミー商人 … アイユーブ朝・マムルーク朝の保護をうけたムスリム商人。ダウ船を利用
→ インド洋～紅海～ナイル川～カイロ～アレクサンドリア～地中海ルートで活躍
○第７回十字軍(1270) … 釈放されたルイ９世が最後の十字軍を率いる
→ アフリカ北岸のチュニスに上陸したが、病の蔓延でルイ９世が死去。成果なく撤退
(4) 各地のイスラーム王朝 ※[7 ベルベル人] … 北アフリカの原住民。アジア・アフリカ混血
①イベリア半島・北アフリカ → ７世紀からアラブ人と同化しはじめ、イスラーム化していった
○イドリース朝(789-925) … モロッコにて、ベルベル人の支持をうけて成立。アラブ系
→ アッバース朝から自立、シーア派。ファーティマ朝によって滅ぼされた
○アグラブ朝(800-905) … チュニジアのアラブ系独立政権。アッバース朝のカリフを奉じた
→ シリチア島も支配したが、ファーティマ朝によって滅ぼされた
○[8 ムラービト朝](1056-1147) … モロッコの遊牧ベルベル人王朝。都[9 マラケシュ]
→ 厳格なスンナ派王朝であり、異教徒や堕落したイスラームにたいして聖戦を展開
⇒ [10 ガーナ王国]を滅ぼし、西アフリカがイスラーム化。イベリア半島にも進出
○[11 ムワッヒド朝](1130-1269) … モロッコの定住ベルベル人王朝。都マラケシュ
→ ムラービト朝を滅ぼしイベリア半島にも進出したが、キリスト教勢力におされて滅亡
○[12 ナスル朝](1232-1492) … イベリア半島最後のイスラーム王朝。アラブ系
→ 都[13 グラナダ]には[14 アルハンブラ宮殿]が造営された
⇒ キリスト教徒によるレコンキスタ(国土回復運動)をうけて滅亡( 1492)
